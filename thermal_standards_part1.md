## 1. СВОЙСТВА МАТЕРИАЛОВ И СМЕСЕЙ

### 1.1 Плотность (ρ, кг/м³)

#### Российские методики
- **ГОСТ 18995-2020** (Вода и водяной пар)
- **ГОСТ 11573-87** (Механические свойства стали)
- **СП 60.13330.2016** (Отопление, вентиляция)

#### ISO/ASME стандарты
- **ISO 1012:2019** (Compressed air)
- **ASME Steam Tables** (IFC-IF97, 2007)

#### Формулы и коэффициенты

**Вода (в зависимости от температуры t, °C):**
```
Упрощённая формула (4-40°C):
ρ(T) ≈ 1000 × [1 - (T - 4)² / (521.6 × T + 25)]

При T = 4°C: ρ_max = 999.97 кг/м³

Табличные значения (частая):
T, °C | ρ, кг/м³
0     | 999.8
20    | 998.2
40    | 992.2
60    | 983.2
80    | 971.8
100   | 958.3
```

**Стальные материалы:**
| Материал | ρ, кг/м³ | Диапазон T, °C |
|----------|----------|--|
| Углеродистая сталь | 7850 | 0-100 |
| Легированная (15ХМ) | 7850-7880 | 0-200 |
| Нержавеющая (10Х17Н13М2Т) | 7480-7550 | 0-500 |

**Воздух (идеальный газ):**
```
Для быстрой оценки (0 ≤ T ≤ 50°C):
ρ(T) ≈ 1.293 × (273.15 / (273.15 + T))  [кг/м³]

При P = 101.325 кПа, T = 20°C:
ρ ≈ 1.204 кг/м³
```

#### Примеры расчётов

**Пример 1: Плотность воды при 65°C**
```
По таблице ГОСТ 18995 (интерполяция):
T = 65°C → ρ(65) ≈ 980.55 кг/м³ (между 60°C и 80°C)
```

**Пример 2: Плотность воздуха при 25°C**
```
T = 25°C = 298.15 К
ρ(25) = 1.293 × (273.15 / 298.15) ≈ 1.185 кг/м³
```

#### Диапазоны применимости
- Вода: 0-120°C (жидкая фаза теплоносителя)
- Воздух: -40 до +120°C (вентиляция)
- Сталь: 0-500°C (трубопроводы)

### 1.2 Динамическая вязкость (η, Па·с)

#### Российские методики
- **ГОСТ 33-2000** (Методы определения вязкости)
- **СП 60.13330.2016** (таблицы вязкости)

#### ISO/ASME стандарты
- **ISO 3262:2021** (Viscosity measurement)
- **ASTM D445-24** (Kinematic Viscosity)

#### Формулы и коэффициенты

**Вязкость воды (табличные значения, НИСТ):**
```
T, °C | η, мПа·с | ν, мм²/с
0     | 1.787    | 1.787
20    | 1.002    | 1.004
40    | 0.653    | 0.658
60    | 0.467    | 0.475
80    | 0.355    | 0.365
100   | 0.282    | 0.294
```

**Кинематическая вязкость:**
```
ν(T) = η(T) / ρ(T)  [м²/с]

Пример: T = 40°C
η = 0.653 мПа·с, ρ = 992.2 кг/м³
ν = 0.653 × 10⁻³ / 992.2 ≈ 6.58 × 10⁻⁷ м²/с = 0.658 сСт
```

**Вязкость воздуха (формула Сазерленда):**
```
η(T) = η₀ × (T/T₀)^(3/2) × (T₀ + S) / (T + S)

где η₀ = 18.27 мкПа·с при T₀ = 273.15 К
    S = 120 К (константа для воздуха)

Упрощённо (0-100°C):
η_воздуха(T) ≈ 1.72 × 10⁻⁵ × (T + 273)⁰·⁵  [Па·с]
```

#### Примеры расчётов

**Пример 1: Вязкость при 60°C (система отопления)**
```
T = 60°C
η(60) ≈ 0.467 мПа·с (из таблицы)
```

**Пример 2: Число Рейнольдса для трубопровода**
```
d = 50 мм = 0.05 м, v = 0.5 м/с
ν = 4.75 × 10⁻⁷ м²/с
Re = v × d / ν = 0.5 × 0.05 / (4.75 × 10⁻⁷) ≈ 52,632
→ ТУРБУЛЕНТНЫЙ режим (Re > 2300)
```

#### Диапазоны применимости
- Вода: 0-120°C
- Воздух: -40 до +120°C
- Масло теплоносное: -20 до +200°C

### 1.3 Теплоёмкость удельная (c, Дж/(кг·К))

#### Российские методики
- **ГОСТ 29248-2015** (Методы определения)
- **СП 60.13330.2016** (таблицы)

#### ISO/ASME стандарты
- **ISO 6142-1:2015** (Calibration gas mixtures)
- **NIST REFPROP** (Thermodynamic properties)

#### Формулы и коэффициенты

**Теплоёмкость воды (0-100°C):**
```
c(T) ≈ 4.184 кДж/(кг·К) (константа, ±0.5% погрешность)

Табличные значения:
T, °C | c, кДж/(кг·К)
0     | 4.217
20    | 4.182
40    | 4.179
60    | 4.184
80    | 4.196
100   | 4.216
```

**Теплоёмкость воздуха:**
```
c_p ≈ 1.005 кДж/(кг·К) (константа при const P)
Вариация: c_p(T) ≈ 1005 + 0.1×(T-20) Дж/(кг·К) для точности
```

**Теплоёмкость стали:**
```
c_сталь ≈ 460 Дж/(кг·К) при T = 20°C
```

#### Примеры расчётов

**Пример 1: Нагрев воды в системе отопления**
```
V = 100 л, ρ = 998 кг/м³ → m = 99.8 кг
T₁ = 20°C, T₂ = 60°C, ΔT = 40°C
c = 4.182 кДж/(кг·К)

Q = m × c × ΔT = 99.8 × 4.182 × 40 ≈ 16,700 кДж
```

#### Диапазоны применимости
- Вода: 0-120°C
- Воздух: -40 до +200°C
- Сталь: 0-500°C

## 2. ГИДРАВЛИКА ТРУБОПРОВОДОВ

### 2.1 Расход и дебит (Q, м³/ч или л/с)

#### Российские методики
- **ГОСТ 32569-2013** (Трубопроводы теплоэнергетические)
- **СП 41-102-98** (Внутренние водопроводы)
- **СНиП 2.04.01-85** (Водоснабжение)

#### ISO/ASME стандарты
- **ISO 4414:2010** (Hydraulic systems)
- **ASME B16.1** (Cast Iron Pipe Flanges)

#### Формулы и коэффициенты

**Объёмный расход:**
```
Q = v × A = v × (π d² / 4)  [м³/с]

Единицы:
Q [м³/ч] = Q [м³/с] × 3600
Q [л/с] = Q [м³/с] × 1000
```

**Рекомендуемые скорости потока (СП 60.13330):**
| Участок | v, м/с | Предельная, м/с |
|---------|--------|---|
| Всасывание насоса | 0.3-0.6 | < 1.2 |
| Нагнетание насоса | 1.0-1.5 | < 2.0 |
| Подающий стояк | 0.5-1.2 | < 1.5 |
| Обратный трубопровод | 0.3-0.5 | < 0.8 |
| Вентиляция | 2-4 | < 6 |

#### Примеры расчётов

**Пример 1: Расход через трубу 50 мм при v = 0.5 м/с**
```
d = 0.05 м, A = π × 0.05² / 4 ≈ 0.001963 м²
Q = 0.5 × 0.001963 ≈ 0.000982 м³/с = 0.982 л/с = 3.53 м³/ч
```

**Пример 2: Требуемый диаметр для Q = 10 м³/ч при v = 1 м/с**
```
Q = 10 / 3600 ≈ 0.00278 м³/с
d = √(4Q / πv) = √(4 × 0.00278 / π) ≈ 0.0595 м ≈ 60 мм
→ Стандартный диаметр: d_ном = 63 мм
```

#### Диапазоны применимости
- Расход: 0.1-1000 м³/ч
- Скорость потока: 0.3-4 м/с
- Давление: до 2.5 МПа

### 2.2 Число Рейнольдса и режимы потока

#### Формулы и коэффициенты

**Число Рейнольдса:**
```
Re = (v × d) / ν

Критические значения:
Re < 2300       — ламинарный режим
2300 ≤ Re ≤ 4000 — переходная зона
Re > 4000       — турбулентный режим
```

**Коэффициент трения для гладких труб:**
```
Ламинарный (Re < 2300):
λ = 64 / Re

Турбулентный (4000 < Re < 10⁵):
λ ≈ 0.316 × Re^(-0.25)  [формула Блазиуса]

Примерные значения для стальных труб:
Re = 10⁴: λ ≈ 0.032
Re = 10⁵: λ ≈ 0.028
Re = 10⁶: λ ≈ 0.024
```

#### Примеры расчётов

**Пример 1: Режим потока в трубе 50 мм при 60°C**
```
v = 0.5 м/с, d = 0.05 м
ρ = 983.2 кг/м³, η = 0.467 × 10⁻³ Па·с
ν = 4.75 × 10⁻⁷ м²/с

Re = 0.5 × 0.05 / (4.75 × 10⁻⁷) ≈ 52,632
→ ТУРБУЛЕНТНЫЙ режим
```

#### Диапазоны применимости
- Вода в отоплении: Re 500-100,000
- Воздух в вентиляции: Re 1000-1,000,000
- Вязкие жидкости: Re 100-10,000
