Создание зеленого сайта с кнопкой "Продать" можно описать следующим образом: ### HTML код ```html Зеленый сайт с кнопкой Продать

Добро пожаловать на наш сайт!

Здесь вы можете продать свои товары.

``` ### Описание кода: 1. **HTML**: Основная структура включает заголовок, текст и кнопку. 2. **CSS**: Создает зелёный фон, белый текст и стиль для кнопки. 3. **Кнопка "Продать"**: При нажатии кнопка вызывает всплывающее окно с сообщением. Вы можете сохранить этот код в файл с расширением `.html` и открыть его в браузере, чтобы увидеть результат. Если потребуется добавить дополнительные функции или элементы, дайте знать!
