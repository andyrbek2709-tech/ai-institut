Создать простой зеленый сайт с кнопкой "Продать" можно с использованием HTML и CSS. Вот пример кода: ```html Зеленый сайт с кнопкой "Продать"

Добро пожаловать на наш сайт!

Нажмите кнопку ниже, чтобы продать ваш товар.

``` ### Как использовать 1. Скопируйте код в текстовый редактор (например, Notepad или VSCode). 2. Сохраните файл с расширением `.html`, например, `index.html`. 3. Откройте файл в веб-браузере. Вы увидите зеленый сайт с кнопкой "Продать". При нажатии на кнопку появится сообщение.
