# 1. Мощность и Ток

## 1.1 Однофазные Цепи (230V / 110V)

### Методика 1.1.1: Расчёт мощности и тока (ПУЭ 7-е издание, раздел 1.3)

**Формулы:**

| Величина | Формула | Единица |
|----------|---------|---------|
| Активная мощность | P = U × I × cos(φ) | Вт |
| Реактивная мощность | Q = U × I × sin(φ) | ВАр |
| Полная мощность | S = √(P² + Q²) | ВА |
| Ток (по P) | I = P / (U × cos(φ)) | А |
| Ток (по S) | I = S / U | А |
| Коэффициент мощности | cos(φ) = P / S | — |

**Стандартные значения cos(φ) (ГОСТ 13109):**

| Тип нагрузки | cos(φ) |
|--------------|---------|
| Резистивная (нагреватель) | 1.0 |
| Лампы накаливания | 1.0 |
| Светодиодные (с драйвером) | 0.95–0.98 |
| Асинхронный двигатель (номинал) | 0.85–0.95 |
| Трансформатор (холостой ход) | 0.1–0.3 |
| Тиристорный выпрямитель | 0.6–0.8 |

**Пример 1: Расчёт тока для электрического чайника**

Дано:
- Номинальная мощность: P = 2000 Вт
- Напряжение: U = 230 В
- cos(φ) = 1.0 (резистивная нагрузка)

Решение:
```
I = P / (U × cos(φ)) = 2000 / (230 × 1.0) = 8.7 А
```

Вывод: нужна розетка не менее 10 А, автомат не менее 10 А (ГОСТ 60898).

**Практические замечания:**

1. **Проверка нагрева кабеля:** Imax провода = I × 1.25 (запас)
2. **Выбор автомата:** IBreaker = I × 1.25 (ПУЭ 1.3.26)
3. **Перегрузка:** Если I_реальная > I_номинальная, срабатывает автомат за 5 сек (тепловой элемент)

---

### Методика 1.1.2: Расчёт параллельных нагрузок (сумма токов)

**Формула:**

I_total = I₁ + I₂ + … + Iₙ (для однофазной цепи)

**Коррекция при разных cos(φ):**

Для нескольких нагрузок:
```
P_total = P₁ + P₂ + … + Pₙ
Q_total = Q₁ + Q₂ + … + Qₙ
S_total = √(P_total² + Q_total²)
I_total = S_total / U
```

**Пример 2: Расчёт автомата для квартиры**

Дано (однофазная сеть 230V):
- Чайник: P₁ = 2000 W, cos(φ₁) = 1.0
- Холодильник: P₂ = 400 W, cos(φ₂) = 0.95 (асинхронный компрессор)
- Стиральная машина: P₃ = 2200 W, cos(φ₃) = 0.9

Решение:
```
Q₁ = 0 (нет реактивной части)
Q₂ = 400 × tan(arccos(0.95)) = 400 × 0.329 = 131.6 ВАр
Q₃ = 2200 × tan(arccos(0.9)) = 2200 × 0.484 = 1064.8 ВАр

Q_total = 0 + 131.6 + 1064.8 = 1196.4 ВАр
P_total = 2000 + 400 + 2200 = 4600 W

S_total = √(4600² + 1196.4²) = √(21160000 + 1431321) = 4687 ВА

I_total = 4687 / 230 = 20.4 А

Требуемый автомат: IBreaker = 20.4 × 1.25 = 25.5 А → выбираем 32 А (ГОСТ 60898)
```

## 1.2 Трёхфазные Цепи (380V / 400V)

### Методика 1.2.1: Расчёт мощности и тока (ПУЭ 1.3.2)

**Формулы для симметричной нагрузки:**

| Величина | Формула | Примечание |
|----------|---------|-----------|
| Активная мощность | P = √3 × U × I × cos(φ) | U — линейное напряжение |
| Реактивная мощность | Q = √3 × U × I × sin(φ) | |
| Ток на фазу | I = P / (√3 × U × cos(φ)) | Для 380V: √3 ≈ 1.732 |
| Напряжение фазы (фаза-земля) | U_фаза = U_линия / √3 | |

**Таблица выбора тока по мощности (380V, cos(φ)=0.95):**

| P (кВт) | I (А) | Рекомендуемый кабель (Cu) | Автомат (А) |
|---------|-------|--------------------------|-------------|
| 5.5 | 8.8 | 2.5 мм² | 10 |
| 7.5 | 12 | 4 мм² | 16 |
| 11 | 17.6 | 6 мм² | 20 |
| 15 | 24 | 10 мм² | 25 |
| 22 | 35 | 16 мм² | 40 |
| 30 | 48 | 25 мм² | 50 |
| 37 | 59 | 35 мм² | 63 |
| 45 | 72 | 50 мм² | 80 |

**Пример 3: Расчёт тока для трёхфазного асинхронного двигателя**

Дано:
- Номинальная мощность: P = 7.5 кВт
- Напряжение: U = 380 В
- cos(φ) = 0.88 (типичное для АД)

Решение:
```
I = P / (√3 × U × cos(φ))
I = 7500 / (1.732 × 380 × 0.88)
I = 7500 / 579.4 = 12.94 А → 13 А

Выбираем кабель: 4 мм² Cu (допустимый ток 30 А)
Выбираем автомат: 16 А (ГОСТ 60898)
```

---

### Методика 1.2.2: Балансировка нагрузок по фазам

**Принцип:** токи по трём фазам должны отличаться не более чем на 10% (ГОСТ 13109-97).

**Формула контроля дисбаланса:**

```
K_дисб = (I_max - I_min) / I_avg × 100% ≤ 10%

I_avg = (I_A + I_B + I_C) / 3
```

**Пример 4: Балансировка нагрузок в трёхфазной сети**

Дано (380V сеть):
- Фаза A: чайник (2 кВт) → I_A = 5.04 А
- Фаза B: холодильник (0.4 кВт) + TV (0.2 кВт) → I_B = 1.04 А
- Фаза C: свободна → I_C = 0 А

I_avg = (5.04 + 1.04 + 0) / 3 = 2.03 А
K_дисб = (5.04 - 0) / 2.03 × 100% = 248% (ЭТО НЕДОПУСТИМО!)

**Решение:** подключить тепловой источник на фазу C (электрокотёл 2 кВт на C) → I_C = 5.04 А
Новый K_дисб = (5.04 - 1.04) / 3.4 × 100% = 118% (всё ещё плохо)

**Оптимальное решение:** распределить:
- Фаза A: чайник (2 кВт)
- Фаза B: холодильник (0.4 кВт) + одна половина котла (1 кВт из 2)
- Фаза C: вторая половина котла (1 кВт)

Новые токи: I_A = 5.04 А, I_B = 3.46 А, I_C = 3.46 А → K_дисб = 18% (хорошо)

## 1.3 Влияние коэффициента мощности на размеры оборудования

### Методика 1.3.1: Рассчитываемые потери при низком cos(φ) (ПУЭ 1.3.30)

**Потери энергии в кабеле:**

```
ΔP = I² × R = (S / U)² × R = S² × R / U²

где S = P / cos(φ)

Следовательно:
ΔP ∝ 1 / cos²(φ)
```

**Таблица: потери мощности при разных cos(φ) (для фиксированной P):**

| cos(φ) | S (кВА) при P=10кВт | I (A) при U=380V | Потери × |
|--------|--------|---------|---------|
| 1.0 | 10 | 15.2 | 1.0 ×  |
| 0.95 | 10.5 | 16 | 1.1 × |
| 0.90 | 11.1 | 16.9 | 1.2 × |
| 0.80 | 12.5 | 19 | 1.6 × |
| 0.70 | 14.3 | 21.8 | 2.0 × |

**Вывод:** При cos(φ) = 0.7, потери в 2 раза выше, чем при cos(φ) = 1.0 для той же активной мощности.

**Практические замечания:**

1. **Конденсаторы коррекции:** Для асинхронных двигателей cos(φ)=0.85 → рекомендуется установить конденсаторы, поднять до cos(φ)=0.95.
2. **Штраф энергосбыта:** При cos(φ) < 0.95 многие энергосбыты берут штраф 1-3% за каждые 0.05 (ГОСТ 30286).
3. **Размер трансформатора:** Трансформатор выбирается по S (кВА), а не по P (кВт).

# 2. Кабели и Линии

## 2.1 Подбор сечения кабеля

### Методика 2.1.1: По допустимому нагреву (ГОСТ 52373, ПУЭ 1.3.26)

**Принцип:** Ток в кабеле должен быть ≤ допустимому току кабеля при температуре окружения 20–30°C.

**Таблица допустимых токов для проводов Cu (изоляция ПВХ, прокладка в воздухе при 30°C):**

| Сечение (мм²) | Ток (А) для 2-жильного | Ток (А) для 3-жильного | Ток (А) для 4-5-жильного |
|----------------|----------------------|----------------------|-------------------------|
| 1.5 | 19 | 17 | 15 |
| 2.5 | 27 | 25 | 23 |
| 4 | 38 | 35 | 32 |
| 6 | 50 | 46 | 42 |
| 10 | 70 | 64 | 60 |
| 16 | 95 | 85 | 80 |
| 25 | 125 | 110 | 105 |
| 35 | 160 | 140 | 130 |
| 50 | 200 | 175 | 160 |

**Метод расчёта:**

1. Найти I_эксплуатации = P / (U × cos(φ)) [или заданный ток]
2. Найти I_выбора = I_эксплуатации × K_запас, где K_запас = 1.25 для однофазного, 1.15 для трёхфазного
3. Из таблицы выбрать сечение, где I_допустимый ≥ I_выбора

**Пример 5: Подбор кабеля для однофазной линии**

Дано:
- Мощность: P = 5 кВт (кухня, однофазная сеть 230V)
- Расстояние: L = 20 м
- cos(φ) = 0.95

Решение:
```
I = P / (U × cos(φ)) = 5000 / (230 × 0.95) = 22.9 А
I_выбора = 22.9 × 1.25 = 28.6 А

По таблице: 27 А < 28.6 А (маловато)
Выбираем сечение 4 мм² → допустимый ток 38 А ✓
Итог: кабель ПВЗ 2×4 мм² (или NYM 2×4 мм²)
```

---

### Методика 2.1.2: По потерям напряжения (ПУЭ 1.3.33)

**Допустимые потери напряжения:**

| Назначение линии | Δu (%) |
|---|---|
| Магистральная линия (до трансформатора) | 4-5% |
| Линия до вводного автомата (питающая линия) | 3% |
| Линия от щита до потребителя (распределительная) | 2-3% |
| Итого до розетки/прибора | 5-7% |

**Формула потерь напряжения:**

```
ΔU (В) = (ρ × L × I) / S

где:
ρ — удельное сопротивление материала (медь: 0.0178 Ом×мм²/м)
L — длина кабеля (одного проводника, м)
I — ток (А)
S — сечение (мм²)

ΔU (%) = (ΔU / U_номинальное) × 100%
```

**Таблица: Потери напряжения на 1А и 1м длины (Cu, одножильный):**

| Сечение (мм²) | ΔU/AI (мВ/А/м) | ПРИ I=10A на L=10m |
|---|---|---|
| 1.5 | 11.9 | 1.19 В |
| 2.5 | 7.1 | 0.71 В |
| 4 | 4.4 | 0.44 В |
| 6 | 3.0 | 0.30 В |
| 10 | 1.8 | 0.18 В |
| 16 | 1.1 | 0.11 В |

**Пример 6: Проверка потерь напряжения**

Дано:
- Линия: от электрощита до розетки в ванной комнате
- Длина кабеля (туда-обратно): L = 2 × 25 м = 50 м
- Ток в линии: I = 13 А (асинхронный компрессор холодильника, пиковый)
- Сечение: S = 2.5 мм²
- Номинальное напряжение: U = 230 В

Решение:
```
ΔU = (0.0178 × 50 × 13) / 2.5 = 11.59 / 2.5 = 4.64 В

ΔU(%) = (4.64 / 230) × 100% = 2.0% ✓ (допустимо, < 3%)
```

**Если бы взяли S = 1.5 мм²:**
```
ΔU = (0.0178 × 50 × 13) / 1.5 = 7.73 В = 3.36% ✗ (недопустимо)
```

### Методика 2.1.3: По нагреву (тепловая устойчивость)

**Учитывает:** температуру окружающей среды, условия прокладки, наличие соседних кабелей.

**Коэффициенты температурной коррекции (ГОСТ 52373):**

| Температура окружения | K_T |
|---|---|
| 10°C | 1.13 |
| 20°C | 1.05 |
| 30°C | 1.0 |
| 40°C | 0.91 |
| 50°C | 0.79 |
| 60°C | 0.71 |

**Коэффициент скопления (если рядом несколько кабелей):**

| Количество кабелей | K_скоп |
|---|---|
| 1 | 1.0 |
| 2 | 0.8 |
| 3 | 0.7 |
| 4 | 0.6 |
| 5 | 0.55 |

**Рассчитанный допустимый ток:**

```
I_доп_реальн = I_доп_табл × K_T × K_скоп
```

**Пример 7: Коррекция допустимого тока**

Дано:
- Кабель ПВЗ 4 мм², табличный допустимый ток I_доп = 38 А
- Температура в щитке: 50°C
- Количество кабелей в гофре: 4
- Нужно найти реальный допустимый ток

Решение:
```
K_T(50°C) = 0.79
K_скоп(4 кабеля) = 0.6

I_доп_реальн = 38 × 0.79 × 0.6 = 18 А

Это сечение теперь может пропускать только 18 А вместо 38 А!
```

## 2.2 Расчёт сопротивления и проводимости

### Методика 2.2.1: Сопротивление одного провода

**Формула:**

```
R = ρ × L / S

где:
ρ — удельное сопротивление (Ом×мм²/м):
    - Медь (Cu): 0.0178 при 20°C
    - Алюминий (Al): 0.0286 при 20°C
    - Железо (Fe): 0.1 (для стального воздухопровода)

L — длина провода (м)
S — поперечное сечение (мм²)
```

**Коррекция на температуру:**

```
ρ(T) = ρ(20°C) × [1 + α(T - 20°C)]

где α (температурный коэффициент):
- Медь: 0.004 K⁻¹
- Алюминий: 0.0039 K⁻¹
```

**Таблица типовых сопротивлений (20°C, медь):**

| Сечение (мм²) | R за 1 м | R за 10 м | R за 100 м |
|---|---|---|---|
| 1 | 0.0178 Ом | 0.178 Ом | 1.78 Ом |
| 1.5 | 0.0119 Ом | 0.119 Ом | 1.19 Ом |
| 2.5 | 0.0071 Ом | 0.071 Ом | 0.71 Ом |
| 4 | 0.0044 Ом | 0.044 Ом | 0.44 Ом |
| 6 | 0.0030 Ом | 0.030 Ом | 0.30 Ом |
| 10 | 0.0018 Ом | 0.018 Ом | 0.18 Ом |

**Пример 8: Расчёт сопротивления кабельной линии**

Дано:
- Кабель Cu 2.5 мм²
- Длина: L = 50 м (туда-обратно, т.е. сумма двух проводников)
- Температура: T = 35°C

Решение:
```
Базовое сопротивление:
R(20°C) = 0.0178 × 50 / 2.5 = 0.356 Ом

Температурная коррекция:
ρ(35°C) = 0.0178 × [1 + 0.004 × (35 - 20)] = 0.0178 × 1.06 = 0.0189 Ом×мм²/м

R(35°C) = 0.0189 × 50 / 2.5 = 0.378 Ом

Конечное сопротивление: 0.378 Ом
```

## 2.3 Стабилизация напряжения и компенсация реактивной мощности

### Методика 2.3.1: Выбор ёмкости конденсаторов (ГОСТ 29049)

**Цель:** поднять cos(φ) с исходного до целевого (обычно до 0.95–0.98).

**Формула:**

```
C (мкФ) = (P × tan(φ₁) - P × tan(φ₂)) × 10⁶ / (2π × f × U²)

где:
P — активная мощность (Вт)
φ₁ — исходный угол (arccos(cos(φ)_исходный))
φ₂ — целевой угол (arccos(cos(φ)_целевой))
f — частота сети (50 Гц)
U — напряжение (В)

Упрощённо (для частоты 50 Гц):

Q_конд (ВАр) = P × (tan(φ₁) - tan(φ₂))

C = Q_конд × 10⁶ / (2π × 50 × U²)
```

**Таблица: значения tan(φ) для разных cos(φ):**

| cos(φ) | φ (°) | tan(φ) |
|---|---|---|
| 1.0 | 0 | 0 |
| 0.98 | 11.3 | 0.20 |
| 0.95 | 18.2 | 0.33 |
| 0.90 | 25.8 | 0.48 |
| 0.85 | 31.8 | 0.62 |
| 0.80 | 36.9 | 0.75 |
| 0.75 | 41.4 | 0.88 |
| 0.70 | 45.6 | 1.02 |

**Пример 9: Выбор конденсатора для асинхронного двигателя**

Дано:
- Мощность двигателя: P = 7.5 кВт
- Исходный cos(φ) = 0.85
- Целевой cos(φ) = 0.95
- Напряжение: U = 380 В (трёхфазное)
- Частота: 50 Гц

Решение:
```
tan(φ₁) = tan(arccos(0.85)) = 0.62
tan(φ₂) = tan(arccos(0.95)) = 0.33

Q_конд = 7500 × (0.62 - 0.33) = 7500 × 0.29 = 2175 ВАр

Для трёхфазной сети (конденсаторы подключены звездой или треугольником):
- При подключении звездой (Y): C = 2175 × 10⁶ / (2π × 50 × (380/√3)²) 
  = 2175 × 10⁶ / (2π × 50 × 25333) = 68.5 мкФ
  
- При подключении треугольником (Δ): C = 2175 × 10⁶ / (2π × 50 × 380²)
  = 2175 × 10⁶ / (2π × 50 × 144400) = 47.8 мкФ

Выбираем батарею: 3 × 20 мкФ = 60 мкФ (треугольник) или 3 × 25 мкФ = 75 мкФ (звезда)
```

**Практические замечания:**

1. **Сверхкомпенсация недопустима:** Если cos(φ) > 0.95, конденсатор начинает работать как источник реактивной энергии, что ухудшает сеть.
2. **Выбор по стандартным ёмкостям:** Промышленные конденсаторы выпускаются на 5, 10, 20, 30, 40, 50 мкФ.
3. **Защита:** Между конденсатором и сетью должен быть предохранитель (на случай пробоя конденсатора).

| Магистральные дороги | 20–40 | 40–50 | 100 |
| Спортплощадки (тренировка) | 100–200 | 200–300 | 500 |

### Методика 3.1.2: Графо-аналитический метод расчёта

**Используется для однородного распределения света (подвесные светильники прямого света).**

**Формула светового потока:**

```
Φ (лм) = (E × S × K_неравн × K_отр) / η

где:
E — требуемая освещённость (люкс)
S — площадь помещения (м²)
K_неравн — коэффициент неравномерности (0.4–0.6 для потолочных светильников)
K_отр — коэффициент запаса на загрязнение (1.0–1.5)
η — коэффициент полезного действия светильника (0.5–0.85)
```

**Упрощённая формула (для стандартного офиса η=0.7, K_неравн=0.5, K_отр=1.2):**

```
Φ (лм) ≈ E × S (люкс × м²) / 0.42

Пример: E=400 люкс, S=30 м²
Φ = 400 × 30 / 0.42 ≈ 28500 лм
```

**Пример 10: Расчёт светильников для офиса 30 м²**

Дано:
- Офис площадью 30 м²
- Требуемая освещённость: E = 400 люкс (ГОСТ 54944)
- Вид светильника: потолочный светодиодный с КПД 0.7
- Коэффициент неравномерности: K = 0.5
- Коэффициент запаса: K_з = 1.2

Решение:
```
Φ_требуемый = (400 × 30 × 1.2) / 0.7 = 20571 лм

Выбираем светодиодные светильники серии офисной:
- Вариант 1: 10 светильников × 2100 лм = 21000 лм ✓
- Вариант 2: 6 светильников × 3500 лм = 21000 лм ✓

Проверка равномерности освещения:
При потолке h = 2.8 м и сетке 3×3 метра (9 секций):
- Центральные точки: 440–480 люкс
- Угловые точки: 200–280 люкс
- Отношение мин/макс: 280/480 = 0.58 (допустимо, >0.4)
```

```
где 1.2 — коэффициент запаса на грязь, 0.7 — КПД установки
```

### Методика 3.2.1: Естественное + искусственное освещение

**Используется в дневное время (окна + искусственное).**

**Формула:**

```
Φ_искусств = (E_требуемая - E_естественная) × S / η

где E_естественная зависит от:
- Расположение окна (север / юг)
- Время суток
- Загрязнённость стекла
- Глубина помещения относительно окна
```

**Таблица: естественная освещённость на расстоянии от окна**

| Расстояние от окна (м) | E_естеств на северной ориентации | E_естеств на южной ориентации (полдень) |
|---|---|---|
| 0–1 м | 50–100 люкс | 200–500 люкс |
| 1–3 м | 20–50 люкс | 50–150 люкс |
| 3–5 м | 10–20 люкс | 20–50 люкс |
| >5 м | <10 люкс | 10–20 люкс |

**Пример 11: Смешанное освещение офиса с окном**

Дано:
- Офис 20 м² на юго-восток (угловая комната)
- 2 окна по 2 м² (на юге)
- Требуемая E = 400 люкс (стандартный офис)
- В 12:00 солнца достаточно
- На глубину 3 м от окна: E_естеств = 100 люкс (в облачный день)

Решение:
```
Зона рядом с окном (первые 3 м): 
  Φ_искусст = (400 - 100) × (3 × 7) / 0.7 = 300 × 21 / 0.7 = 9000 лм
  Используем 4–5 потолочных светильников 2000 лм
  
Зона глубина 3–6 м:
  E_естеств = 50 люкс → Φ_искусст = (400 - 50) × (3 × 7) / 0.7 = 10500 лм
  Используем 5–6 светильников 2000 лм
  
Итого: ~10 светильников по 2000 лм (как в примере 10, но часть выключается днём)
```

**Практическое решение:** установить датчик освещённости (фотоэлемент), который отключает половину светильников при E > 200 люкс днём. Экономия: ~30–40% электроэнергии.

# 4. Измерения и Сигналы

## 4.1 Линейная интерполяция и масштабирование сигналов

### Методика 4.1.1: Преобразование сигнала (0–10V) → (0–1000°C) — ГОСТ 8.407

**Принцип:** Многие приборы отправляют унифицированные сигналы 0–10V, 4–20mA, которые нужно пересчитать в инженерные единицы.

**Формула линейного масштабирования:**

```
Y = Y_min + (X - X_min) / (X_max - X_min) × (Y_max - Y_min)

где:
X — входной сигнал (В или мА)
Y — выходное значение (инженерные единицы)
X_min, X_max — диапазон входного сигнала
Y_min, Y_max — диапазон выходного сигнала
```

**Пример 12: Преобразование 0–10V в температуру (0–1000°C)**

Дано:
- Выход датчика: 0–10 В
- Диапазон измерения: 0–1000°C
- Текущий сигнал с датчика: U = 5.3 В

Решение:
```
T = 0 + (5.3 - 0) / (10 - 0) × (1000 - 0) = 5.3 / 10 × 1000 = 530°C
```

---

### Методика 4.1.2: Масштабирование 4–20mA (типичный промышленный сигнал)

**Особенность:** 4 мА соответствует минимуму, 20 мА — максимуму (это даёт диагностику обрыва — 0 мА = обрыв провода).

**Формула:**

```
Y = Y_min + (I - 4) / (20 - 4) × (Y_max - Y_min)
Y = Y_min + (I - 4) / 16 × (Y_max - Y_min)
```

**Пример 13: Преобразование 4–20mA в давление (0–10 бар)**

Дано:
- Передатчик давления выдаёт 4–20 мА
- Диапазон: 0–10 бар
- Текущий ток: I = 12.3 мА

Решение:
```
P = 0 + (12.3 - 4) / 16 × 10 = 8.3 / 16 × 10 = 5.19 бар

Проверка:
- При I = 4 мА: P = 0 бар ✓
- При I = 20 мА: P = 10 бар ✓
- При I = 12 мА: P = 5 бар ✓
```

---

### Методика 4.1.3: Кусочно-линейное масштабирование

**Используется когда датчик имеет нелинейность и нужна перекалибровка по нескольким точкам.**

**Метод:** Снять калибровочные пары (X_i, Y_i) для 3–5 контрольных точек, затем интерполировать между ними.

**Формула для интерполяции:**

```
Если X находится между точками (X_i, Y_i) и (X_{i+1}, Y_{i+1}):

Y = Y_i + (X - X_i) × (Y_{i+1} - Y_i) / (X_{i+1} - X_i)
```

**Пример 14: Калибровка нелинейного датчика расхода**

Дано калибровочные точки (напряжение → расход):

| U (В) | Q (л/мин) |
|-------|----------|
| 0 | 0 |
| 2 | 5 |
| 5 | 18 |
| 8 | 40 |
| 10 | 60 |

Нужно найти расход при U = 6.5 В

Решение:
```
U = 6.5 В находится между (5 В, 18 л/мин) и (8 В, 40 л/мин)

Q = 18 + (6.5 - 5) × (40 - 18) / (8 - 5)
Q = 18 + 1.5 × 22 / 3
Q = 18 + 11 = 29 л/мин
```

## 4.2 Расчёт погрешности измерений

### Методика 4.2.1: Погрешность прибора (ГОСТ 8.401)

**Абсолютная погрешность приборов по классам точности:**

```
ΔX = ± (класс точности / 100) × X_max

где X_max — максимум измеряемой величины для этого предела
```

**Таблица классов точности приборов (ГОСТ 8.401):**

| Класс точности | Погрешность (%) | Типичное применение |
|---|---|---|
| 0.1 | ±0.1% | Лаборатория, эталонные приборы |
| 0.2 | ±0.2% | Точные промышленные измерения |
| 0.5 | ±0.5% | Амперметры, вольтметры в щитках |
| 1.0 | ±1% | Обычные учебные приборы |
| 1.5 | ±1.5% | Манометры, температурные датчики |
| 2.5 | ±2.5% | Бюджетные датчики |

**Пример 15: Расчёт погрешности амперметра**

Дано:
- Амперметр класса 0.5, диапазон 0–5 А
- Текущий ток: I_изм = 4.2 А
- Требуется найти истинное значение

Решение:
```
ΔI = ±(0.5 / 100) × 5 = ±0.025 А

Истинное значение: I_истинн = 4.2 ± 0.025 А
или I_истинн ∈ [4.175; 4.225] А

Относительная погрешность:
δ = ΔI / I_изм × 100% = 0.025 / 4.2 × 100% = 0.6%
```

---

### Методика 4.2.2: Суммарная погрешность сложных измерений (ГОСТ 8.407)

**Когда величину находят через несколько промежуточных измерений:**

```
Если Y = f(X₁, X₂, ..., Xₙ), то:

ΔY = √[(∂Y/∂X₁ × ΔX₁)² + (∂Y/∂X₂ × ΔX₂)² + ... + (∂Y/∂Xₙ × ΔXₙ)²]
```

**Пример 16: Погрешность расчёта мощности по U и I**

Дано:
- Вольтметр класса 0.5, диапазон 0–300 В, показание U = 230 В
- Амперметр класса 0.5, диапазон 0–10 А, показание I = 8.5 А
- cos(φ) известен точно = 0.95

Найти: P и её погрешность

Решение:
```
P = U × I × cos(φ) = 230 × 8.5 × 0.95 = 1857 Вт

Абсолютные погрешности приборов:
ΔU = ±(0.5 / 100) × 300 = ±1.5 В
ΔI = ±(0.5 / 100) × 10 = ±0.05 А

Частные производные:
∂P/∂U = I × cos(φ) = 8.5 × 0.95 = 8.075
∂P/∂I = U × cos(φ) = 230 × 0.95 = 218.5

Суммарная погрешность:
ΔP = √[(8.075 × 1.5)² + (218.5 × 0.05)²]
ΔP = √[(12.11)² + (10.93)²]
ΔP = √[146.7 + 119.5] = √266.2 = 16.3 Вт

P = 1857 ± 16 Вт (погрешность 0.9%)
```

## 4.3 Обработка цифровых сигналов и фильтрация

### Методика 4.3.1: Цифровой фильтр нижних частот (низкочастотный фильтр) — экспоненциальное сглаживание

**Используется для подавления шумов в цифровых датчиках.**

**Формула экспоненциального сглаживания (фильтр первого порядка):**

```
Y_n = α × X_n + (1 - α) × Y_{n-1}

где:
X_n — текущее (зашумленное) измерение
Y_n — отфильтрованное значение
Y_{n-1} — предыдущее отфильтрованное значение
α — коэффициент фильтрации (0 < α < 1, обычно 0.05–0.3)

Чем меньше α, тем больше подавляются колебания
(но медленнее отклик на реальные изменения)
```

**Рекомендуемые значения α:**

| Вид датчика | Типичный α | Постоянная времени |
|---|---|---|
| Быстрый датчик (типа термопары, без инерции) | 0.2–0.4 | 1–3 сек |
| Датчик давления | 0.1–0.2 | 3–10 сек |
| Датчик уровня жидкости | 0.05–0.1 | 10–30 сек |
| Медленный датчик (с инерцией) | 0.01–0.05 | 30+ сек |

**Пример 17: Сглаживание зашумленного датчика температуры**

Дано:
- Датчик выдаёт шумные значения (±2°C шум)
- Примем α = 0.15 (для среднего подавления шумов)
- Предыдущее отфильтрованное значение: Y₀ = 25.0°C

Последовательность измерений:

| n | X_n (сырое) | Y_n (отфильтрованное) | Шум |
|---|---|---|---|
| 0 | — | 25.0 | — |
| 1 | 25.8 | 0.15×25.8 + 0.85×25.0 = 25.12 | +0.8°C |
| 2 | 25.1 | 0.15×25.1 + 0.85×25.12 = 25.11 | −0.9°C |
| 3 | 27.2 | 0.15×27.2 + 0.85×25.11 = 25.71 | +2.0°C |
| 4 | 26.5 | 0.15×26.5 + 0.85×25.71 = 25.88 | +0.6°C |
| 5 | 26.0 | 0.15×26.0 + 0.85×25.88 = 25.92 | +0.1°C |

**Вывод:** Фильтр подавил колебания с амплитудой ±2°C в медленное поднятие из 25°C в 26°C.

---

### Методика 4.3.2: Скользящее среднее (Moving Average) — для удаления импульсных шумов

**Используется когда шум кратковременный (спайк).**

**Формула:**

```
Y_n = (X_n + X_{n-1} + ... + X_{n-k+1}) / k

где k — окно усреднения (обычно 3–10 значений)
```

**Пример 18: Удаление импульсного шума датчика тока**

Дано:
- Датчик выдаёт значение каждые 100 мс
- Ожидаемый ток: 10 А (стабилен)
- Окно усреднения k = 5

Последовательность значений:

| n | X_n (сырое) | Y_n (среднее за 5) |
|---|---|---|
| 1 | 10.1 | — |
| 2 | 9.95 | — |
| 3 | 10.05 | 10.03 |
| 4 | 25.3 | 10.07 (спайк почти не влияет!) |
| 5 | 10.0 | 10.98 (начинает уходить вверх) |
| 6 | 10.2 | 10.82 |
| 7 | 9.95 | 10.17 |
| 8 | 10.1 | 10.05 (вернулось в норму) |

**Вывод:** Спайк на n=4 (25.3 А) был практически подавлен и не вызвал срабатывание защиты.

# 5. Вентиляция и Отопление

## 5.1 Расчёт воздухообмена

### Методика 5.1.1: По интенсивности воздухообмена (СП 60.13330.2020, ГОСТ 30494)

**Принцип:** Определить объём воздуха, который должен пройти через помещение за час.

**Таблица нормативной интенсивности воздухообмена:**

| Тип помещения | Минимум кратности | Воздухообмен (м³/ч на м²) | Минимум свежего воздуха (м³/ч на человека) |
|---|---|---|---|
| **Жилые помещения:** | | | |
| Спальня | 0.5 | — | 25–30 |
| Гостиная | 1.0 | — | 30–40 |
| Кухня с электроплитой | 3.0 | 8–12 | 60–90 |
| Кухня с газовой плитой | 5.0 | 15–20 | 100–150 |
| Ванная комната | 3.0 | — | 50–75 |
| Санузел | 3.0 | — | 30–50 |
| **Офисные помещения:** | | | |
| Офис, класс "А" | 1.0–1.5 | — | 40–60 |
| Офис, общий | 1.5–2.0 | — | 50–70 |
| Конференц-зал | 2.0–3.0 | — | 80–100 |
| **Производственные:** | | | |
| Цех общего назначения | 1.5–2.0 | — | 30–50 |
| Цех с выделением вредных газов | 3.0–5.0 | — | 50–150 |

**Формула расчёта необходимого объёма воздуха:**

```
Вариант 1 — по кратности воздухообмена:
L (м³/ч) = V_помещения (м³) × K

где K — кратность (сколько раз в час воздух полностью меняется)

Пример: спальня 20 м² × 3 м = 60 м³, K = 0.5
L = 60 × 0.5 = 30 м³/ч

Вариант 2 — по числу людей:
L = n_людей × l_норма (м³/ч на одного человека)

Пример: офис 20 м² с 4 людьми, норма 50 м³/ч на чел
L = 4 × 50 = 200 м³/ч

Вариант 3 — по площади помещения:
L = S × удельный расход (м³/ч на м²)

Пример: кухня 15 м², норма 10 м³/ч на м²
L = 15 × 10 = 150 м³/ч
```

**Окончательный выбор — берётся МАКСИМУМ из всех методов:**

```
L_выбора = MAX(L_по_кратности, L_по_людям, L_по_площади)
```

**Пример 19: Расчёт вентиляции квартиры 50 м²**

Дано:
- Квартира: 2 спальни (12 м² + 14 м²), гостиная (15 м²), кухня (9 м²)
- 3 человека постоянно
- Высота потолков: 3 м

Решение:

| Помещение | Объём | Метод | Расход |
|---|---|---|---|
| Спальня 1 | 36 м³ | K=0.5 | 18 м³/ч |
| Спальня 2 | 42 м³ | K=0.5 | 21 м³/ч |
| Гостиная | 45 м³ | K=1.0 | 45 м³/ч |
| Кухня | 27 м³ | K=3.0 | 81 м³/ч |
| **По людям** | — | 3 чел × 40 м³/ч | **120 м³/ч** |

**Выбор:** L = MAX(18, 21, 45, 81, 120) = **120 м³/ч**

Выбираем вентилятор на 120–150 м³/ч (с запасом 25%).

### Методика 5.1.2: Расчёт скорости воздуха в воздуховодах (ПУЭ + ГОСТ 12.1.006)

**Ограничение по скорости (для снижения шума и сопротивления):**

```
V = L / F

где:
L — расход воздуха (м³/ч)
F — площадь сечения воздухопровода (м²)
V — средняя скорость (м/с)
```

**Нормативные скорости воздуха (ГОСТ 12.1.006):**

| Назначение воздухопровода | Макс. скорость (м/с) | Причина ограничения |
|---|---|---|
| Жилые помещения (приточный) | 3–4 | Шум < 30 дБ |
| Жилые помещения (вытяжной) | 4–5 | Сквозняков нет |
| Офисы | 4–5 | Комфорт |
| Производство (общее) | 6–8 | Производительность |
| Производство (с пылью) | 10–15 | Унос пыли |
| Специальные каналы | 15–20 | Специфика процесса |

**Таблица площадей сечений труб (круглые воздуховоды):**

| Диаметр (мм) | Площадь (м²) | При L=200 м³/ч, V макс |
|---|---|---|
| 100 | 0.00785 | V = 200/3600/0.00785 = 7.1 м/с |
| 125 | 0.0123 | V = 4.5 м/с |
| 160 | 0.0201 | V = 2.8 м/с |
| 200 | 0.0314 | V = 1.8 м/с |

**Пример 20: Выбор диаметра воздухопровода**

Дано:
- Требуемый расход: L = 200 м³/ч
- Максимально допустимая скорость: V = 4.5 м/с (жилое помещение)

Решение:
```
L (м³/с) = 200 / 3600 = 0.0556 м³/с

F = L / V = 0.0556 / 4.5 = 0.0123 м²

Из таблицы: диаметр 125 мм (F = 0.0123 м²)

Проверка: V = 0.0556 / 0.0123 = 4.5 м/с ✓

Выбираем гибкую трубу или жёсткую трубу диаметром 125 мм.
```

### Методика 5.1.3: Расчёт потерь давления в воздуховодах (Darcy-Weisbach)

**Используется для подбора мощности вентилятора.**

**Формула потерь давления:**

```
ΔP = λ × (L / D) × (ρ × V²) / 2

где:
λ — коэффициент трения (зависит от шероховатости и числа Рейнольдса)
L — длина воздухопровода (м)
D — диаметр (м)
ρ — плотность воздуха (1.2 кг/м³ при +20°C)
V — скорость воздуха (м/с)

Результат в Па (паскалях)
```

**Упрощённая формула для круглых воздуховодов из оцинкованной стали:**

```
ΔP (Па/м) ≈ 0.5 × (L_м³_ч / 1000)^1.9 / D_мм^4.7

Пример: L = 200 м³/ч, D = 125 мм
ΔP/м = 0.5 × (0.2)^1.9 / (125)^4.7 ≈ 0.8 Па/м
```

**Таблица предварительных потерь давления:**

| Расход (м³/ч) | Диаметр 100 мм | Диаметр 125 мм | Диаметр 160 мм | Диаметр 200 мм |
|---|---|---|---|---|
| 50 | 0.1 Па/м | 0.03 Па/м | — | — |
| 100 | 0.4 Па/м | 0.1 Па/м | 0.04 Па/м | — |
| 200 | 1.5 Па/м | 0.4 Па/м | 0.15 Па/м | 0.05 Па/м |
| 500 | 9 Па/м | 2 Па/м | 0.7 Па/м | 0.2 Па/м |

**Пример 21: Расчёт требуемого давления вентилятора**

Дано:
- Система вентиляции: 200 м³/ч, диаметр трубы 125 мм
- Длина воздухопровода: 15 м (прямая + отвод)
- Воздухозаборник (входное сопротивление): 20 Па
- Выходная решётка: 15 Па
- Фильтр: 50 Па (при нормальной загрязнённости)

Решение:
```
ΔP_трубы = 0.4 Па/м × 15 м = 6 Па

Суммарное давление:
P_общее = 20 + 6 + 50 + 15 = 91 Па ≈ 100 Па

Выбираем вентилятор: номинальный напор 100–120 Па
(канальный вентилятор типоразмера 125 мм, например, 400 об/мин)
```

## 5.2 Расчёт отопления

### Методика 5.2.1: Расчёт тепловой мощности по площади (СП 50.13330)

**Эмпирическая метода для предварительного расчёта (используется до детального анализа).**

**Таблица удельной тепловой мощности на квадратный метр:**

| Тип здания | Удельная мощность (Вт/м²) | Примечание |
|---|---|---|
| **Новые здания (2000+) с утеплением** | | |
| Квартира, панель утёплённая | 60–80 | Оптимальный уровень |
| Офис класса "А" | 70–90 | С кондиционированием |
| Частный дом модернизированный | 80–100 | С теплотрассой |
| **Старые здания (1970–1990)** | | |
| Панельный дом | 100–150 | Низкая теплоизоляция |
| Кирпичное здание | 120–180 | Хуже, чем панель |
| **Малоизолированные (1950–1970)** | | |
| Деревянный дом | 150–250 | Очень плохая изоляция |
| Блочное здание | 200–300 | Требует утепления |

**Расчёт тепловой нагрузки:**

```
Q (кВт) = S × q_удельное

где S — отапливаемая площадь (м²), q_удельное из таблицы (Вт/м²)

Примеры:
- Квартира 70 м² в панельном доме (новом): Q = 70 × 75 = 5.25 кВт
- Офис 200 м² класса "А": Q = 200 × 80 = 16 кВт
- Частный дом 150 м² утеплённый: Q = 150 × 90 = 13.5 кВт
```

**Добавки к расчётной мощности:**

| Фактор | +K (%) |
|---|---|
| Угловые комнаты (доп. стена на холод) | +10% |
| Панорамные окна (южный фасад) | −10–20% |
| Очень холодный климат (< −30°C средней зимы) | +20–30% |
| Высокие потолки (> 3 м) | +10% за каждые 0.5 м |

**Пример 22: Расчёт мощности котла для квартиры**

Дано:
- Квартира в панельном доме (2000 г.), Москва
- Площадь: 70 м²
- Конструкция: панель + пластиковые окна
- Этаж: 3-й, средний (не угловая)

Решение:
```
Базовая мощность (панель): Q_0 = 70 × 75 Вт/м² = 5.25 кВт

Поправки:
- Угловая комната: нет (−0%)
- Панорамные окна: нет (0%)
- Московский климат: стандартный (0%)
- Потолки 2.7 м: стандартные (0%)

Итого: Q = 5.25 кВт

Выбираем котёл: 7 кВт (с запасом 25% для потерь и переменных нагрузок)
```

### Методика 5.2.2: Расчёт по теплопотерям через ограждающие конструкции (детальный)

**Точный расчёт теплопотерь через стены, окна, крышу (ГОСТ 30494).**

**Формула:**

```
Q (Вт) = Σ (A_i × U_i × ΔT)

где:
A_i — площадь i-й конструкции (м²)
U_i — коэффициент теплопередачи конструкции (Вт/м²·К)
ΔT — разность температур внутри и снаружи (К)
Σ — сумма по всем конструкциям (стены, окна, двери, пол, потолок)
```

**Таблица коэффициентов теплопередачи (U) для стандартных конструкций:**

| Конструкция | U (Вт/м²·К) |
|---|---|
| **Окна** | |
| Одинарное стекло | 5.0–5.8 |
| Двойное стекло (2 стекла) | 2.8–3.2 |
| Тройное стекло + энергосбережение | 1.2–1.6 |
| Четырёхкамерный стеклопакет (новый) | 0.8–1.0 |
| **Стены** | |
| Деревянная (не утеплённая) | 1.0–1.5 |
| Кирпичная (не утеплённая, 2.5 кирпича) | 1.2–1.5 |
| Панельная ЖБИ (не утеплённая) | 0.8–1.2 |
| Панель с ЭППС 50 мм (утепление) | 0.4–0.5 |
| Кирпич + пена 100 мм | 0.3–0.4 |
| **Крыша / Чердак** | |
| Без утепления | 1.5–3.0 |
| С минватой 100 мм | 0.5–0.7 |
| С минватой 200 мм | 0.3–0.4 |
| **Полы** | |
| На земле (не утеплённый) | 0.3–0.5 |
| На земле + утепление | 0.1–0.2 |
| На лаге с вентиляцией | 0.4–0.6 |

**Пример 23: Детальный расчёт теплопотерь для комнаты 4 × 5 м**

Дано:
- Комната на первом этаже, угловая (две стены наружу)
- Высота потолков: 2.7 м
- Внутренняя температура: +20°C
- Температура на улице: −20°C (расчётная для Москвы)
- ΔT = 40 К

Конструкции:

| Конструкция | Площадь (м²) | U (Вт/м²·К) | Q (Вт) |
|---|---|---|---|
| **Две наружные стены** | | | |
| Стена 1 (южная, панель без утепл.) | 4 × 2.7 = 10.8 | 1.0 | 432 |
| Стена 2 (западная, панель) | 5 × 2.7 = 13.5 | 1.0 | 540 |
| **Окна (2 шт)** | | | |
| Окна двойные (в панели) | 2 × 1.5 = 3 | 2.8 | 336 |
| **Потолок (чердак выше)** | | | |
| Потолок панельный (чердак не утеп.) | 4 × 5 = 20 | 1.2 | 960 |
| **Пол (на подвалом ниже)** | | | |
| Пол панельный (подвал холодный) | 4 × 5 = 20 | 0.6 | 480 |
| **Дверь наружная** | | | |
| Дверь деревянная | 2.1 × 0.8 = 1.7 | 3.0 | 204 |
| | | **Σ Q =** | **2952 Вт** |

**Итого:** теплопотери = **3 кВт** (для комнаты 20 м²)

Это соответствует удельной мощности ~150 Вт/м² (кирпичное, старое здание).

### Методика 5.2.3: Расчёт производительности радиатора

**Подбор количества секций радиатора или мощности конвектора.**

**Базовая формула:**

```
Мощность одной секции при ΔT_среднего = 70 К (температура подачи 90°C, обратка 70°C, воздух 20°C):

Q_сек (Вт) = K_тип × 1700 Вт

где K_тип:
- Чугунный классический: 1.0 (170 Вт/сек)
- Алюминиевый современный: 1.5–2.0 (255–340 Вт/сек)
- Биметаллический: 1.8–2.2 (306–374 Вт/сек)
```

**Коррекция на реальный ΔT:**

```
Q_реальная = Q_сек × ((ΔT_реальн / 70)^1.3)

где ΔT_реальн — средняя температура (T_подача + T_обратка) / 2 − T_воздуха
```

**Таблица производительности типовых радиаторов (при ΔT = 70K):**

| Тип | Одна секция (Вт) | Типичный размер | Мощность |
|---|---|---|---|
| Чугунный М-140 | 170 | 8 сек | 1360 Вт |
| Алюминиевый Esperado | 204 | 10 сек | 2040 Вт |
| Биметаллический Royal | 340 | 6 сек | 2040 Вт |
| Конвектор плинтусный | 300–500 Вт (целый) | 2–3 м длины | 1500–2500 Вт |

**Пример 24: Подбор радиатора для комнаты**

Дано:
- Теплопотери комнаты: Q_потерь = 2952 Вт ≈ 3 кВт (из примера 23)
- Система отопления: подача 80°C, обратка 60°C, воздух 18°C
- ΔT_среднего = ((80 + 60) / 2) − 18 = 52 К

Решение:
```
Чугунный радиатор М-140:
Q_сек_база = 170 Вт (при ΔT = 70K)
Q_сек_реальная = 170 × (52 / 70)^1.3 = 170 × 0.57 = 97 Вт

Требуемое количество секций: N = 3000 / 97 = 31 сек

Берём 32 сек = 4 радиатора × 8 сек (стандартный размер)

Альтернатива — биметаллический (меньше, красивее):
Q_сек_база = 340 Вт
Q_сек_реальная = 340 × (52 / 70)^1.3 = 340 × 0.57 = 194 Вт
N = 3000 / 194 = 15.5 сек = 16 сек = 2 радиатора × 8 сек
```

## 5.3 Совместный расчёт вентиляции и отопления

### Методика 5.3.1: Координация вентиляции с потерей тепла (ГОСТ + СП)

**Проблема:** вентиляция уносит тепло из помещения. Это нужно учитывать в расчёте котла.

**Формула дополнительных потерь тепла на вентиляцию:**

```
Q_вент (Вт) = L (м³/ч) × Δρ × c × ΔT / 3600

где:
L — расход воздуха (м³/ч)
Δρ — плотность воздуха (1.2 кг/м³)
c — теплоёмкость воздуха (1000 Дж/кг·К)
ΔT — разница температур (К)
3600 — пересчёт из часов в секунды

Упрощённо:
Q_вент (Вт) ≈ L (м³/ч) × 0.33 × ΔT (К)
```

**Пример 25: Учёт вентиляции в расчёте котла**

Дано:
- Квартира 70 м², требуемый расход воздуха L = 120 м³/ч (из примера 19)
- Базовые теплопотери (через ограждения): Q_огр = 5250 Вт
- ΔT = 40 К (внутри +20°C, снаружи −20°C)

Решение:
```
Потери на вентиляцию:
Q_вент = 120 × 0.33 × 40 = 1584 Вт ≈ 1.6 кВт

Суммарные потери:
Q_итого = 5250 + 1584 = 6834 Вт ≈ 7 кВт

Выбираем котёл: 10 кВт (с запасом 45% на пиковые нагрузки)
```

**Вывод:** Вентиляция добавляет ~30% к расходу тепла!

---

### Методика 5.3.2: Рекуперация тепла (heat recovery ventilation — HRV)

**Решение проблемы избыточных теплопотерь:**

Рекуператор позволяет вернуть часть тепла из вытяжного воздуха в приточный.

**КПД рекуператора (η_rec):**

```
Типовые значения:
- Пластинчатые (статические): η = 50–70%
- С энтальпийными мембранами: η = 65–85%
- С промежуточным теплоносителем: η = 70–90%
```

**Эффективные теплопотери с рекуператором:**

```
Q_вент_эффект = Q_вент × (1 − η_rec)

Пример (из примера 25):
Q_вент_эффект = 1584 × (1 − 0.70) = 1584 × 0.30 = 475 Вт

Новый расход котла:
Q_итого = 5250 + 475 = 5725 Вт ≈ 6.5 кВт

Экономия: (1584 − 475) / 1584 = 70% (на вентиляции)
Экономия по дому: (6834 − 5725) / 6834 = 16% (на всей отопительной нагрузке)
```
