§5.1Общие положения

Расчёт нежёстких дорожных одежд на прочность выполняется по трём независимым критериям. Конструкция считается прочной, если все три условия выполняются одновременно.

Критерии прочности п. 5.1.1
Рис. 1 — Расчётная схема нежёсткой дорожной одежды

Три критерия прочности, которые должны выполняться одновременно:

Критерий 1 (п. 5.1.4): Расчётный общий модуль упругости не должен быть ниже требуемого:
Eрасч ≥ Kп · Eтр
Критерий 2 (п. 5.1.5): Активное напряжение сдвига не должно превышать допускаемое в грунте и в песчаном слое:
Tдоп ≥ Kп · Tр
Критерий 3 (п. 5.1.6): Растягивающее напряжение в монолитных а/б слоях не должно превышать предельно допустимое с учётом усталости:
RN ≥ Kп · σr
Примечание: Коэффициент прочности Kп определяется в зависимости от уровня надёжности Kн по Табл. В.2 Прил. В. Для дорог I–II категории Kн = 0.95, Kп = 1.20.

АЛГОРИТМПошаговый порядок расчёта

Полный алгоритм расчёта нежёсткой дорожной одежды на прочность по СП РК 3.03-104-2014:

  1. Исходные данные. Задать категорию дороги, тип одежды, группу расчётной нагрузки (А1/А2), ДКЗ, срок службы T, уровень надёжности Kн.
  2. Транспортная нагрузка Np. По форм. (6): собрать состав потока, задать коэффициенты полосности и приведения → получить Np, авт/сут.
  3. Суммарная нагрузка ΣNp. По форм. (8): с учётом роста интенсивности q и числа расчётных дней nр → получить ΣNp за T лет.
  4. Требуемый модуль Eтр. По форм. (9): Eтр = A + B·lg(ΣNp·C). Сравнить с минимальными значениями по Табл. 4.
  5. Конструкция одежды. Принять или проверить толщины и материалы слоёв. Назначить Ei, hi из Прил. Б по расчётной температуре (+10°C для Крит. 1).
  6. Критерий 1 (прогиб). Послойным приведением снизу вверх по Рис. 1 вычислить Eрасч. Проверить: Eрасч ≥ Kп·Eтр.
  7. Критерий 2 (сдвиг). При температуре +30…+50°C вычислить Tр по форм. (10). Вычислить Tдоп по форм. (11) для грунта и для песка. Проверить оба условия.
  8. Интенсивность последнего года Nt. По форм. (12): Nt = Np·nр·qT−1 — для расчёта коэффициента усталости Kу.
  9. Критерий 3 (изгиб). При низких температурах вычислить σr по форм. (14). Вычислить Kу по форм. (16). Вычислить RN по форм. (15). Проверить: RN ≥ Kп·σr.
  10. Итог. Если все три критерия выполнены → конструкция принята. Иначе → изменить толщины или материалы слоёв и повторить расчёт с шага 5.
