## МЕТОДИКИ ЗЕМЛЯНЫХ РАБОТ

### 1. РАСЧЁТ ОБЪЁМОВ ЗЕМЛЯНЫХ МАСС

#### 1.1 Метод срезов (метод горизонтальных слоёв)

**Применение:** Для территорий с неправильным рельефом, выемок, насыпей.

**Принцип:** Участок разбивается на горизонтальные слои высоты h; объём слоя = среднее арифметическое площадей основания и вершины × высота слоя.

**Формула:**
```
V = Σ(h/3 × (S_i + S_{i+1} + √(S_i × S_{i+1})))  для неправильной формы
или
V = h × (S₁ + S₂)/2  для прямого слоя между двумя параллельными поверхностями
```

где:
- h — высота слоя [м]
- S_i, S_{i+1} — площадь нижнего и верхнего основания слоя [м²]

**Пример расчёта выемки:**
```
Выемка прямоугольной формы 50 × 30 м, глубина 2 м
V = 50 × 30 × 2 = 3000 м³

Выемка переменной глубины (глубины на углах: 1.8, 2.0, 2.1, 1.9 м)
Средняя глубина = (1.8 + 2.0 + 2.1 + 1.9)/4 = 1.95 м
V = 50 × 30 × 1.95 = 2925 м³
```

#### 1.2 Метод треугольников (для нерегулярной сетки)

**Применение:** Когда участок разбит на треугольные элементы; используется для сложного рельефа.

**Принцип:** Каждый треугольник образует призму; объём призмы = площадь основания × среднее значение высот трёх вершин.

**Формула:**
```
V_треугольника = S_основ × (h₁ + h₂ + h₃)/3

где:
- S_основ — площадь треугольника в плане [м²]
- h₁, h₂, h₃ — высоты вершин треугольника (отметки) [м]

Площадь треугольника в плане:
S = 0.5 × |x₁(y₂ - y₃) + x₂(y₃ - y₁) + x₃(y₁ - y₂)|
```

**Пример с TIN (Triangulated Irregular Network):**
```
Треугольник с вершинами:
A(0, 0, h=2.5 м), B(10, 0, h=2.8 м), C(5, 8, h=3.1 м)

S_основ = 0.5 × |0×(0-8) + 10×(8-0) + 5×(0-0)| = 0.5 × 80 = 40 м²
V = 40 × (2.5 + 2.8 + 3.1)/3 = 40 × 2.8 = 112 м³
```

#### 1.3 Метод призм (для регулярной сетки)

**Применение:** Когда участок разбит на прямоугольные элементы равного размера (ортогональная сетка).

**Принцип:** Каждая ячейка сетки образует призму; объём = площадь ячейки × среднее значение высот четырёх углов.

**Формула:**
```
V_ячейки = a × b × (h₁ + h₂ + h₃ + h₄)/4

где:
- a, b — размеры ячейки в плане [м]
- h₁, h₂, h₃, h₄ — высоты углов ячейки [м]

Общий объём участка:
V_полный = Σ V_ячейки по всем ячейкам
```

**Пример для регулярной сетки 20×20 м:**
```
Участок 50×50 м, разбит на сетку 10 ячеек (по 5 ячеек в каждом направлении)
Размер ячейки: 10×10 м

Отметки вершин (сетка 6×6):
h = [
  2.5, 2.6, 2.7, 2.8, 2.9, 3.0,
  2.6, 2.7, 2.8, 2.9, 3.0, 3.1,
  2.7, 2.8, 2.9, 3.0, 3.1, 3.2,
  2.8, 2.9, 3.0, 3.1, 3.2, 3.3,
  2.9, 3.0, 3.1, 3.2, 3.3, 3.4,
  3.0, 3.1, 3.2, 3.3, 3.4, 3.5
]

Первая ячейка (h₁=2.5, h₂=2.6, h₃=2.6, h₄=2.7):
V = 10 × 10 × (2.5 + 2.6 + 2.6 + 2.7)/4 = 100 × 2.6 = 260 м³

Полный объём = Σ (суммирование по 25 ячейкам)
```

### 2. БАЛАНС ЗЕМЛЯНЫХ МАСС

**Назначение:** Определение соотношения между объёмом выемки и насыпи для оптимизации процесса и затрат.

**Формула баланса:**
```
V_выемка ≥ V_насыпь + V_уплотнение

где V_уплотнение = V_насыпь × (K_разр - 1)
```

**Базовая формула:**
```
Баланс земляных масс = V_выемка - V_насыпь (в естественном состоянии)

Если баланс > 0: избыток грунта (вывоз)
Если баланс < 0: дефицит грунта (подвоз)
Если баланс ≈ 0: грунт в equilibrium (оптимально)
```

**Расширенный расчёт с коэффициентом разрыхления:**
```
V_выемки_раздр = V_выемка × K_разр  где K_разр ≈ 1.1–1.4 (см. таблица 6)

V_надо_для_насыпи = V_насыпь / K_уплотн  где K_уплотн ≈ 0.8–0.9 (зависит от метода)

Баланс с учётом разрыхления:
ΔV = V_выемки_раздр - V_надо_для_насыпи

Если ΔV > 0: излишек раздробленного грунта
Если ΔV < 0: требуется дополнительный привоз
```

**Пример расчёта:**
```
Участок 100×100 м:
- Выемка: V_выемка = 5000 м³ (в естественном состоянии)
- Насыпь: V_насыпь = 3000 м³

Грунт: супесь, K_разр = 1.15

V_выемки_раздр = 5000 × 1.15 = 5750 м³
V_надо_для_насыпи = 3000 / 0.85 = 3529 м³ (K_уплотн = 0.85 для супеси)

Баланс = 5750 - 3529 = 2221 м³ (излишек для вывоза)
```

### 3. МЕТОДИКА ОПРЕДЕЛЕНИЯ КОЭФФИЦИЕНТА РАЗРЫХЛЕНИЯ

**Определение:** Отношение объёма грунта в разрыхленном состоянии к объёму в естественном состоянии (в скале).

**Физический смысл:** При разработке грунта между частицами появляются пустоты (пористость увеличивается).

**Формула:**
```
K_разр = V_разр / V_естеств = 1 + ε / (1 - ε)

где ε — коэффициент пустотности (доля пустот в общем объёме)
```

Или эмпирическая формула через плотности:
```
K_разр = ρ_естеств / ρ_разр

где ρ — объёмная плотность грунта [кг/м³]
```

## КООРДИНАТНЫЕ СИСТЕМЫ И ПРИВЯЗКА

### 1. СИСТЕМЫ КООРДИНАТ В ГЕОДЕЗИИ

#### 1.1 Плоские прямоугольные системы (СК-42, СК-95, МСК)

**СК-42 (Система координат 1942 года) — УСТАРЕВШАЯ**
```
Основа: эллипсоид Крассовского
Зоны: 6-градусные, нумерация от Гринвича
Пример: Зона 36 (37°–43° в.д., Москва)
Сдвиг по X (север): 0–6 850 км (от экватора)
Сдвиг по Y (восток): добавка 500 км к истинным координатам (избежать отрицательных)
```

**СК-95 (Система координат 1995 года) — ТЕКУЩИЙ СТАНДАРТ РФ**
```
Основа: эллипсоид ПЗ-90
Зоны: 6-градусные (как в СК-42, но разные координаты)
Проекция Гаусса–Крюгера (равноугольная коническая)
Добавка по Y: +500 км (в каждой зоне)
Пример координат Москвы (Зона 37):
X = 6 363 км (от экватора на север)
Y = 407 км (в зоне 37, с добавкой 500 км)

Абсолютная точность: ±1–5 см в плане
```

**WGS84 (ГЛОБАЛЬНАЯ, используется в GPS)**
```
Основа: эллипсоид WGS84
Координаты: широта (φ) и долгота (λ) в градусах
Пример: Москва φ = 55.75°N, λ = 37.62°E
Точность GPS: ±5–10 м (обычно), ±0.5–1 м (дифференциальный GPS)
```

#### 1.2 Взаимное преобразование координат

**СК-95 → WGS84 (упрощённая формула):**
```
Из плоских координат (X, Y, зона Z) в географические (φ, λ):

1. Определить центральный меридиан зоны:
   λ₀ = 3° + 6° × (Z - 1)  где Z ∈ [1, 60]
   
2. Исправить Y (удалить сдвиг +500):
   y = Y - 500000  [м]
   
3. Вычислить поправки (используя таблицы или калькулятор):
   Δφ ≈ (y / ρ) × (1 - e² cos²φ)  [рад, где ρ = 6371000 м]
   Δλ ≈ (x / (N cos φ))  [рад, где N — радиус кривизны]
   
4. Географические координаты:
   φ = φ₀ + Δφ  [рад → градусы × 180/π]
   λ = λ₀ + Δλ  [рад → градусы × 180/π]
```

**Обратная трансформация (WGS84 → СК-95):**
```
1. Выбрать зону по долготе: Z = floor((λ + 3) / 6) + 1
2. λ₀ = 3° + 6° × (Z - 1)
3. Δλ = λ - λ₀  [в радианах]
4. Вычислить смещения:
   y ≈ N cos φ × Δλ + 500000  [м]
   x ≈ (ρ / (1 - e² cos²φ)) × φ
5. Результат: (X, Y, Zone)
```

**Практический пример:**
```
Дано: WGS84 координаты точки: φ = 55.7558°N, λ = 37.6173°E

Определяем зону: Z = floor((37.6173 + 3) / 6) + 1 = floor(6.77) + 1 = 37
Центральный меридиан: λ₀ = 3° + 6°×36 = 219° = -141° (или 219° в.д.)

Хм, ошибка выше. Правильно:
Z = floor((37.6173 + 3) / 6) + 1 = floor(40.6173 / 6) + 1 = floor(6.77) + 1 = 7+1 = 8  ❌

Заново: для λ = 37°, Z = (37 / 6) + 1 = 7.17 → Zone 37 (стандарт)
```

### 2. ПРИВЯЗКА ОБЪЕКТОВ К КООРДИНАТАМ

**Определение:** Процесс определения абсолютных координат (X, Y, Z) расположения объекта в пространстве относительно государственной или локальной системы отсчёта.

#### 2.1 Методы привязки

| Метод | Точность | Применение | Примечание |
|-------|----------|-----------|-----------|
| **GPS (ГЛОНАСС)** | ±5–10 м | Предварительная привязка, разбивка | Дифференциальный GPS: ±0.5–1 м |
| **Теодолит + рулетка** | ±0.05–0.1 м | Точная разбивка, строительство | Требует видимости, измерения углов |
| **Нивелир** | ±0.01–0.1 м | Определение отметок (Z) | Для высотной привязки |
| **Полярные координаты** | ±0.1 м | От существующего пункта | Если дальнее видно |
| **Ортогональные координаты** | ±0.05 м | От осей теодолита | Точная разбивка |

#### 2.2 Полярный способ привязки

**Суть:** Объект определяется расстоянием и углом от известной точки (опорного пункта).

**Формулы:**
```
От пункта O с координатами (X₀, Y₀) виден объект A.
Измерены:
- Расстояние r = |OA| [м]
- Горизонтальный угол α (от опорного направления) [°]
- Угол наклона ν (высотный угол) [°]

Координаты объекта A:
X_A = X₀ + r × cos(α)
Y_A = Y₀ + r × sin(α)
Z_A = Z₀ + r × tg(ν)

где ν — угол наклона линии визирования
```

**Пример:**
```
Опорный пункт (маяк): O (X=1000, Y=2000, Z=150)
Точка A на расстоянии 150 м под углом 45° с высотным углом ν = 2°

X_A = 1000 + 150 × cos(45°) = 1000 + 150 × 0.707 = 1106 м
Y_A = 2000 + 150 × sin(45°) = 2000 + 150 × 0.707 = 2106 м
Z_A = 150 + 150 × tg(2°) = 150 + 150 × 0.035 = 155.25 м

Координаты: A(1106, 2106, 155.25)
```

#### 2.3 Ортогональный способ привязки (для строительства)

**Суть:** Объект определяется перпендикулярными расстояниями от двух взаимно перпендикулярных осей (бази́сов).

**Формулы:**
```
Базис AB с координатами A(X₀, Y₀) и B(X₁, Y₁)
Направление AB — основная ось X (ось базиса)
Перпендикуляр AB — ось Y (ось ортогонализации)

Для точки P с измеренными расстояниями:
d_1 — расстояние вдоль AB (по оси X)
d_2 — расстояние перпендикулярно AB (по оси Y)

Вектор AB = (X₁ - X₀, Y₁ - Y₀)
Длина AB: |AB| = √((X₁ - X₀)² + (Y₁ - Y₀)²)

Единичные векторы:
e_x = (X₁ - X₀) / |AB|
e_y = (Y₀ - Y₁) / |AB|  (перпендикуляр, левая сторона)

Координаты P:
X_P = X₀ + d_1 × e_x + d_2 × e_y[0]
Y_P = Y₀ + d_1 × e_x[1] + d_2 × e_y[1]
```

**Практический пример:**
```
Базис: A(100, 100), B(200, 100) — горизонтальная линия длиной 100 м
Точка P: d_1 = 50 м (вдоль AB), d_2 = 30 м (вверх от AB)

e_x = ((200-100)/100, (100-100)/100) = (1, 0)
e_y = ((100-100)/100, (100-200)/100) = (0, -(-1)) = (0, 1)

X_P = 100 + 50×1 + 30×0 = 150 м
Y_P = 100 + 50×0 + 30×1 = 130 м

Координаты: P(150, 130)
```

### 3. ОПРЕДЕЛЕНИЕ РАССТОЯНИЙ (РАСЧЁТНЫЕ ФОРМУЛЫ)

#### 3.1 Расстояние в горизонтальной плоскости (плоское расстояние)

**Формула:**
```
d = √((X₂ - X₁)² + (Y₂ - Y₁)²)

где (X₁, Y₁), (X₂, Y₂) — плоские координаты двух точек
```

**Пример:**
```
Точка A: (100, 100)
Точка B: (250, 150)

d = √((250 - 100)² + (150 - 100)²) = √(150² + 50²) = √(22500 + 2500) = √25000 ≈ 158.1 м
```

#### 3.2 Расстояние в 3D пространстве (пространственное расстояние)

**Формула:**
```
d_3D = √((X₂ - X₁)² + (Y₂ - Y₁)² + (Z₂ - Z₁)²)

где Z — отметка (высота)
```

**Пример:**
```
Точка A: (100, 100, 50)
Точка B: (250, 150, 75)

d_3D = √((250-100)² + (150-100)² + (75-50)²) = √(150² + 50² + 25²)
     = √(22500 + 2500 + 625) = √25625 ≈ 160.1 м
```

#### 3.3 Расстояние с учётом кривизны Земли

**Применение:** Для расстояний > 1 км используется формула гаверсинуса.

**Формула (гаверсинус):**
```
d = 2R × arcsin(√(sin²((φ₂-φ₁)/2) + cos(φ₁)×cos(φ₂)×sin²((λ₂-λ₁)/2)))

где:
- R = 6371 км (средний радиус Земли)
- φ₁, φ₂ — широты [рад]
- λ₁, λ₂ — долготы [рад]
```

**Более простая приближённая формула для малых расстояний (≤ 10 км):**
```
Δφ = (φ₂ - φ₁) [рад]
Δλ = (λ₂ - λ₁) [рад]

d ≈ R × √(Δφ² + (cos(φ_сред) × Δλ)²)

где φ_сред = (φ₁ + φ₂) / 2
```

**Пример:**
```
Две точки в Москве:
A: φ₁ = 55.75°N, λ₁ = 37.62°E
B: φ₂ = 55.76°N, λ₂ = 37.64°E

Δφ = 0.01° = 0.01 × π/180 ≈ 0.000175 рад
Δλ = 0.02° = 0.02 × π/180 ≈ 0.000349 рад
φ_сред = 55.755° ≈ 0.973 рад

d ≈ 6371 × √((0.000175)² + (cos(0.973) × 0.000349)²)
  ≈ 6371 × √(3.06×10⁻⁸ + (0.556 × 0.000349)²)
  ≈ 6371 × √(3.06×10⁻⁸ + 3.75×10⁻⁸)
  ≈ 6371 × 0.0000265 ≈ 0.169 км ≈ 169 м
```

## ПЛОЩАДИ, ОБЪЁМЫ И ГАБАРИТЫ

### 1. РАСЧЁТ ПЛОЩАДЕЙ УЧАСТКОВ

#### 1.1 Площадь по координатам вершин (формула Гаусса)

**Формула:**
```
S = 0.5 × |Σ(X_i × (Y_{i+1} - Y_{i-1}))|  (для замкнутого многоугольника)

или

S = 0.5 × |Σ(X_i × Y_{i+1} - X_{i+1} × Y_i)|
```

**Пример для четырёхугольника ABCD:**
```
A(0, 0), B(10, 0), C(10, 5), D(0, 5)

S = 0.5 × |(0×0 - 10×0) + (10×5 - 10×0) + (10×5 - 0×5) + (0×0 - 0×5)|
  = 0.5 × |0 + 50 + 50 + 0|
  = 0.5 × 100
  = 50 м²

Проверка: простой прямоугольник 10×5 = 50 м² ✓
```

#### 1.2 Площадь для нерегулярного многоугольника

**Пример с 5 вершинами:**
```
Вершины участка (по часовой стрелке):
P₁(100, 100)
P₂(200, 120)
P₃(220, 180)
P₄(150, 200)
P₅(80, 150)

S = 0.5 × |(100×120 - 200×100) + (200×180 - 220×120) + (220×200 - 150×180) + (150×150 - 80×200) + (80×100 - 100×150)|

S = 0.5 × |(12000 - 20000) + (36000 - 26400) + (44000 - 27000) + (22500 - 16000) + (8000 - 15000)|

S = 0.5 × |(-8000) + (9600) + (17000) + (6500) + (-7000)|

S = 0.5 × |18100| = 9050 м²
```

#### 1.3 Площадь треугольника (по координатам)

**Формула:**
```
S = 0.5 × |X₁(Y₂ - Y₃) + X₂(Y₃ - Y₁) + X₃(Y₁ - Y₂)|

или через длины сторон (формула Герона):
a = |P₁P₂|, b = |P₂P₃|, c = |P₃P₁|
p = (a + b + c) / 2  (полупериметр)
S = √(p(p-a)(p-b)(p-c))
```

**Пример по координатам:**
```
A(0, 0), B(10, 0), C(5, 8)

S = 0.5 × |0(0 - 8) + 10(8 - 0) + 5(0 - 0)|
  = 0.5 × |0 + 80 + 0|
  = 40 м²

Проверка (через основание и высоту): S = 0.5 × 10 × 8 = 40 м² ✓
```

### 2. РАСЧЁТ ОБЪЁМОВ (ОБЩИЕ ФОРМУЛЫ)

#### 2.1 Объём параллелепипеда (строительный контейнер)

```
V = a × b × h

где a, b — ширина и длина основания [м], h — высота [м]
```

#### 2.2 Объём цилиндра

```
V = π × r² × h = (π/4) × d² × h

где r — радиус основания, d — диаметр, h — высота
```

#### 2.3 Объём конуса

```
V = (1/3) × π × r² × h

где r — радиус основания, h — высота
```

#### 2.4 Объём сферы

```
V = (4/3) × π × r³

где r — радиус
```

#### 2.5 Объём усечённого конуса (воронка, котлован)

```
V = (1/3) × π × h × (R² + R×r + r²)

где h — высота, R — радиус нижнего основания, r — радиус верхнего основания
```

---

### 3. ПРОВЕРКА ГАБАРИТОВ (СООТВЕТСТВИЕ ПРОСТРАНСТВУ)

#### 3.1 Проверка умещаемости объекта в контейнер

**Формула:**
```
Объект умещается если:
X_объ ≤ X_контейнер AND Y_объ ≤ Y_контейнер AND Z_объ ≤ Z_контейнер

Зазоры (требуются для монтажа):
δ_min = 0.1–0.3 м (минимально допустимый зазор)

Проверка с зазорами:
X_объ + 2×δ_min ≤ X_контейнер  (±δ_min с обеих сторон)
```

#### 3.2 Проверка соответствия нормативным ограничениям

**Примеры:**
```
Максимальная высота здания в жилой зоне: 20–35 м (зависит от генплана)
Расстояние от границы до соседнего объекта: ≥ 5–6 м (санитарный разрыв)
Минимальная ширина подъезда: 3.5–4.5 м (для пожарной машины)
```

### 3. ДОПУСКИ НА ОТКЛОНЕНИЯ

#### 3.1 Допуск на линейные размеры

**Формула:**
```
Абсолютный допуск: Δ_abs = t × √2  × m

где:
- t — коэффициент надёжности (t = 2 для 95%, t = 3 для 99%)
- m — средняя квадратическая погрешность (СКП) одного измерения
- √2 — множитель для двух независимых измерений (начало и конец)

Относительный допуск: Δ_отн = Δ_abs / L

где L — длина линии (для проверки полигонометрии: Δ ≤ 1/2000)
```

#### 3.2 Допуск на угловые измерения

**Допуск на угол:** 
```
Δα ≤ 1'–5' (в зависимости от точности теодолита)

Допуск на невязку в полигоне из n сторон:
f_α ≤ 1' × √n (для бюджетного теодолита)
f_α ≤ 0.5' × √n (для высокоточного)
```

#### 3.3 Допуск на высотные отметки

```
При нивелировании:
Δh ≤ 5 мм на каждый км пройденного хода

Для участка площадью S [м²] разбит на сетку:
Δh ≤ 10–20 см (на всей площади, в зависимости от типа грунта)
```

### 2. ПРИМЕР РАСЧЁТА БАЛАНСА С РАЗРЫХЛЕНИЕМ

**Задача:**
```
На строительной площадке требуется:
- Выемка котлована 50×30×2 м = 3000 м³ (в естественном состоянии)
- Засыпка образовавшегося рва 30×20×1.5 м = 900 м³

Грунт: суглинок плотный, K_разр = 1.12

Требуется: определить баланс земляных масс
```

**Решение:**
```
1. Объём выемки в разрыхленном состоянии:
   V_выемка_разрыхлено = 3000 × 1.12 = 3360 м³

2. Объём грунта, необходимый для засыпки (в естественном состоянии):
   V_засыпка_естеств = 900 м³

3. Объём грунта для засыпки (в разрыхленном состоянии):
   V_засыпка_разрыхлено = 900 × 1.12 = 1008 м³

4. Баланс земляных масс (в разрыхленном состоянии):
   ΔV = V_выемка_разрыхлено - V_засыпка_разрыхлено
   ΔV = 3360 - 1008 = 2352 м³

5. Излишек грунта для вывоза:
   V_вывоз = 2352 м³ разрыхленного грунта
   или 2352 / 1.12 ≈ 2100 м³ в естественном состоянии

6. Количество машин (самосвалы 20 м³):
   N_машин = 2352 / 20 = 117.6 ≈ 118 рейсов
```

## ПРИМЕРЫ РАСЧЁТНЫХ ПРОЦЕДУР

### Пример 1: Расчёт объёма выемки методом призм

**Исходные данные:**
```
Участок 40×40 м, разбит на сетку 4×4 м (5×5 сетка вершин)
Отметки вершин (в метрах):
50.2  50.5  50.8  51.1  51.4
50.1  50.3  50.6  50.9  51.2
50.0  50.1  50.4  50.7  51.0
49.9  50.0  50.2  50.5  50.8
49.8  49.9  50.0  50.3  50.6

Требуемая отметка выемки: 50.0 м
Найти: объём грунта для выемки
```

**Решение:**
```
1. Для каждой ячейки сетки 4×4 м вычислить среднюю глубину:
   Глубина = (высота вершин) - 50.0 м
   
   Ячейка (1,1): h_средн = ((50.2 + 50.5 + 50.1 + 50.3) / 4) - 50.0 = 0.275 м
   Ячейка (1,2): h_средн = ((50.5 + 50.8 + 50.3 + 50.6) / 4) - 50.0 = 0.55 м
   ... (всего 16 ячеек)

2. Объём для каждой ячейки:
   V_ячейки = 4 × 4 × h_средн [м³]
   
   V₁₁ = 16 × 0.275 = 4.4 м³
   V₁₂ = 16 × 0.55 = 8.8 м³
   ...

3. Суммарный объём:
   V_полный = Σ V_ячейки = примерно 160–180 м³
```

### Пример 2: Привязка объекта по полярным координатам

**Исходные данные:**
```
Опорный пункт: маяк на точке O(1000, 2000, 150 м)
Объект A:
- Расстояние r = 250 м
- Азимут α = 120° (от магнитного севера)
- Высотный угол ν = +3°
- Направление: на северо-восток
```

**Решение:**
```
1. Преобразовать азимут в декартов угол:
   α_декартов = 90° - 120° = -30° (или 330°)
   
   При этом:
   cos(120°) = -0.5
   sin(120°) = 0.866

2. Вычислить координаты на местности:
   ΔX = 250 × cos(120°) = 250 × (-0.5) = -125 м (западнее)
   ΔY = 250 × sin(120°) = 250 × 0.866 = 216.5 м (севернее)
   
   Хм, азимут обычно считается от северного направления в сторону востока.
   Пересчитаем:
   
   X = X₀ + r × sin(α) = 1000 + 250 × sin(120°) = 1000 + 216.5 = 1216.5 м
   Y = Y₀ + r × cos(α) = 2000 + 250 × cos(120°) = 2000 - 125 = 1875 м

3. Высота:
   Z = Z₀ + r × sin(ν) = 150 + 250 × sin(3°) = 150 + 250 × 0.052 = 163 м

4. Итоговые координаты объекта:
   A(1216.5, 1875, 163) или округлённо A(1217, 1875, 163)
```

### Пример 3: Баланс земляных масс

**Исходные данные:**
```
Участок застройки:
- Площадь участка: 100 × 60 м
- Текущие отметки: от 48.5 до 50.0 м (неровный рельеф)
- Требуемая отметка: 49.5 м (выравнивание)
- Грунт: глина среднепластичная, K_разр = 1.18, K_уплотн = 0.85
```

**Решение:**
```
1. Определить объёмы выемок и насыпей по сечениям:
   
   По среднему профилю:
   - Средняя текущая отметка: 49.25 м
   - Требуемая отметка: 49.5 м
   - Средняя высота насыпи: 49.5 - 49.25 = 0.25 м
   - Площадь участка: 100 × 60 = 6000 м²
   
   V_насыпь ≈ 6000 × 0.25 = 1500 м³ (в среднем)
   
   Но нужно учесть неровность:
   - Низкие места (48.5 м): объём выемки = 30 × 10 × 1 = 300 м³
   - Высокие места (50 м): объём срезки = 20 × 15 × 0.5 = 150 м³
   
   V_выемка = 300 м³, V_срезка = 150 м³

2. Баланс (в естественном состоянии):
   V_выемка = 300 м³ (донизу)
   V_срезка = 150 м³ (со вверху)
   V_выемка_общ = 300 м³
   
   V_насыпь_требуемо = 1500 м³ (для выравнивания)

3. Дефицит грунта:
   Δ = V_насыпь_требуемо - V_выемка_общ = 1500 - 300 = 1200 м³
   Требуется подвоз 1200 м³ грунта

4. С учётом разрыхления и уплотнения:
   Объём подвезённого грунта (в естественном состоянии):
   V_подвоз_естеств = 1200 / K_уплотн = 1200 / 0.85 ≈ 1412 м³
   
   Объём после разрыхления (на доставку):
   V_подвоз_разрыхл = 1412 × 1.18 ≈ 1666 м³

5. Машины (самосвал 20 м³):
   N_рейсов = 1666 / 20 ≈ 83 рейса
```
