## РАЗДЕЛ 1: КЛАССИФИКАЦИЯ ПОМЕЩЕНИЙ ПО ВЗРЫВОПОЖАРНОЙ ОПАСНОСТИ

### 1.1 Методика определения категорий помещений

**Нормативные документы:**
- **Федеральные нормы и правила (ФНП) в области промышленной безопасности** — Приказ Ростехнадзора №543 от 12.11.2013
- **ГОСТ 50671-94** — Межгосударственная система стандартизации. Пожарная опасность. Классификация и номенклатура показателей пожарной опасности материалов и изделий (актуализирован в ГОСТ Р 50671-2012)
- **РД 153-34.0-03.150-00** — Инструкция по расчёту категорий зданий и помещений по взрывопожарной и пожарной опасности
- **СНиП 21-01-97*** — Пожарная безопасность зданий и сооружений
- **ПУЭ (Правила устройства электроустановок)** — Раздел 7.3 (Сети и электрооборудование взрывоопасных помещений)

**Область применения:**
- Определение категории взрывопожарной опасности помещения для целей проектирования и эксплуатации
- Выбор типа электрооборудования (вибукозащищённое, каталитическое и т.д.)
- Расчёт требований к вентиляции, молниезащите, антистатической защите
- Комплектация помещений в соответствии с нормами техники безопасности

#### Формула 1.1.1: Классификация по взрывопожарной опасности

Помещение относится к **категории А, Б, В, Г или Д** в зависимости от присутствия взрывоопасных веществ и их количества:

```
Категория помещения = f(вид вещества, фазовое состояние, концентрация, условия)
```

#### Таблица 1.1: Категории взрывопожарной опасности помещений

| Категория | Определение | Примеры веществ | Опасность |
|-----------|-----------|-----------------|----------|
| **А** | Взрывоопасная атмосфера в нормальных условиях | Водород, ацетилен, оксид этилена, некоторые углеводороды | Очень высокая — горючие газы/пары в взрывчатых концентрациях |
| **Б** | Взрывоопасная атмосфера при нарушении технологии | Пропан, бутан, чёрный порошок, некоторые пластмассы | Высокая — при разгерметизации |
| **В** | Горючие пыли (условие: пыль есть, в концентрации > НКПР) | Угольная пыль, мучная пыль, древесная пыль, металлический порошок | Средняя — при образовании облака |
| **Г** | Горючие жидкости с температурой вспышки > 61°C | Масла, дизельное топливо, бензин (выше 61°C) | Низкая — в нормальных условиях не образуют опасную концентрацию |
| **Д** | Негорючие вещества в раскалённом или расплавленном состоянии | Расплавленный металл, расплавленный шлак, раскалённые предметы | Низкая — при охлаждении опасность снижается |

#### Таблица 1.2: Определение категории на основе свойств вещества

| Параметр | Значение для категории |||||
|----------|------------------------|---|---|---|---|
| | **А** | **Б** | **В** | **Г** | **Д** |
| **Температура вспышки паров, °C** | < 0 | 0 – 61 | − | > 61 | − |
| **Самовоспламенение** | Возможно < 200°C | < 200°C | < 200°C | > 200°C | − |
| **Участие в реакции горения** | Газ/пар | Газ/пар | Пыль | Жидкость | Твёрдое тело |
| **Давление взрыва, МПа** | > 0,5 | > 0,5 | > 0,5 | − | − |

**Примеры реальной классификации:**

##### Пример 1.1a: Склад с пропаном (категория Б)
```
Вещество: пропан (C₃H₈)
Свойства:
  - Состояние: газ (СНП), может быть в жидком (при хранении под давлением)
  - Температура вспышки: −104°C (намного ниже температуры окружающей среды)
  - Нижний концентрационный предел взрываемости (НКПР): 2.2% об.
  - Давление взрыва при НКПР: ~0.8 МПа

Вывод: Категория Б (взрывоопасная атмосфера при нарушении герметичности)
```

##### Пример 1.1b: Зернохранилище с пшеничной пылью (категория В)
```
Вещество: пшеничная мука (пыль)
Свойства:
  - Состояние: пыль (твёрдые частицы в воздухе)
  - НКПР пыли: ~35 г/м³ = 3.5%
  - Температура самовоспламенения пыли: ~450°C
  - Давление взрыва пыли: ~0.7–0.9 МПа

Условие: Взрывоопасная атмосфера образуется только при концентрации пыли в воздухе > НКПР

Вывод: Категория В (горючая пыль, опасна при образовании облака)
```

##### Пример 1.1c: Котельная с дизельным маслом (категория Г)
```
Вещество: дизельное топливо
Свойства:
  - Состояние: жидкость при 20°C
  - Температура вспышки: ~60–65°C
  - При комнатной температуре (20°C) пары не образуют взрывчатую концентрацию

Вывод: Категория Г (в нормальных условиях безопасно, опасно при нагреве > 65°C)
```

### 1.2 Нижний концентрационный предел распространения пламени (НКПР)

**Определение:** Минимальная концентрация горючего вещества в воздухе (или другом окислителе), при которой возможно распространение пламени при наличии источника воспламенения.

**Синонимы:** Lower Explosive Limit (LEL), Lower Flammability Limit (LFL)

#### Таблица 1.3: НКПР для основных промышленных веществ

| Вещество | НКПР, % об. (vol) | НКПР, г/м³ | Верхний предел (ВКПР), % об. | Применение |
|----------|----------|-----------|------------|-----------|
| **Газы** |  |  |  |  |
| Водород (H₂) | 4.0 | 3.2 | 75.0 | Электролиз, синтез |
| Метан (CH₄) | 5.0 | 3.2 | 15.0 | Природный газ |
| Пропан (C₃H₈) | 2.2 | 3.8 | 9.5 | ГБА, теплоэнергетика |
| Бутан (C₄H₁₀) | 1.9 | 3.8 | 8.5 | Тоже что пропан |
| Ацетилен (C₂H₂) | 2.3 | 1.5 | 81.0 | Сварка, резка |
| Оксид этилена (C₂H₄O) | 3.0 | 3.6 | 100.0 | Газовая стерилизация |
| Этилхлорид | 3.8 | 5.0 | 15.0 | Хладоген |
| Диметиловый эфир | 3.4 | 4.2 | 17.0 | Холодильное оборудование |
| **Пары жидкостей** |  |  |  |  |
| Ацетон (CH₃COCH₃) | 2.6 | 3.1 | 13.0 | Растворитель |
| Этанол (C₂H₅OH) | 3.3 | 5.0 | 19.0 | Спирт |
| Бензин | 0.7–1.3 | 1.2–2.2 | 6.0–7.0 | Топливо (низкий НКПР!) |
| Керосин | 0.5–0.9 | 1.0–1.8 | 5.0–5.5 | Топливо |
| Дизельное топливо | 0.5–1.0 | 1.0–2.0 | 4.5–5.0 | Топливо |
| **Пыли (в воздухе)** |  |  |  |  |
| Угольная пыль | 30–50 | 40–67 | 500+ | Углеобрабатывающее производство |
| Мучная пыль | 30–40 | 35–47 | 500+ | Мельницы, хлебопекарни |
| Древесная пыль | 20–30 | 24–36 | 500+ | Деревообработка |
| Сахарная пыль | 20–30 | 24–36 | 500+ | Кондитерские производства |
| Металлический алюминий | 20–30 | 27–41 | 500+ | Металлургия |
| Железный порошок | 60–100 | 81–135 | 500+ | Железообрабатывающая промышленность |

**Примечания:**
- **Газы** — концентрация в % объёма относительно воздуха
- **Пыли** — концентрация в г/м³ (масса в 1 м³ воздуха)
- Для некоторых веществ (напр. водород, ацетилен) НКПР очень низкий → высокая опасность
- Бензин имеет НКПР 0.7–1.3% — критически низко, требует особого внимания

## РАЗДЕЛ 2: КЛАССИФИКАЦИЯ ЗОН ПО ВЕРОЯТНОСТИ ВОЗНИКНОВЕНИЯ ОПАСНОЙ АТМОСФЕРЫ

### 2.1 Методика определения класса зоны

**Нормативные документы:**
- **ПУЭ (Правила устройства электроустановок)** — Раздел 7.3
- **ГОСТ Р МЭК 60079-14-2013** — Взрывоопасные атмосферы. Классификация площадок
- **ГОСТ Р МЭК 60079-10-1-2014** — Взрывоопасные газопаровоздушные смеси. Классификация площадок, помещений и установок

**Область применения:**
- Определение, какие зоны помещения являются потенциально взрывоопасными
- Выбор типа электрооборудования в зависимости от класса зоны
- Проектирование электросетей и светильников в опасных помещениях

#### Таблица 2.1: Классификация зон по вероятности возникновения взрывоопасной атмосферы

| Класс зоны | Вероятность наличия опасной атмосферы | Время наличия опасной атмосферы | Примеры | Требования к оборудованию |
|-----------|----------|-----------|----------|-----------|
| **Зона 0** | Постоянно или часто (длительное время, > 10 ч/год) | Бóльшую часть времени эксплуатации | Внутри резервуаров с горючим газом; закрытые помещения с герметичными технологическими процессами | Только категория 1 (высочайший уровень защиты); источники воспламенения исключены |
| **Зона 1** | Иногда (периодически, 10–100 ч/год) | При нарушении технологии или при аварийных ситуациях | Около технологических аппаратов; вблизи люков резервуаров; системы загрузки/выгрузки | Категория 2 (высокий уровень защиты); источники воспламенения ограничены |
| **Зона 2** | Редко (очень редко, < 10 ч/год) | Кратковременно, случайно | На расстоянии от источника опасной атмосферы; вспомогательные помещения | Категория 3 (обычный уровень защиты); источники воспламенения предполагаются маловероятными |

**Для пыльных помещений:**

| Класс зоны | Вероятность наличия опасной атмосферы пыли | Примеры | Требования |
|-----------|----------|-----------|-----------|
| **Зона 20** | Постоянно, часто (длительное время, взрывоопасная концентрация пыли) | Внутри пылесборников, размольные камеры | Категория 1D (специальное оборудование для пыльных зон) |
| **Зона 21** | Периодически при нарушениях технологии | Около мельниц, в помещениях сушки | Категория 2D |
| **Зона 22** | Редко при возможных нарушениях | На расстоянии от основного оборудования | Категория 3D |

**Примеры реальной классификации помещения:**

##### Пример 2.1a: АЗС (автозаправочная станция)
```
Помещение: Навес над островом с топливными колонками
Технология: Разлив бензина при заправке (НКПР бензина = 0.7–1.3%)

Классификация зон:
1. Зона 1 (радиус 0.5 м от носика) — НКПР достигается часто при неправильной заправке
2. Зона 2 (радиус 0.5–1.5 м) — НКПР может быть при разливе, но редко
3. Зона 0 (внутри топливных баков) — постоянно

Требования:
- Электрооборудование в Зоне 0 и 1: категория 2 (провода по воздуху запрещены)
- Электрооборудование в Зоне 2: категория 3 (обычное оборудование с ограничениями)
- Освещение: взрывобезопасные светильники
- Провода в металлических трубах
```

##### Пример 2.1b: Мукомольный завод
```
Помещение: Зернохранилище и мельница
Технология: Помол зерна → образование пыли

Классификация зон:
1. Зона 20 (внутри мельницы) — пыль постоянно в опасной концентрации
2. Зона 21 (вокруг мельницы, радиус 1–2 м) — пыль при работе
3. Зона 22 (на расстоянии > 2 м) — пыль при сбое вентиляции

Требования:
- Зона 20: взрывобезопасные двигатели (категория 2D)

```

## РАЗДЕЛ 3: РАСЧЁТ УТЕЧЕК И НАКОПЛЕНИЯ ГАЗОВ

### 3.1 Методика простейшей оценки утечек

**Нормативные документы:**
- **ФНП в области ПБ** — расчёты утечек для оценки категории
- **ГОСТ 12.1.004-91** — Взрывобезопасность. Общие требования
- **РД 52.04.13-90** — Банк данных по оценке вероятности утечек на трубопроводах

**Область применения:**
- Оценка вероятности и объёма утечки при разгерметизации оборудования
- Прогноз времени достижения НКПР в помещении
- Расчёт требуемого воздухообмена для разбавления паров

#### Формула 3.1.1: Базовая модель утечки (для небольшого отверстия)

```
G_утечки = ρ · A · √(2 · ΔP / ρ)  [кг/с или м³/с]

где:
  G_утечки — расход вещества через отверстие
  ρ — плотность вещества (кг/м³)
  A — площадь отверстия (м²)
  ΔP — разность давлений (Па)
```

#### Таблица 3.1: Площади отверстий по размерам пробоины

| Диаметр отверстия (мм) | Площадь отверстия (м²) | Описание |
|-----------|-----------|-----------|
| 1 | 7.85 × 10⁻⁷ | Микротрещина (невидимая) |
| 2 | 3.14 × 10⁻⁶ | Мелкая пробоина |
| 5 | 1.96 × 10⁻⁵ | Заметная утечка (капает) |
| 10 | 7.85 × 10⁻⁵ | Значительная утечка (тонкая струя) |
| 20 | 3.14 × 10⁻⁴ | Большая пробоина (значительный поток) |

**Примеры расчётов утечек:**

##### Пример 3.1a: Утечка пропана из контейнера 50 л под давлением 10 бар
```
Дано:
- Вещество: пропан (ρ = 1.8 кг/м³ при 1 атм, но в контейнере жидкий ρ_жидкость = 580 кг/м³)
- Диаметр отверстия: 5 мм → A = 1.96 × 10⁻⁵ м²
- Давление в контейнере: P = 10 бар = 1 МПа = 10⁵ Па
- Атмосферное давление: P_атм = 0.1 МПа = 10⁴ Па
- ΔP = 10⁵ − 10⁴ = 9 × 10⁴ Па

Расчёт (для жидкого пропана):
G_утечки = 580 × 1.96 × 10⁻⁵ × √(2 × 9 × 10⁴ / 580)
         = 580 × 1.96 × 10⁻⁵ × √(310)
         = 580 × 1.96 × 10⁻⁵ × 17.6
         ≈ 0.2 кг/с ≈ 12 кг/мин

Вывод: Утечка очень быстрая! Контейнер 50 л (≈29 кг пропана) опустошится за ~2.5 минуты
```

##### Пример 3.1b: Микротрещина в трубе с природным газом
```
Дано:
- Вещество: метан (ρ = 0.67 кг/м³ при атм.)
- Диаметр отверстия: 1 мм → A = 7.85 × 10⁻⁷ м²
- Давление в трубе: P = 0.5 МПа = 5 × 10⁴ Па
- ΔP = 5 × 10⁴ − 10⁴ = 4 × 10⁴ Па

Расчёт:
G_утечки = 0.67 × 7.85 × 10⁻⁷ × √(2 × 4 × 10⁴ / 0.67)
         = 0.67 × 7.85 × 10⁻⁷ × √(119400)
         = 0.67 × 7.85 × 10⁻⁷ × 345
         ≈ 0.00018 кг/с ≈ 10 г/мин

Вывод: Медленная утечка, но при длительном воздействии в помещении достигнет НКПР
```

### 3.2 Расчёт времени достижения НКПР в помещении

**Определение:** Время, за которое концентрация горючего вещества в помещении достигнет нижнего концентрационного предела воспламеняемости.

#### Формула 3.2.1: Время накопления газа (без вентиляции)

```
t_НКПР = (C_НКПР × V) / G_утечки  [сек]

где:
  t_НКПР — время достижения НКПР
  C_НКПР — концентрация при НКПР (г/м³ или % об.)
  V — объём помещения (м³)
  G_утечки — расход утечки (г/с)
```

#### Формула 3.2.2: Время накопления газа (с естественной вентиляцией)

```
t_НКПР = (C_НКПР × V) / (G_утечки − L_вент × V)  [сек]

где:
  L_вент — интенсивность вентиляции (воздухообмен, 1/сек = 1/h в единицах обмена в час)
```

**Примеры расчётов:**

##### Пример 3.2a: Накопление пропана в закрытом подвале (примечание: опасно!)
```
Дано:
- Объём подвала: V = 100 м³
- Утечка пропана: G_утечки = 0.2 кг/с = 200 г/с (из примера 3.1a)
- НКПР пропана: 2.2% объёма ≈ 3.8 г/м³
- Вентиляция: отсутствует (L_вент = 0)

Расчёт:
t_НКПР = (3.8 × 100) / 200 = 380 / 200 = 1.9 сек

Вывод: ОЧЕНЬ ОПАСНО! НКПР достигается за 1.9 секунды. Взрыв возможен за 2 сек.
```

##### Пример 3.2b: Накопление паров бензина в гараже с вентиляцией
```
Дано:
- Объём гаража: V = 50 м³
- Утечка бензина из трещины в баке: G_утечки = 5 г/ч = 0.0014 г/с
- НКПР бензина: 0.7–1.3% объёма ≈ 1.2–2.2 г/м³ (берём среднее = 1.7 г/м³)
- Вентиляция гаража: L_вент = 1 обмен в час = 1/3600 сек⁻¹ = 2.78 × 10⁻⁴ сек⁻¹

Расчёт (без вентиляции):
t_НКПР_без_вент = (1.7 × 50) / 0.0014 = 85 / 0.0014 ≈ 60700 сек ≈ 17 часов

Расчёт (с вентиляцией):
L_вент × V = 2.78 × 10⁻⁴ × 50 = 0.0139 г/с
t_НКПР_с_вент = (1.7 × 50) / (0.0014 − 0.0139) = 85 / (−0.0125) → НЕ достигает НКПР!

Вывод: С вентиляцией гараж БЕЗОПАСЕН (вентиляция > утечка). Без вентиляции опасно через ~17 часов.
```

## РАЗДЕЛ 4: ПРОВЕРКА ЭФФЕКТИВНОСТИ ВЕНТИЛЯЦИИ

### 4.1 Методика расчёта требуемого воздухообмена

**Нормативные документы:**
- **СП 60.13330.2020** — Отопление, вентиляция и кондиционирование воздуха
- **СП 50.13330.2012** — Тепловая защита зданий
- **ГОСТ 30494-2011** — Параметры микроклимата в помещениях
- **ПУЭ (Раздел 7.3)** — Вентиляция взрывоопасных помещений

**Область применения:**
- Расчёт кратности воздухообмена (n обменов в час) для разбавления паров/газов
- Проверка достаточности естественной вентиляции
- Выбор мощности вентилятора

#### Формула 4.1.1: Требуемый воздухообмен для разбавления опасной атмосферы

```
L_требуемый = (G_утечки × k_безопасности) / (ρ_воздуха × (C_доп − C_фон))  [м³/ч]

где:
  L_требуемый — требуемый объёмный расход воздуха
  G_утечки — масса вещества, поступающего за единицу времени (г/ч)
  k_безопасности — коэффициент безопасности (обычно 3–5)
  ρ_воздуха — плотность воздуха (≈1.2 кг/м³)
  C_доп — допустимая концентрация (обычно НКПР или ПДК)
  C_фон — фоновая концентрация (обычно 0)
```

#### Формула 4.1.2: Кратность воздухообмена

```
n_обменов = L / V  [обмен/ч или h⁻¹]

где:
  L — объёмный расход вентиляции (м³/ч)
  V — объём помещения (м³)
```

#### Таблица 4.1: Требуемая кратность воздухообмена по нормам

| Тип помещения | n, обмен/ч | Примечание |
|-----------|-----------|-----------|
| Жилые комнаты (без источников загрязнения) | 0.5–1 | Минимум для жилья |
| Офисы (норма для СО₂) | 1–2 | Зависит от занятости |
| Лаборатории | 3–8 | Зависит от опасности |
| Помещения с источником опасной атмосферы | 6–20 | ПУЭ 7.3 для взрывоопасных |
| Камеры с токсичными испарениями | 10–30 | Высокая опасность |

**Примеры расчётов:**

##### Пример 4.1a: Вентиляция складского помещения с растворителем (ацетон)
```
Дано:
- Объём помещения: V = 200 м³
- Утечка ацетона: G_утечки = 50 г/ч = 0.84 г/мин
- НКПР ацетона: 2.6% объёма ≈ 3.1 г/м³
- Коэффициент безопасности: k = 3 (требуемая концентрация = НКПР/3 = 1 г/м³)
- Плотность воздуха: ρ = 1.2 кг/м³ = 1200 г/м³

Расчёт требуемого воздухообмена:
L_требуемый = (50 × 3) / (1200 × 1) = 150 / 1200 = 0.125 м³/ч

Расчёт кратности:
n = 0.125 / 200 = 0.000625 обмен/ч (очень слабая утечка!)

Вывод: Даже естественная вентиляция (~0.5 обмена/ч) достаточна, но рекомендуется n ≥ 1 обмен/ч для запаса
```

##### Пример 4.1b: Вентиляция цеха с высокой утечкой пропана
```
Дано:
- Объём цеха: V = 1000 м³
- Утечка пропана: G_утечки = 100 г/ч = 1.67 г/мин (более серьёзная утечка)
- НКПР пропана: 3.8 г/м³
- Коэффициент безопасности: k = 5
- Плотность воздуха: ρ = 1.2 кг/м³ = 1200 г/м³

Расчёт требуемого воздухообмена:
L_требуемый = (100 × 5) / (1200 × 3.8) = 500 / 4560 = 0.11 м³/ч

Расчёт кратности:
n = 0.11 / 1000 = 0.00011 обмена/ч

Вывод: НЕДОСТАТОЧНО! Нужна принудительная вентиляция с кратностью n ≥ 6–10 обменов/ч (см. таблицу 4.1)
```

### 4.2 Проверка естественной вентиляции

#### Формула 4.2.1: Расход воздуха при естественной вентиляции (тяга через дымоход)

```
L_естеств = A × √(2 × g × h × (ρ_снаружи − ρ_внутри) / ρ_снаружи)  [м³/с]

где:
  A — площадь проёма (м²)
  g — ускорение свободного падения (9.81 м/с²)
  h — высота дымохода (м)
  ρ_снаружи, ρ_внутри — плотность воздуха снаружи и внутри
  (ρ_снаружи − ρ_внутри) / ρ_снаружи ≈ (T_внутри − T_снаружи) / T_снаружи
```

**Примеры:**

##### Пример 4.2a: Естественная вентиляция через форточку
```
Дано:
- Площадь открытия: A = 0.5 м² (форточка 0.5 × 1 м)
- Высота помещения: h = 3 м
- Разница температур: ΔT = 20°C (внутри +20°C, снаружи 0°C)
- T_снаружи = 273 К

Расчёт плотности:
Δρ / ρ = ΔT / T = 20 / 273 = 0.073

L_естеств = 0.5 × √(2 × 9.81 × 3 × 0.073 / 1)
          = 0.5 × √(4.3)
          = 0.5 × 2.07
          ≈ 1 м³/с ≈ 3.6 м³/ч

Кратность при V = 200 м³:
n = 3.6 / 200 ≈ 0.018 обмена/ч (ОЧЕНЬ СЛАБО!)

Вывод: Естественная вентиляция неэффективна, нужна принудительная
```

## РАЗДЕЛ 5: ОЦЕНКА СЦЕНАРИЕВ И ПОСЛЕДСТВИЙ

### 5.1 Базовая методика оценки сценариев

**Нормативные документы:**
- **РД 03-418-01** — Методические рекомендации по проведению анализа опасностей и оценке риска аварий на опасных производственных объектах
- **ГОСТ Р 51901.1-2002** — Управление надёжностью. Анализ видов отказов и их последствий (FMEA)
- **ISO 31010:2019** — Risk management — Risk assessment techniques

**Область применения:**
- Определение вероятности и последствий аварийного сценария (разрыв трубопровода, разлив и т.д.)
- Расчёт последствий взрыва (давление, радиус поражения)
- Оценка необходимости инженерных защит

#### Таблица 5.1: Основные сценарии и их характеристики

| Сценарий | Причина | Последствие | Вероятность (год⁻¹) | Категория риска |
|----------|---------|-----------|-----------|-----------|
| **Микротрещина в трубе** | Коррозия, вибрация | Утечка газа (часы–дни) | 10⁻³ – 10⁻² | СРЕДНИЙ |
| **Разрыв фланца** | Вибрация, неправильный монтаж | Быстрая утечка (минуты) | 10⁻⁴ – 10⁻³ | ВЫСОКИЙ |
| **Разлив жидкости из контейнера** | Перекос, падение | Испарение, облако пара (часы) | 10⁻⁵ – 10⁻³ | ВЫСОКИЙ–СРЕДНИЙ |
| **Взрыв взрывоопасной атмосферы** | Источник воспламенения (искра, трение) | Волна давления, осколки | 10⁻⁶ – 10⁻⁴ | КРИТИЧНЫЙ |
| **Пожар в помещении** | Самовоспламенение, контакт с горячей поверхностью | Тепловое излучение | 10⁻⁵ – 10⁻⁴ | КРИТИЧНЫЙ |

#### Формула 5.1.1: Риск как функция вероятности и последствий

```
R = P × C

где:
  R — риск (условные единицы)
  P — вероятность события (год⁻¹)
  C — тяжесть последствий (баллы 1–5)
```

#### Таблица 5.2: Матрица риска (вероятность × последствия)

```
Последствия ↓   |  1 (незначит)  |  2 (легкие)  |  3 (серьёзные)  |  4 (крайне серьёз)  |  5 (катастроф)
Вероятность →   |                |              |                 |                      |
─────────────────┼────────────────┼──────────────┼─────────────────┼──────────────────────┼────────────────
10⁻¹ (очень вероятно) |  10⁻¹  |    0.2       |      0.3         |       0.4             |      0.5
10⁻² (вероятно)   |  10⁻²  |    0.02      |      0.03        |       0.04            |      0.05
10⁻³ (возможно)    |  10⁻³  |    0.002     |      0.003       |       0.004           |      0.005
10⁻⁴ (редко)       |  10⁻⁴  |    0.0002    |      0.0003      |       0.0004          |      0.0005
10⁻⁵ (очень редко) |  10⁻⁵  |    0.00002   |      0.00003     |       0.00004         |      0.00005
```

**Интерпретация риска:**
- **R < 10⁻⁴** — Приемлемо (зелёная зона)
- **10⁻⁴ < R < 10⁻³** — Требуется мониторинг
- **10⁻³ < R < 10⁻²** — Требуются защиты (оранжевая зона)
- **R > 10⁻²** — Неприемлимо, требуется вмешательство (красная зона)

#### Формула 5.1.2: Энергия взрыва и радиус поражения (упрощённо)

```
E_взрыва = ρ × V × ΔH  [Дж]

где:
  ρ — плотность горючего вещества в воздухе (кг/м³)
  V — объём облака (м³)
  ΔH — теплота сгорания (Дж/кг)

r_поражения ≈ (E / E_0)^(1/3)  [м]

где:
  E_0 — опорная энергия (например, 1 МДж = 1 тонна ТНТ)
```

**Примеры оценки сценариев:**

##### Пример 5.1a: Утечка пропана из контейнера и последствия
```
Сценарий: Микротрещина 5 мм в контейнере с пропаном (50 л)
Вероятность: P = 10⁻³ год⁻¹ (один случай на тысячу контейнеров в год)

Утечка: G = 0.2 кг/с = 200 г/с
Время достижения НКПР в помещении V = 100 м³ без вентиляции:
t = (3.8 × 100) / 200 = 1.9 сек

Если источник воспламенения присутствует:
- Энергия взрыва: E ≈ 100 м³ × 3.8 г/м³ × 46 МДж/кг ≈ 1.7 × 10⁶ Дж ≈ 1.7 МДж ≈ 0.4 тонн ТНТ
- Радиус поражения (при полном сгорании облака): r ≈ 10–20 м

Последствия (C = 5, катастрофа):
Риск R = 10⁻³ × 5 = 5 × 10⁻³ (НЕПРИЕМЛИМО!)

Меры защиты:
1. Исключить источники воспламенения (искрозащита, заземление)
2. Установить принудительную вентиляцию (n ≥ 10 обмен/ч)
3. Установить датчик утечки с сигнализацией
4. Окружить контейнер изоляцией
```

##### Пример 5.1b: Пожар в складе с легковоспламеняющимися жидкостями
```
Сценарий: Разлив бензина (200 л) на площади 10 м²
Время испарения (при 20°C): t ≈ 10–20 часов

Облако пара:
- Концентрация на расстоянии 5 м: C ≈ 1 % об. (НКПР бензина)
- Объём облака в нижнем слое (0–1 м): V ≈ 50 м³

Если источник воспламенения (искра, чиркач):
- Давление взрыва в помещении: P ≈ 0.5–1.5 МПа (зависит от герметичности)
- Повреждения: разрушение конструкций, разлёт осколков на расстояние до 20 м

Последствия (C = 5):
Вероятность источника воспламенения: P ≈ 10⁻⁴ (искра, трение)
Риск R = 10⁻⁴ × 5 = 5 × 10⁻⁴ (ТРЕБУЕТСЯ МОНИТОРИНГ)

Меры защиты:
1. ПУЭ 7.3 — электрооборудование категория 2–3
2. Заземление контейнеров
3. Видеонаблюдение + запрет на открытый огонь
4. Система пожаротушения (пена для легковоспламеняющихся жидкостей)
```


## РАЗДЕЛ 8: ВЫВОДЫ И РЕКОМЕНДАЦИИ

1. **Классификация помещения** — первый этап оценки безопасности
   - Определяет требования к оборудованию, электроснабжению, вентиляции
   - Зависит от вида вещества, его количества и условий

2. **НКПР и ВКПР** — критические параметры для оценки взрывоопасности
   - Вещества с низким НКПР (< 1%) особо опасны (бензин)
   - Для пыльных смесей НКПР выше, чем для газов

3. **Вентиляция** — основной инструмент снижения риска
   - Требуемая кратность зависит от вероятности утечки
   - Естественная вентиляция часто недостаточна

4. **Источники воспламенения** — необходимо исключить или ограничить
   - ПУЭ определяет тип оборудования по классам зон
   - Заземление, антистатика, контроль — обязательны

5. **Оценка риска** — инструмент для принятия решений по защитам
   - Риск > 10⁻² неприемлим
   - Требуется многоуровневая защита: вентиляция + электробезопасность + контроль
