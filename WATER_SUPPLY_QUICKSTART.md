## 🎯 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Пример 1: Выбрать диаметр для дома 250 человек

**Решение (1 минута):**

1. Используем таблицу "Численность → Потребление"
   - 250 чел → Q_сут = 42 м³/сут

2. Из таблицы "Потребление → Расход за час"
   - K_макс = 2,0 для 250 человек
   - Q_ч_макс = (42 × 2,0) / 24 = 3,5 м³/ч

3. Из таблицы "Расход → Диаметр"
   - 3,5 м³/ч ≈ 3,0 м³/ч → **DN 32 мм**

**Ответ:** DN 32 мм ✓

### Пример 2: Выбрать резервуар для офиса

**Решение (2 минуты):**

1. Офис 250 человек → q_м = 15 л/чел (по таблице в Standards.md)
2. Q_сут = (15 × 250) / 1000 = 3,75 м³/сут

3. Для офиса K_рез = 0,12 (из таблицы)
4. V_рез = 3,75 × 0,12 = 0,45 м³ ≈ **0,5 м³ (500 л)**

**Ответ:** Резервуар 500 л ✓

### Пример 3: Проверить скорость в трубе

**Решение (1 минута через калькулятор):**

1. Откройте water-calculator.html
2. Вкладка "Гидравлика", расчёт 1
3. Ввод: Q = 1,5 м³/ч, d = 25 мм
4. Результат: v = 0,60 м/с ✓ (в норме 0,5–1,0)

**Ответ:** Скорость в норме ✓
