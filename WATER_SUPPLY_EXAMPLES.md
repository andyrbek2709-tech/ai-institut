## ПРОЕКТ 1: ЖИЛОЙ КОМПЛЕКС (МНОГОЭТАЖНОЕ ЖИЛОЕ ЗДАНИЕ)

### Исходные данные:

| Параметр | Значение |
|---|---|
| **Объект** | Жилой комплекс (250 квартир) |
| **Численность** | 800 человек (в среднем 3,2 чел/квартира) |
| **Высота здания** | 30 этажей × 3,3 м = 99 м |
| **Площадь лота** | 5000 м² |
| **Система ВС** | Внутридомовое, холодное + горячее |
| **Система КС** | Внутридомовая самотечная, 3 стояка |
| **Местоположение** | Московская область (глубина промерзания 1,2 м) |

### 1. РАСЧЁТ ВОДОПОТРЕБЛЕНИЯ

#### Этап 1.1: Определение суточного потребления

**Исходные параметры:**
- Тип здания: жилой дом с центральным отоплением и ГВС
- Удельное потребление q_м = 200 л/чел·сут (СНиП 2.04.01)
- Коэффициент загрузки K_загр = 0,85 (дом в нормальной эксплуатации)

**Расчёт:**
```
Q_сут = (q_м × n × K_загр) / 1000
Q_сут = (200 × 800 × 0,85) / 1000 = 136 м³/сут
```

**Вывод:** Суточное потребление воды составляет **136 м³/сут** или **5,67 м³/ч в среднем**.

#### Этап 1.2: Определение часовых расходов (максимум и минимум)

**Выбор коэффициентов по таблице СНиП 2.04.01:**
- Для n = 800 человек: K_макс = 1,6 (максимальное часовое потребление)
- K_мин = 0,4 (минимальное часовое потребление)

**Расчёты:**
```
Q_ч_макс = (Q_сут × K_макс) / 24
Q_ч_макс = (136 × 1,6) / 24 = 9,07 м³/ч = 151 л/мин

Q_ч_мин = (Q_сут × K_мин) / 24
Q_ч_мин = (136 × 0,4) / 24 = 2,27 м³/ч = 38 л/мин
```

**Вывод:**
- **Максимальное часовое потребление: 9,07 м³/ч (151 л/мин)**
- **Минимальное часовое потребление: 2,27 м³/ч (38 л/мин)**

#### Этап 1.3: Расчёт расходов по санитарным приборам (для проверки)

**Типовая квартира (2–3 человека):**
- 2 раковины (кран) × 0,2 м³/ч = 0,4
- 2 унитаза × 0,3 м³/ч = 0,6
- 1 ванна = 0,7 м³/ч
- 1 душ = 0,4 м³/ч
- 1 кухонная раковина = 0,25 м³/ч

Σq_пр = 2,35 м³/ч

Коэффициент одновременности: K_в = 0,45 (для 7 приборов)

Q_квартиры = 0,45 × 2,35 = **1,06 м³/ч**

Для 250 квартир:
```
Q_всего = 250 × 1,06 × 0,40 = 106 м³/ч

(0,40 — коэффициент одновременности для группы квартир)
```

**Сравнение:** 106 м³/ч ≈ 9,5 м³/ч × 11 часов пиковой нагрузки ✓ Расчёты согласуются.

### 2. ГИДРАВЛИЧЕСКИЕ РАСЧЁТЫ

#### Этап 2.1: Выбор диаметра магистрального трубопровода

**Требуемый расход:** Q = 9,07 м³/ч = 0,00252 м³/с

**Выбор нормативной скорости:**
- Для магистрального трубопровода холодного ВС: v = 0,9 м/с (норма 0,7–1,5 м/с)

**Расчёт диаметра:**
```
Q = π × d² × v / 4
d² = 4Q / (π × v)
d² = 4 × 0,00252 / (π × 0,9)
d² = 0,01008 / 2,827 = 0,00357 м²
d = 0,0598 м ≈ 60 мм
```

**Выбор стандартного диаметра:**
Из ГОСТ 3185 выбираем **DN 63 мм** (внутренний диаметр ≈ 60 мм)

**Проверка скорости:**
```
v = 4 × 0,00252 / (π × 0,06²)
v = 0,01008 / 0,01131 = 0,89 м/с ✓ (в норме)
```

#### Этап 2.2: Расчёт потерь давления в вертикальном стояке

**Данные:**
- Диаметр стояка: DN 50 мм (внутренний диаметр 46 мм)
- Материал: оцинкованная сталь новая (k_e ≈ 0,015 мм)
- Длина: L = 99 м
- Скорость потока на стояке: v ≈ 0,5 м/с (для распределённого потребления)

**Расчёт числа Рейнольдса:**
```
Re = (v × d) / ν = (0,5 × 0,046) / (1,0 × 10⁻⁶) = 23000 (турбулентный режим)
```

**Определение коэффициента трения λ:**
- По диаграмме Муди для Re = 23000 и k_e/d ≈ 0,015/46 ≈ 0,00033: **λ ≈ 0,038**

**Потери на трение:**
```
Δh = λ × (L/d) × (v²/(2g))
Δh = 0,038 × (99/0,046) × (0,5²/(2×9,81))
Δh = 0,038 × 2152 × 0,01274
Δh = 1,04 м вод. ст.
```

**Местные сопротивления (2 колена, 1 вентиль, переходы):**
- ζ_общ ≈ 0,3 + 0,3 + 0,2 = 0,8
- Δh_мест = 0,8 × v²/(2g) = 0,8 × 0,01274 = 0,010 м

**Общие потери на стояке:**
```
Δh_общ = 1,04 + 0,010 = 1,05 м вод. ст. ≈ 10,3 кПа
```

**Вывод:** Потери давления на стояке высотой 99 м составляют примерно **1,05 м вод. ст.** (т.е. только ~1% от геодезической высоты).

#### Этап 2.3: Расчёт давления в насосной станции

**Условия:**
- Минимальное давление на верхнем этаже: p_потреб = 10 м вод. ст. (норма)
- Потери давления в сети (магистраль + стояк): Δh = 1,5 м вод. ст.
- Геодезическая высота: h_гид = 99 м

**Требуемое давление в насосной станции:**
```
p_насоса = p_потреб + Δh + h_гид
p_насоса = 10 + 1,5 + 99 = 110,5 м вод. ст.

В паскалях: p = 110,5 × 9810 = 1084 кПа ≈ 1,08 МПа
В барах: p = 1084 / 100 = 10,84 бара ≈ 11 бара
В атм: p = 110,5 / 10,33 ≈ 10,7 атм
```

**Выбор насоса:**
- Производительность: Q = 9,07 м³/ч
- Напор: H = 110,5 м вод. ст.
- Мощность (приблизительно): P = Q × H × g × ρ / (1000 × η)
  где η ≈ 0,75 (КПД насоса)

```
P = 0,00252 × 110,5 × 9,81 × 1000 / (1000 × 0,75)
P = 36,7 кВт ≈ 40 кВт (норм. мощность: 30–45 кВт)
```

**Рекомендация:** Центробежный насос **11 кВт, 9–10 м³/ч, напор 110–120 м** (например, Grundfos TP 50-160/2 или аналог).

### 3. РАСЧЁТ РЕЗЕРВУАРА

#### Этап 3.1: Определение объёма резервуара

**Исходные данные:**
- Суточное потребление: Q_сут = 136 м³/сут
- Тип объекта: жилой комплекс с центральным водоснабжением
- Коэффициент резервирования: K_рез = 0,20 (по СНиП для ЖД)

**Расчёт объёма:**
```
V_рез = Q_сут × K_рез
V_рез = 136 × 0,20 = 27,2 м³
```

**Выбор стандартного объёма:**
Из серии резервуаров выбираем **V = 30 м³** (ближайший больший объём).

#### Этап 3.2: Проверка режимов заполнения и опорожнения

**Данные:**
- Объём резервуара: V = 30 м³
- Подача от городской сети: Q_подача = 2 м³/ч (договор с водоканалом)
- Максимальное потребление: Q_макс = 9,07 м³/ч

**Время заполнения:**
```
t_заполнения = V / Q_подача = 30 / 2 = 15 часов
```

Практически: заполнение происходит ночью (когда потребление низкое).

**Время опорожнения при пиковом потреблении:**
```
t_опорожнения = V / Q_макс = 30 / 9,07 = 3,3 часа
```

**Интерпретация:** Резервуар вмещает пиковое потребление примерно на 3 часа, чего достаточно для регулирования суточного графика потребления.

#### Этап 3.3: Выбор конструкции резервуара

**Рекомендации:**
- **Тип:** Металлический резервуар на крыше здания или подземный железобетонный
- **Материал:** Нержавеющая сталь (пищевой класс) или ФБС блоки (подземный вариант)
- **Размеры:** Примерно 3 м × 3 м × 3,5 м (длина × ширина × высота для подземного)
- **Оборудование:**
  - Люк для очистки (DN 600 мм)
  - Переливная труба (DN 50 мм, с сеткой)
  - Дренажная труба (DN 32 мм)
  - Датчик уровня (электрический или поплавковый)
  - Воздухопровод с фильтром (DN 25 мм)

### 4. РАСЧЁТ КАНАЛИЗАЦИИ

#### Этап 4.1: Определение расхода сточных вод

**Принцип:** Расход сточных вод в канализацию ≈ потребляемая вода минус потери на испарение и полив.

```
Q_сточ = Q_потреб × K_сточ

где K_сточ ≈ 0,85–0,95 (обычно 0,90 для жилых домов)

Q_сточ = 136 × 0,90 = 122,4 м³/сут ≈ 1,42 м³/ч
```

#### Этап 4.2: Выбор диаметров внутридомовых стояков

**Расход на один стояк (3 стояка на комплекс):**
```
Q_стояка = 122,4 / 3 = 40,8 м³/сут = 0,477 м³/ч
```

**Уклон внутридомовой сети:** i = 0,02 (2% = 2 см на 1 м)

**Материал стояков:** Пластиковые (ПВХ, внутренний диаметр номинальный)

**Методика выбора диаметра (для самотечного потока):**

По формуле Маннинга для d = 100 мм, i = 0,02, n = 0,010:
```
Q = A × v = (π × d²/8) × (1/n) × √(R × i)

где при заполнении 50%:
  A = π × d² / 8 = 0,00785 м²
  R = d/4 = 0,025 м
  v = (1/0,010) × √(0,025 × 0,02) = 100 × √0,0005 = 100 × 0,0224 = 2,24 м/с

Q = 0,00785 × 2,24 = 0,0176 м³/с = 63,4 м³/ч ✓ (достаточно для 40,8 м³/ч)
```

**Проверка скорости:** v = 2,24 м/с находится в норме 0,6–1,5 м/с? 

Скорость выше нормы (2,24 > 1,5 м/с), но это при 50% заполнении. Фактически при расходе 0,477 м³/ч заполнение будет ≈ 15%, скорость снизится до нормальной.

**Выбор:** **DN 100 мм** (пластиковые стояки ПВХ или ПНД)

#### Этап 4.3: Расчёт наружной сети до центральной канализации

**Условия:**
- Расстояние до центральной сети: L = 150 м
- Уклон местности: примерно 1% (1,5 м перепада)
- Грунт: суглинок (по геологии)
- Глубина центральной сети: 2,5 м (ниже промерзания)
- Глубина залегания сети на участке: переменная, от 1,2 м до 2,5 м

**Выбор уклона наружной сети:**
```
i = (2,5 − 1,2) / 150 = 0,0087 ≈ 0,009 = 0,9%
```

Это в пределах норм (минимум 0,003 = 0,3%, максимум 0,15 = 15%).

**Выбор диаметра наружной сети:**
- Расход: Q = 122,4 м³/сут = 0,0014 м³/с (в среднем)
- Максимальный расход: Q_макс = 1,42 м³/ч = 0,000395 м³/с

По расчётам для i = 0,009, n = 0,012 (бетон), пропускная способность при d = 150 мм, заполнение 50% достаточна.

**Выбор:** **DN 150 мм** (бетонные или ПВХ трубы класса SN8 для глубокого залегания)

#### Этап 4.4: Монтажные детали

**Сменные колодцы:**
- На каждом повороте (минимум каждые 40 м при прямом участке)
- На входе в центральную сеть
- На каждом изменении уклона

**Детали сопряжений:**
- С внутридомовой сетью: переход с DN 100 на DN 150 через конусный переход
- На выходе из здания: смотровой колодец с защитой от засоров

## ПРОЕКТ 2: ОФИСНОЕ ЗДАНИЕ (НЕБОЛЬШОЕ)

### Исходные данные:

| Параметр | Значение |
|---|---|
| **Объект** | Офисное здание (2000 м², 250 сотрудников) |
| **Численность** | 250 человек (переменная нагрузка) |
| **Высота здания** | 5 этажей × 3,3 м = 16,5 м |
| **График работы** | 8:00–18:00 (пик в 10:00–11:00 и 14:00–15:00) |

### Расчёты:

#### 1. Водопотребление

**Удельное потребление:**
- Офисное здание: q_м = 15 л/чел·смену (по СНиП)
- Коэффициент загрузки: K_загр = 0,50 (половина дня, переменная нагрузка)

```
Q_сут = (15 × 250 × 0,50) / 1000 = 1,875 м³/сут
```

Но это не полные сутки. Фактическое потребление за рабочий день (10 ч):
```
Q_раб_день = 15 × 250 / 1000 = 3,75 м³ (за 10 часов)
Q_раб_ч_средн = 3,75 / 10 = 0,375 м³/ч
Q_раб_ч_пик = 0,375 × 2,5 = 0,94 м³/ч (с коэффициентом пика 2,5)
```

#### 2. Выбор диаметра

**Для пикового расхода Q = 0,94 м³/ч = 0,000261 м³/с:**
```
Требуемая скорость: v = 0,7 м/с

d = √(4Q/(π×v)) = √(4×0,000261/(π×0,7)) = √(0,00047) = 0,0216 м ≈ 22 мм

Выбираем: DN 25 мм
```

#### 3. Резервуар

**Для офисного здания коэффициент резервирования меньше:**
```
K_рез = 0,12 (низкие колебания потребления)
V_рез = 1,875 × 0,12 = 0,225 м³ ≈ 200 л

Практически выбираем: 1 м³ (бака на крыше или ниже)
```

#### 4. Канализация

**Расход сточных вод:**
```
Q_сточ = 0,94 × 0,85 = 0,80 м³/ч
```

**Диаметр:**
```
Для офиса используется DN 50 мм (один внутридомовой стояк)
```

**Вывод:** Офисное здание требует простой инфраструктуры, типовое оборудование.

## ПРОЕКТ 3: ПРОИЗВОДСТВЕННОЕ ПРЕДПРИЯТИЕ

### Исходные данные:

| Параметр | Значение |
|---|---|
| **Объект** | Машиностроительный завод |
| **Численность рабочих** | 300 человек |
| **Технологическое водоснабжение** | 50 м³/ч (охлаждение, промывка) |
| **Хоз. нужды** | 10 м³/ч (поливка, уборка) |
| **Площадь участка** | 50,000 м² |

### Расчёты:

#### 1. Общее потребление

```
Бытовое (300 чел × 25 л/чел = 7,5 м³/ч)
Технологическое: 50 м³/ч
Хозяйственное: 10 м³/ч
ИТОГО: 67,5 м³/ч (максимум)
```

#### 2. Резервуар

```
Для производства требуется большой запас:
K_рез = 0,35 (критичность технологического процесса)
Q_сут = 67,5 × 8 ч работы = 540 м³
V_рез = 540 × 0,35 = 189 м³

Выбираем: 200 м³ (бассейн 10 м × 5 м × 4 м)
```

#### 3. Насосная станция

```
Производительность: 67,5 м³/ч
Напор: p = 10 (хозяйство) + 2 (потери) + 10 (высоты цехов) = 22 м вод. ст. ≈ 2,2 бара
Мощность: P = (67,5 × 22 × 9,81) / (1000 × 0,75) ≈ 20 кВт
```

**Рекомендация:** Два модульных насоса по 10 кВт (один в работе, один в резерве) для надёжности.

## КОНТРОЛЬНЫЕ ВОПРОСЫ ДЛЯ ПРОВЕРКИ РАСЧЁТОВ

### Водоснабжение:
1. ✓ Выбран ли коэффициент K_загр адекватен типу здания?
2. ✓ Выбран ли K_макс из таблицы СНиП в зависимости от численности?
3. ✓ Совпадает ли диаметр с нормативной скоростью (0,5–1,5 м/с)?
4. ✓ Рассчитаны ли потери давления и добавлены к требуемому напору насоса?
