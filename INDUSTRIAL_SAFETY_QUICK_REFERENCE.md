# Промышленная безопасность — Краткий справочник расчётов

> Быстрая справка для выполнения расчётов категорий помещений, классов зон и оценки вентиляции.

---

## 1. БЫСТРОЕ ОПРЕДЕЛЕНИЕ КАТЕГОРИИ ПОМЕЩЕНИЯ

### Алгоритм:

1. **Какое вещество хранится/используется?**
   - Газ с Т_вспышки < 0°C → **Категория А или Б**
   - Жидкость с Т_вспышки 0–61°C → **Категория В**
   - Жидкость с Т_вспышки > 61°C → **Категория Г**
   - Горючая пыль → **Категория В**
   - Раскалённые/расплавленные вещества → **Категория Д**

2. **Какова вероятность образования опасной атмосферы?**
   - **А:** в нормальных условиях
   - **Б:** при нарушении герметичности
   - **В:** при образовании облака пыли
   - **Г–Д:** низкая вероятность

### Таблица определения (быстро):

```
┌─────────────────────────────────────────────────────────────┐
│ Вещество (примеры)        │ Категория │ Класс зоны          │
├─────────────────────────────────────────────────────────────┤
│ Водород, ацетилен        │ А         │ 0, 1, 2             │
│ Пропан, бутан, метан     │ Б         │ 1, 2                │
│ Пыль (уголь, мука)       │ В         │ 20, 21, 22          │
│ Масло, дизель (>61°C)    │ Г         │ вне Зон 0–2         │
│ Раскалённый металл       │ Д         │ вне Зон 0–2         │
└─────────────────────────────────────────────────────────────┘
```

## 4. БЫСТРЫЙ РАСЧЁТ ВЕНТИЛЯЦИИ

### Формула (упрощённая):

```
L_вент [м³/ч] = (G_утечки [г/ч] × k_безопасности) / C_допуст [г/м³]

Кратность = L_вент / V_помещения
```

### Таблица требуемой кратности (готовые значения):

| Тип помещения | Кратность n, обмен/ч | Примечание |
|-----------|-----------|-----------|
| Жилое (без источников) | 0.5–1 | Минимум |
| Офис | 1–2 | СП 60.13330 |
| Помещение категории Г | 2–4 | Низкий риск |
| Помещение категории В | 4–8 | Средний риск |
| Помещение категории Б | 8–15 | Высокий риск |
| Помещение категории А | 15–25 | Очень высокий риск |
| Зона 0 (пыль) | 15–25 | Постоянный контроль |
| Зона 1 (пыль) | 8–15 | Периодический контроль |
| Зона 2 (пыль) | 4–8 | Редкий контроль |

### Быстрый расчёт требуемого воздухообмена:

```
Шаг 1: Определить категорию помещения (см. п. 1)
Шаг 2: Выбрать из таблицы требуемую кратность n
Шаг 3: L_вент = n × V_помещения

Пример:
- Котельная (категория Б), V = 280 м³
- n = 10 обменов/ч (из таблицы)
- L_вент = 10 × 280 = 2800 м³/ч
```

## 6. ПРОВЕРКА ДОСТАТОЧНОСТИ ВЕНТИЛЯЦИИ

### Алгоритм:

```
Шаг 1: Определить ожидаемую утечку (г/ч)
Шаг 2: Определить НКПР вещества (% объёма или г/м³)
Шаг 3: Расчёт требуемого воздухообмена:
        L_треб = (G_утечки × k_безоп) / (ρ_воздуха × C_доп)
Шаг 4: Определить кратность:
        n = L_треб / V_помещения
Шаг 5: Сравнить с требуемой из таблицы
Шаг 6: Если n_факт < n_требуемой → нужна вентиляция
```

### Пример расчёта:

```
ДАНО:
- Склад с пропаном (50 л, возможная утечка 10 г/ч)
- Объём помещения: V = 500 м³
- НКПР пропана: 3.8 г/м³
- Коэффициент безопасности: k = 3

РАСЧЁТ:
C_доп = НКПР / k = 3.8 / 3 = 1.27 г/м³
L_треб = (10 × 3) / (1.2 × 1.27) = 30 / 1.52 ≈ 20 м³/ч
n = 20 / 500 = 0.04 обмена/ч

ВЫВОД:
Требуемая кратность = 0.04 обмена/ч
Из таблицы для категории Б: нужна ≥ 8 обменов/ч
НЕДОСТАТОЧНО! Нужна принудительная вентиляция ~4000 м³/ч
```

## 8. ЧЕКЛИСТ БЕЗОПАСНОСТИ ДЛЯ ИНЖЕНЕРА

### При проектировании помещения:

- [ ] Определена категория по взрывопожарной опасности
- [ ] Определены классы зон (газ/пыль)
- [ ] Выбрана требуемая кратность вентиляции
- [ ] Выбрано электрооборудование правильной категории
- [ ] Спроектирована система заземления (R < 10 Ом)
- [ ] Предусмотрены средства антистатической защиты
- [ ] Спроектирована система пожаротушения (если требуется)
- [ ] Предусмотрено видеонаблюдение и мониторинг
- [ ] Составлены инструкции для персонала

### При эксплуатации:

- [ ] Вентиляция работает на полную мощность
- [ ] Нет видимых утечек газа/жидкости
- [ ] Датчики утечки функционируют корректно
- [ ] Электрооборудование в хорошем состоянии (нет следов дуги/перегрева)
- [ ] Заземление проверено (R < 10 Ом)
- [ ] Персонал обучен правилам безопасности
- [ ] Ведётся журнал инцидентов и работ по обслуживанию
