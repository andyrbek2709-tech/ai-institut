Установка на телефон: разреши «установка из неизвестных источников», открой .apk, дай разрешения на микрофон и фоновую работу.

1 Создать Telegram-бота через @BotFather

Это твой персональный бот для приёма сообщений. Бот ничего сам делать не будет — только примет твоё сообщение и положит в чат.
  • В Telegram открой @BotFather
  • Отправь команду /newbot
  • Имя бота: любое (например, Андрей помощник)
  • Username: должен заканчиваться на bot (например, andrey_helper_bot)
  • BotFather пришлёт TOKEN — длинная строка вида 7891234567:AAH-...
  • Найди своего бота в поиске Telegram → нажми START → напиши «привет»

2 Узнать свой chat_id

Это твой ID в Telegram, куда бот будет писать. Узнаётся через браузер.
  • Сначала напиши боту любое сообщение в Telegram (если ещё не написал)
  • Открой в браузере (на этом же телефоне или на компе): укажи TOKEN выше
  • Откроется JSON. Найди в нём "chat":{"id": — число после двоеточия и есть твой chat_id
  • Пример: "chat":{"id":123456789, "first_name":"..."}

4 Купить и установить Tasker + AutoVoice

Tasker — автоматизация на Android. AutoVoice — плагин который слушает голос и передаёт текст в Tasker.
  • Google Play: Tasker — ~1500 тенге
  • Google Play: AutoVoice — ~1000 тенге
  • Установи оба, дай разрешения которые попросят (микрофон, уведомления, фоновую работу)
  • В настройках телефона: Батарея → Tasker и AutoVoice → выключи оптимизацию (важно!)

5 Создать задачу в Tasker (Send Telegram)

Это «исполнитель» — что делать когда придёт команда. Один HTTP-запрос к Bot API.
  • Открой Tasker → внизу вкладка Tasks → нажми «+»
  • Имя задачи: Send Telegram
  • На пустом экране задачи нажми «+» → Net → HTTP Request
  • Method: POST
  • URL: укажи TOKEN выше
  • Body: укажи chat_id выше
  • Content Type: application/x-www-form-urlencoded
  • Назад → дискета (сохранить)

6 Создать профиль AutoVoice (триггер на голос)

Это «слушатель» — что должно сработать на твою команду.
  • Tasker → вкладка Profiles → «+»
  • Event → Plugin → AutoVoice → Recognized
  • В настройках AutoVoice Recognized:
    • Command Filter: отправь в телеграм (.*)
    • Включи галочку «Use Regex»
  • Назад → Tasker спросит какую задачу запустить → выбери Send Telegram
  • Открой AutoVoice отдельно → включи Continuous Listening (постоянное прослушивание)

7 Тест с голоса

Самый красивый момент. Скажи телефону вслух:
«Отправь в телеграм проверка работает»

Через 1-2 секунды в чате с твоим ботом должно появиться сообщение «проверка работает». Если появилось — поздравляю! Можно усложнять команды.

Если НЕ появилось: открой Tasker → нижний бар → значок журнала (свиток) — увидишь логи с ошибкой. Самые частые причины:

  • Опечатка в TOKEN или chat_id
  • AutoVoice Continuous Listening выключен
  • Не выключена оптимизация батареи для Tasker / AutoVoice
