# Коды отделов

Справочник кодов отделов в системе EngHub и их эквивалентов в QA.

## Таблица кодов

| Код | Русское имя | Эквивалент QA |
|-----|-------------|---------------|
| АК  | Автоматизированные комплексы | КМ |
| АС  | Автоматизированные системы | ОВ |
| ВК  | Встроенные комплексы | АР |
| ГП  | Главное производство | ЭО |
| ПБ  | Производство блоков | ВК |
| СМ  | Сборка и монтаж | — |
| ТХ  | Технологические установки | — |
| ЭС  | Энергетические системы | — |

## Использование

Коды используются в системе для:
- Классификации проектов по отделам
- Фильтрации задач по подразделениям
- Назначения ответственных лиц из нужного отдела
- Аналитики и отчётности по подразделениям

## Примечания

Коды АС, СМ, ТХ, ЭС не имеют прямых эквивалентов в системе QA и используются только в EngHub.
