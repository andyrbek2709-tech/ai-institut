## ОСНОВНЫЕ ОПРЕДЕЛЕНИЯ

### Что такое коэффициент разрыхления?

**Определение:** Коэффициент разрыхления (K_разр) — отношение объёма грунта в разрыхленном (рассыпчатом) состоянии к объёму этого же грунта в естественном (исходном) состоянии.

```
K_разр = V_разрыхл / V_естеств

где:
- V_разрыхл — объём грунта после разработки [м³]
- V_естеств — объём грунта в скальном залегании [м³]
```

### Физический смысл

При механической разработке грунта (экскавацией, буровзрывными работами) нарушается структура массива:
- Между частицами грунта увеличивается расстояние
- Появляются воздушные пустоты (пористость растёт)
- Объём увеличивается на 5–70% в зависимости от типа грунта

**Пример:**
```
Исходный слой скалы: 1000 м³ (спрессованное состояние)
После взрыва и экскавации: 1300–1500 м³
Коэффициент разрыхления: K = 1300 / 1000 = 1.3–1.5
```

### Связь с объёмной плотностью

```
K_разр = ρ_естеств / ρ_разрыхл

где ρ — объёмная плотность грунта [кг/м³]

Пример:
Суглинок плотный: ρ_естеств = 1900 кг/м³, ρ_разрыхл = 1700 кг/м³
K_разр = 1900 / 1700 = 1.118 ≈ 1.12
```

## ФАКТОРЫ, ВЛИЯЮЩИЕ НА РАЗРЫХЛЕНИЕ

### 1. Влажность грунта

**Сухое состояние:** K_разр максимален
- Грунт легко разрыхляется
- Частицы не слипаются
- Примеры: K(сухая суглинок) = 1.15, K(влажная суглинок) = 1.10

**Влажное состояние:** K_разр уменьшается
- Влага действует как смазка (некоторое уплотнение)
- Частицы слипаются в комки
- Может быть ещё меньше при водонасыщении

**Водонасыщенное состояние:** K_разр минимален
- Вода заполняет все поры
- Грунт не разрыхляется, а течёт
- K(песок водонасыщенный) ≈ 1.03–1.06

### 2. Тип оборудования

| Оборудование | K_разр (типовой) | Примечание |
|----------|---------|-----------|
| **Экскаватор (гусеничный)** | Базовое значение | Стандартная разработка |
| **Экскаватор (колёсный)** | +5% к базовому | Меньше уплотнение грунта |
| **Бульдозер** | -3% к базовому | Больше уплотнение, меньше разрыхл. |
| **Вибропогружной молот** | +10% к базовому | Много вибрации, максимум разрыхл. |
| **Взрывные работы** | +15–20% к базовому | Для скальных грунтов |
| **Вручную (кирка, лопата)** | -10% к базовому | Слабое разрыхление, много уплотн. |
| **Гидромониторный способ** | -5% к базовому | Смывание грунта, мало разрыхл. |

### 3. Время после разработки

**Свежеразработанный грунт (первые часы):**
- K_разр максимален
- Структура нарушена, воздушные поры

**После несколько дней на открытом воздухе:**
- K_разр уменьшается на 5–10%
- Грунт естественно уплотняется (осадка 3–7%)
- Поверхностные слои подсыхают (сокращение объёма)

**После 1–2 недель хранения:**
- K_разр может уменьшиться на 10–20%
- Произойдёт естественное переуплотнение

### 4. Способ выемки

| Способ | Влияние на K | Примечание |
|--------|----------|-----------|
| **Послойная выемка (тонкие слои)** | K уменьшается | Частичное уплотнение при выемке каждого слоя |
| **Разработка целиком** | K максимален | Грунт полностью нарушен |
| **Выемка с подрывкой** | K увеличивается | Взрыв максимально разрыхляет |
| **Выемка с вибрацией** | K уменьшается | Вибрация уплотняет грунт при разработке |

### 5. Структурные особенности грунта

**Слоистые грунты (сланец, шифер):**
- Вдоль слоистости: K выше (расслоение)
- Поперёк слоистости: K ниже (компактнее)

**Грунты с трещинами:**
- Естественно разрушенные: K выше (трещины облегчают разработку)
- Монолитные: K ниже (требуют подрывки)

**Грунты с включениями (галька, щебень):**
- В суглинке: K немного выше (включения разрыхляют структуру)
- В песке: K выше (обломки острые, много пустот)

## РАСЧЁТ И ПРИМЕНЕНИЕ

### 1. Направление расчёта: выемка → насыпь

**Задача:** Известен объём выемки (V_выемка в естественном состоянии). Найти, сколько будет в разрыхленном состоянии (для вывоза, доставки).

**Формула:**
```
V_разрыхл = V_выемка × K_разр
```

**Пример:**
```
Котлован 50×30×2 м, грунт суглинок, K_разр = 1.15

V_выемка = 50 × 30 × 2 = 3000 м³ (естественное)
V_разрыхл = 3000 × 1.15 = 3450 м³ (для вывоза)

Машины (самосвал 20 м³):
N = 3450 / 20 = 172.5 ≈ 173 рейса
```

### 2. Обратное направление: насыпь → исходный грунт

**Задача:** Требуется насыпать V_насыпь [м³] в отведённое место. Сколько нужно привезти грунта в естественном состоянии (учитывая, что он будет разрыхлен и уплотнён)?

**Методология:**
1. Грунт в естественном состоянии V_доставить будет разрыхлен: V_разрыхл = V_доставить × K_разр
2. Разрыхленный грунт при уплотнении сужается: V_уплотнённый = V_разрыхл / K_уплотн_место_укладки
3. Требуемый объём: V_уплотнённый = V_насыпь
4. Отсюда: V_доставить = V_насыпь / (K_разр / K_уплотн_место) = V_насыпь × K_уплотн_место / K_разр

**Формула (упрощённая):**
```
V_доставить = V_насыпь / K_уплотн

где K_уплотн ≈ 0.80–0.95 (зависит от грунта)
или K_уплотн ≈ 1 / K_разр (обратное соотношение)
```

**Пример:**
```
Нужно засыпать яму 40×20×1 м = 800 м³
Грунт: глина, K_разр = 1.20, K_уплотн = 0.83

V_доставить = 800 / 0.83 = 964 м³ (в естественном состоянии)

Проверка:
- Привезли 964 м³ (естественное)
- Разрыхлили при выемке: 964 × 1.20 = 1157 м³
- Уплотнили при укладке: 1157 × 0.83 = 960 м³ ≈ 800 м³ ✓

Машины (самосвал 20 м³):
N = 964 / 20 = 48.2 ≈ 49 рейсов
```

### 3. Баланс земляных масс

**Задача:** На строительной площадке есть область выемки (V_выемка) и область засыпки (V_засыпка). Нужно определить баланс (излишек или дефицит грунта).

**Формула:**
```
Баланс = V_выемка × K_разр - V_засыпка / K_уплотн

Если баланс > 0: излишек (вывоз)
Если баланс < 0: дефицит (подвоз)
Если баланс ≈ 0: идеальный баланс (оптимально)
```

**Пример с расчётом стоимости:**
```
Площадь участка 100×60 м
- Выемка на 1 м (неровный рельеф): V_выемка ≈ 2000 м³
- Засыпка (выравнивание): V_засыпка ≈ 1200 м³
- Грунт: суглинок, K_разр = 1.12, K_уплотн = 0.89

Баланс = 2000 × 1.12 - 1200 / 0.89
       = 2240 - 1348
       = 892 м³ (излишек для вывоза)

Рейсы (20 м³):
N = 892 / 20 = 44.6 ≈ 45 рейсов вывоза

Стоимость (по 500 руб./рейс):
Сумма = 45 × 500 = 22 500 руб.
```

## ПРИМЕРЫ РАСЧЁТОВ

### Пример 1: Расчёт выемки под фундамент

**Исходные данные:**
```
Фундамент 30×20 м, глубина 2 м
Грунт: глина жёсткая (в скальном залегании)
K_разр = 1.20 (для жёсткой глины)
```

**Расчёт:**
```
1. Объём выемки (в естественном состоянии):
   V_выемка = 30 × 20 × 2 = 1200 м³

2. Объём в разрыхленном состоянии (для вывоза):
   V_разрыхл = 1200 × 1.20 = 1440 м³

3. Средний коэффициент вывоза (самосвалы 20 м³):
   N_рейсов = 1440 / 20 = 72 рейса

4. Если часть грунта можно использовать для обратной засыпки:
   V_засыпка_обратной = 400 м³ (опалубка, подушка)
   V_вывезти = 1440 - 400 = 1040 м³
   N_рейсов_вывоза = 1040 / 20 = 52 рейса
```

### Пример 2: Расчёт насыпи под площадку

**Исходные данные:**
```
Площадка 50×40 м, требуемая высота насыпи 0.5 м
Грунт: суглинок средний, K_разр = 1.14, K_уплотн = 0.88
```

**Расчёт:**
```
1. Требуемый объём насыпи:
   V_насыпь = 50 × 40 × 0.5 = 1000 м³

2. Объём грунта для подвоза (естественное состояние):
   V_подвоз = V_насыпь / K_уплотн = 1000 / 0.88 = 1136 м³

3. Рейсы самосвалов:
   N = 1136 / 20 = 56.8 ≈ 57 рейсов

4. Проверка (обратный расчёт):
   - Привезено: 1136 м³ (естеств.)
   - Разрыхлено при разгрузке: 1136 × 1.14 = 1295 м³
   - Уплотнено при уплотнении: 1295 × 0.88 = 1140 м³ ≈ 1000 м³ ✓

5. Время уплотнения (вибротрамбовка 200 м²/ч):
   Время = 50 × 40 / 200 = 10 часов
```

### Пример 3: Баланс смешанной площадки

**Исходные данные:**
```
Участок 120×80 м разделён на две части:
- Западная (60×80): выемка 1.2 м (текущая отметка 49.0, требуемая 47.8)
- Восточная (60×80): насыпь 0.8 м (текущая отметка 48.0, требуемая 48.8)

Грунт везде: суглинок тяжёлый
K_разр = 1.18, K_уплотн = 0.85
```

**Расчёт:**
```
1. Выемка на западе:
   V_выемка = 60 × 80 × 1.2 = 5760 м³

2. Насыпь на востоке:
   V_насыпь = 60 × 80 × 0.8 = 3840 м³

3. Грунта в разрыхленном состоянии:
   V_разрыхл = 5760 × 1.18 = 6797 м³

4. Грунта нужно для насыпи (в разрыхленном виде):
   V_для_насыпи_разрыхл = 3840 / 0.85 = 4518 м³

5. Баланс:
   ΔV = 6797 - 4518 = 2279 м³ (излишек для вывоза)

6. Вывоз (самосвалы 20 м³):
   N = 2279 / 20 = 114 рейсов

7. Если снять и вернуть плодородный слой (0.3 м):
   V_плодородный = 120 × 80 × 0.3 = 2880 м³
   V_плодородный_разрыхл = 2880 × 1.15 = 3312 м³
   Мощность экскаватора: ≈ 50 м³/ч → 66 часов на снятие и возврат
```

## ВТОРИЧНОЕ РАЗРЫХЛЕНИЕ

**Определение:** Коэффициент вторичного разрыхления — отношение объёма грунта при повторной разработке уже разрыхленного (выветренного) грунта.

```
K_разр_2 = V_разр_2 / V_разр_1

где:
- V_разр_2 — объём при вторичной разработке
- V_разр_1 — объём при первичной разработке
```

**Физический смысл:**
Когда разрыхленный грунт некоторое время лежит на открытом воздухе, он частично уплотняется (осадка, выветривание), но при повторной разработке может разрыхлиться ещё больше.

### Таблица вторичного разрыхления

| Тип грунта | Время ≤1 недели | Время 1–4 недели | Время > 1 месяца |
|-----------|---------|---------|----------|
| **Скала (гранит)** | K₂ = 1.00 K₁ | K₂ = 0.98 K₁ | K₂ = 0.95 K₁ |
| **Суглинок** | K₂ = 1.05 K₁ | K₂ = 1.08 K₁ | K₂ = 1.10 K₁ |
| **Песок** | K₂ = 1.02 K₁ | K₂ = 1.03 K₁ | K₂ = 1.05 K₁ |
| **Глина** | K₂ = 1.08 K₁ | K₂ = 1.12 K₁ | K₂ = 1.15 K₁ |

**Пример:**
```
Суглинок выкопан 2 недели назад
K₁ = 1.14 (первичное разрыхление)
K₂ = 1.08 × 1.14 = 1.231 (вторичное, после 2 недель)

Если снова перекопать этот грунт:
V_разр_2 = V_разр_1 × 1.231 / 1.14 = V_разр_1 × 1.08

То есть объём увеличится ещё на 8%.
```

## УПЛОТНЕНИЕ ОБРАТНО

**Определение:** Коэффициент уплотнения (K_уплотн) — способность разрыхленного грунта вернуться к исходной плотности (или близко к ней) при воздействии (вибрация, трамбовка, собственная тяжесть).

```
K_уплотн = V_разрыхл / V_уплотнённый

где V_уплотнённый ≤ V_естеств (обычно 5–20% больше)
```

### Стандартные коэффициенты уплотнения для различных типов грунтов

| Грунт | K_разр | K_уплотн (в норме) | V_остаточный | Примечание |
|-------|--------|----------|-----------|-----------|
| **Глина** | 1.18 | 0.85 | 103% V_естеств | Плотно уплотняется |
| **Суглинок** | 1.14 | 0.88 | 102% V_естеств | Стандартный случай |
| **Супесь** | 1.10 | 0.91 | 101% V_естеств | Хорошо уплотняется |
| **Песок крупный** | 1.10 | 0.91 | 101% V_естеств | Требует вибрации |
| **Песок мелкий** | 1.07 | 0.94 | 101% V_естеств | Уплотняется вибрацией + вода |
| **Гравий с песком** | 1.15 | 0.87 | 102% V_естеств | Требует виброуплотнения |
| **Щебень** | 1.40 | 0.71 | 141% V_естеств | Много пустот, требует вибрации |

### Методы уплотнения и эффективность

| Метод | Эффективность | Толщина слоя | Время | Примечание |
|-------|----------|----------|--------|-----------|
| **Самоуплотнение (со временем)** | K_уплотн ≈ 0.92–0.96 | — | 2–4 недели | Естественная осадка |
| **Виброплита (легкая, 20 кН)** | K_уплотн ≈ 0.88–0.92 | 10–15 см | 4–6 проходов | Для песка, супеси |
| **Вибротрамбовка (50 кН)** | K_уплотн ≈ 0.85–0.90 | 15–30 см | 6–8 проходов | Для суглинков |
| **Катки вибрирующие (100 кН)** | K_уплотн ≈ 0.83–0.88 | 30–50 см | 4–6 проходов | Для насыпей, площадок |
| **Трамбовка вручную (кувалда)** | K_уплотн ≈ 0.80–0.85 | 5–10 см | 15–20 ударов | Локальное уплотнение |
| **Трамбовка с механизацией** | K_уплотн ≈ 0.82–0.87 | 20–30 см | 8–10 проходов | Средний вариант |

### Требования стандартов на степень уплотнения

**СНиП 2.02.01-83 (Основания и фундаменты):**
```
Под фундаменты зданий:
- K_уплотн ≥ 0.95 (это 95% плотности исходного грунта)
- Контроль: зондирование динамическое, статическое испытание

Под дорожные основания:
- K_уплотн ≥ 0.98 (для подтопляемых грунтов)
- K_уплотн ≥ 1.00 (для скальных грунтов — не уплотняются)

Под отмостки и благоустройство:
- K_уплотн ≥ 0.90 (менее требовательно)
```

## ПРАКТИЧЕСКИЕ РЕКОМЕНДАЦИИ

### Выбор коэффициента для расчёта

1. **Если точный тип грунта неизвестен:**
   - Используй среднее значение: K_разр ≈ 1.12–1.15 (суглинок)

2. **Если грунт влажный:**
   - Уменьшай K на 5–8% (грунт слипается, меньше разрыхляется)

3. **Если используются взрывные работы:**
   - Увеличивай K на 15–20% (максимальное разрыхление)

4. **Если грунт выветривается (хранится на открытом воздухе > 2 недель):**
   - Применяй коэффициент вторичного разрыхления K₂

5. **Для обратной засыпки:**
   - Используй K_уплотн = 1 / K_разр (обратное соотношение)
   - Или смотри таблицу выше (более точно)

### Типичная последовательность расчёта проекта

```
1. Определить границы выемки и насыпи (по топопланам)
2. Выявить тип грунта (бурение, скважины)
3. Выбрать K_разр по таблице (с учётом влажности, оборудования)
4. Вычислить V_разрыхл = V_выемка × K_разр
5. Определить баланс: Баланс = V_разрыхл - (V_насыпь / K_уплотн)
6. Если баланс > 0: рассчитать вывоз
   Если баланс < 0: рассчитать подвоз
7. Оценить стоимость транспорта
8. Выбрать оборудование уплотнения (виброплита, каток и т.д.)
9. Согласовать сроки с генподрядчиком
```
