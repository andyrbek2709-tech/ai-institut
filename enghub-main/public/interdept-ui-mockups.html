Логика SLA расчёта
awaiting_since — устанавливается когда
assignment_status = 'pending_accept'
⏱ elapsed = now() − awaiting_since
Зелёный: elapsed < sla_hours × 0.5
Жёлтый: elapsed < sla_hours
Красный: elapsed ≥ sla_hours
Расчёт на клиенте через useMemo

  • reviews — нет location → тоже в миграцию 030
