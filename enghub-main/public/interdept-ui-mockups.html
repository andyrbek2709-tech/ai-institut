§3.2 Нагрузки от технологического оборудования ⚠ Требует проверки
По ТЗ (п.4.3): нагрузки от оборудования 8 кН/м². Смежный раздел ЭО указывает расчётную нагрузку 9.6 кН/м² (с учётом кабельных лотков).
🔍 AI-сверка: Расхождение +20%. ТЗ п.4.3 устарело. Требуется согласование с ГИПом и актуализация задания на проектирование. Ссылка: СП 20.13330.2017 п.8.2.4.
§5.1 Защитный слой бетона ✓ Соответствует
По ТЗ (п.6.1): защитный слой ≥30 мм для фундаментов. Раздел КЖ: 35 мм (Ф1), 40 мм (Ф2).
✅ AI-сверка: Требования ТЗ выполнены. Соответствие ГОСТ 31384-2017 п.7.3.2. Дополнительных согласований не требуется.
§7.3 Вводы инженерных сетей ❌ Конфликт
По ТЗ: вводы ВК и ЭО — северная стена. Раздел АС: окно запроектировано на северной стене ось 1/А-В, перекрывает зону ввода.
🚨 AI-сверка: Конфликт позиции ввода и проёма. Необходима координация АС + ВК + ЭО. Рекомендация: перенести ввод на ось 2 или пересмотреть положение окна в АС.
API
POST /api/interdept-ai/cross-check — принимает project_id + section_id, сравнивает текст ТЗ, данные смежных разделов и нормативную базу (AGSK), возвращает вердикт по каждому пункту.
