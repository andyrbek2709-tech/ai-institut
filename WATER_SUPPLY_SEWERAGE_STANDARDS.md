# Методики и стандарты расчётов водоснабжения и канализации

> Комплексный справочник формул, коэффициентов, нормативных значений и примеров для инженерных расчётов водоснабжения и внутридомовой канализации.

---

## РАЗДЕЛ 1: РАСЧЁТЫ ВОДОПОТРЕБЛЕНИЯ

### 1.1 Методика расчёта суточного и часового потребления воды

**Нормативные документы:** СНиП 2.04.01-85 (обновление в СП 31.13330.2012), СНиП 2.04.02-84

#### Формула 1.1.1: Суточное водопотребление

```
Q_сут = (q_м × n × K_загр) / 1000  [м³/сут]

где:
  Q_сут    — суточное потребление воды, м³/сут
  q_м      — удельное потребление воды на 1 человека, л/сут
  n        — численность населения (пользователей), чел
  K_загр   — коэффициент загрузки сооружения (0,7...1,0 для ЖД, 0,5...0,8 для офисов)
```

**Удельное водопотребление (СНиП 2.04.01):**

| Тип объекта | q_м, л/чел·сут | Примечание |
|---|---|---|
| Жилой дом (без дополнительных удобств) | 60 | Минимум |
| Жилой дом (с ванной, горячее водоснабжение) | 160–200 | Стандарт |
| Жилой дом (с центральным отоплением + ГВС) | 220–240 | Комфорт |
| Офисное здание | 15–20 | На 1 чел за рабочий день |
| Ресторан, кафе | 50–150 | На 1 посетителя за прием пищи |
| Больница | 200–250 | На 1 пациента в сутки |
| Производственное предприятие | 20–100 | На 1 рабочего, зависит от типа производства |

#### Формула 1.1.2: Часовое водопотребление (максимальное)

```
Q_ч_макс = (Q_сут × K_макс) / 24  [м³/ч]

где:
  Q_ч_макс — максимальное часовое потребление
  K_макс   — коэффициент максимума часовой подачи (зависит от численности)
  24       — количество часов в сутках
```

**Значения коэффициента K_макс по СНиП 2.04.01 в зависимости от численности:**

| Численность, чел | K_макс |
|---|---|
| до 100 | 2,4–3,0 |
| 100–500 | 2,0–2,4 |
| 500–1000 | 1,8–2,0 |
| 1000–5000 | 1,5–1,8 |
| свыше 5000 | 1,3–1,5 |

#### Формула 1.1.3: Минимальное часовое потребление

```
Q_ч_мин = (Q_сут × K_мин) / 24  [м³/ч]

где:
  K_мин = 0,3...0,5  (для жилых домов и большинства объектов)
```

#### Формула 1.1.4: Баланс водопотребления (за период времени)

```
V_баланс = Q_сут × t - ∑V_отведено  [м³]

где:
  V_баланс     — остаток воды в системе (разница между подачей и отведением)
  Q_сут        — суточное потребление
  t            — период наблюдения, сут
  ∑V_отведено  — суммарное отведение (в канализацию), м³
```

**Практический пример 1.1:**

Расчёт водопотребления жилого комплекса (250 квартир, 800 человек):

1. Выбираем q_м = 180 л/чел·сут (жилой дом с центральным ГВС)
2. K_загр = 0,85 (коэффициент загрузки)
3. Q_сут = (180 × 800 × 0,85) / 1000 = **122,4 м³/сут**
4. K_макс = 1,6 (для 800 человек, по таблице)
5. Q_ч_макс = (122,4 × 1,6) / 24 = **8,16 м³/ч = 136 л/мин**
6. K_мин = 0,4
7. Q_ч_мин = (122,4 × 0,4) / 24 = **2,04 м³/ч = 34 л/мин**

### 1.2 Методика расчёта расходов по точкам потребления (расход воды в приборах)

**Нормативные документы:** СП 31.13330 (внутренние санитарно-технические системы)

#### Таблица 1.2: Расходы воды санитарными приборами (СП 31.13330)

| Прибор | Расход на приборе, л/мин | Q_пр, м³/ч | Условный расход (q_усл) |
|---|---|---|---|
| Кран для личных нужд (умывальник) | 3–4 | 0,18–0,24 | 0,1–0,15 |
| Кран для хоз. нужд | 5–6 | 0,30–0,36 | 0,2 |
| Унитаз с бачком (смыв) | 0,5–2,0 л/смыв | — | 0,3 |
| Ванна (заполнение за 10–15 мин) | 10–15 | 0,6–0,9 | 0,5 |
| Душ | 5–10 | 0,3–0,6 | 0,3–0,4 |
| Кухонная раковина | 3–5 | 0,18–0,3 | 0,15 |
| Стиральная машина (при подпитке) | 0,5–1,0 | — | 0,3–0,5 |
| Водонагреватель (проточный) | вход 3–6 л/мин | — | — |
| Питьевой фонтан | 0,2–0,3 | — | 0,05 |
| Пожарный кран (50 мм) | 40 | 2,4 | — |

#### Формула 1.2.1: Расчётный расход по вероятности использования

```
Q_расч = Σ(K_в × q_пр)  [м³/ч или л/мин]

где:
  K_в     — коэффициент вероятности одновременного использования
  q_пр    — расход прибора
  Σ       — сумма по всем приборам в группе
```

**Коэффициент вероятности K_в (СП 31.13330):**

```
При числе приборов n:
  n = 2–3  →  K_в = 0,6–0,7
  n = 4–6  →  K_в = 0,5–0,6
  n = 7–10 →  K_в = 0,4–0,5
  n > 10   →  K_в = 0,3–0,4

Или по формуле:
  K_в = 1 / √(n × K_зан)
  
  где K_зан — коэффициент занятости (0,03...0,05 для жилья)
```

**Практический пример 1.2:**

Квартира (площадь 80 м², 2–3 человека):
- 2 унитаза с бачками (q_пр = 0,3 м³/ч каждый)
- 2 раковины (q_пр = 0,2 м³/ч каждый)
- 1 ванна (q_пр = 0,7 м³/ч)
- 1 душ (q_пр = 0,4 м³/ч)
- 1 кух. раковина (q_пр = 0,25 м³/ч)

Σq_пр = 0,3 + 0,3 + 0,2 + 0,2 + 0,7 + 0,4 + 0,25 = **2,35 м³/ч**

K_в = 1 / √(7 × 0,04) = 1 / √0,28 = 1 / 0,53 = **1,89** (невозможный коэффициент > 1, используем табличный)

По таблице: n = 7, K_в = 0,45

Q_расч = 0,45 × 2,35 = **1,06 м³/ч = 17,6 л/мин**

## РАЗДЕЛ 2: ГИДРАВЛИЧЕСКИЕ РАСЧЁТЫ ТРУБОПРОВОДОВ

### 2.1 Методика расчёта скорости потока и потерь давления

**Нормативные документы:** СНиП 2.04.01-85, СП 31.13330, ГОСТ 3185 (трубы стальные), ГОСТ 6867 (трубы пластиковые)

#### Формула 2.1.1: Скорость потока в трубе

```
v = Q / A  [м/с]

где:
  v — скорость потока
  Q — объёмный расход, м³/с
  A — площадь поперечного сечения трубы, м²
  
Для круглой трубы:
  A = π × d² / 4
  
  откуда:
  v = (4 × Q) / (π × d²)  [м/с]
```

**Нормативные скорости потока (СНиП 2.04.01):**

| Участок сети | v_мин, м/с | v_макс, м/с | Примечание |
|---|---|---|---|
| Стояк холодного водоснабжения | 0,5 | 1,0 | Основной критерий — шум |
| Главный трубопровод (холодное ВС) | 0,7 | 1,5 | Экономика: потери энергии |
| Тупиковая ветвь (холодное ВС) | 0,3 | 0,7 | Минимум — застой, максимум — шум |
| Горячее водоснабжение | 0,7 | 1,2 | Ниже горячего из-за тепловых потерь |
| Циркуляционный трубопровод (ГВС) | 0,3 | 0,6 | Низкая скорость из-за потерь в окружающей среде |
| Канализационный трубопровод | 0,6 | 1,5 | Минимум — самоочистка, максимум — абразивный износ |
| Дренажный трубопровод | 0,5 | 1,2 | |

#### Формула 2.1.2: Потери давления (формула Дарси-Вейсбаха)

```
Δp = λ × (L / d) × (ρ × v²) / 2  [Па]

или в метрах водяного столба:

Δh = λ × (L / d) × (v²) / (2g)  [м вод. ст.]

где:
  λ       — коэффициент трения (для турбулентного течения)
  L       — длина участка, м
  d       — внутренний диаметр трубы, м
  ρ       — плотность воды, ≈ 1000 кг/м³
  v       — скорость потока, м/с
  g       — ускорение свободного падения, 9,81 м/с²
  v²/(2g) — динамическое давление, м вод. ст.
```

#### Формула 2.1.3: Коэффициент трения λ (диаграмма Муди)

**Для гладких труб (пластмасса, новая медь) — число Рейнольдса 10^4 < Re < 10^5:**

```
λ ≈ 0,316 / Re^0,25  (формула Блазиуса)

где Re = (v × d) / ν

  v — скорость потока, м/с
  d — диаметр трубы, м
  ν — кинематическая вязкость воды при 10°C ≈ 1,3 × 10^-6 м²/с
      при 20°C ≈ 1,0 × 10^-6 м²/с
      при 60°C ≈ 0,47 × 10^-6 м²/с (горячая вода)
```

**Таблица 2.1: Коэффициент трения λ для гладких труб (при Re = 10^4...10^5):**

| Диаметр трубы, мм | Re = 20000 | Re = 50000 | Re = 100000 |
|---|---|---|---|
| 10 | 0,060 | 0,050 | 0,043 |
| 16 | 0,058 | 0,048 | 0,041 |
| 20 | 0,057 | 0,047 | 0,040 |
| 25 | 0,056 | 0,046 | 0,039 |
| 32 | 0,055 | 0,045 | 0,038 |
| 40 | 0,054 | 0,044 | 0,037 |
| 50 | 0,052 | 0,043 | 0,036 |

**Для ржавых или грязных труб (стальные, прокатные) — эквивалентная шероховатость k_e:**

| Материал | k_e, мм | λ при Re = 10^5 |
|---|---|---|
| Полиэтилен, ПВХ, новая медь | 0,001–0,002 | 0,032–0,035 |
| Оцинкованная сталь (новая) | 0,01–0,015 | 0,037–0,040 |
| Чугунная труба (новая) | 0,05–0,1 | 0,045–0,050 |
| Стальная оцинкованная (эксплуатируемая 10 лет) | 0,1–0,3 | 0,050–0,055 |
| Стальная без покрытия (эксплуатируемая 10 лет) | 0,3–0,5 | 0,055–0,060 |
| Чугун (эксплуатируемый 10 лет) | 0,25–0,4 | 0,053–0,058 |

**Формула Коулбрука-Уайта (для любого режима):**

```
1/√λ = -2 × log₁₀[(k_e/(3,7d) + 5,74/Re^0,9)]

Приблизительное решение для турбулентного режима:
λ ≈ 0,11 × (k_e/d + 6,8/Re)^0,25
```

#### Формула 2.1.4: Потери давления на местные сопротивления

```
Δp_мест = ζ × (ρ × v²) / 2  [Па]

или в метрах водяного столба:

Δh_мест = ζ × v² / (2g)  [м вод. ст.]

где:
  ζ — коэффициент местного сопротивления
```

**Таблица 2.2: Коэффициенты местных сопротивлений ζ:**

| Элемент | ζ |
|---|---|
| Вход в трубу (с краев) | 0,5–0,8 |
| Вход в трубу (с окраской, гладкий) | 0,05–0,1 |
| **Колена и отводы:** | |
| Колено 90° (R/d > 1) | 0,25–0,35 |
| Колено 90° (R/d ≈ 1) | 0,4–0,6 |
| Колено 90° (R/d < 1, острое) | 0,7–1,0 |
| Колено 45° | 0,15–0,2 |
| **Тройники (в точке разветвления):** | |
| Через (боковой выход) | 0,6–1,0 |
| Прямой проход (в сторону) | 0,2–0,3 |
| **Вентили и задвижки:** | |
| Полностью открытая задвижка | 0,1–0,2 |
| Шаровой кран (полностью открыт) | 0,04–0,08 |
| Дроссельный клапан (полностью открыт) | 0,5–1,5 |
| Обратный клапан | 2,0–3,0 |
| **Диафрагмы и редукционные клапаны:** | |
| Редукционный клапан | 2–5 (зависит от конструкции) |

#### Формула 2.1.5: Общие потери давления в участке сети

```
Δp_общ = Δp_трение + Δp_мест  [Па]

Δh_общ = Δh_трение + Δh_мест  [м вод. ст.]

где обычно представляется как "эквивалентная длина местного сопротивления":

L_эквив = ζ × d / λ  [м]

тогда:
Δh_общ = λ × (L_реальная + L_эквив) / d × v² / (2g)
```

#### Практический пример 2.1:

Расчёт потерь давления в стояке холодного водоснабжения:

**Данные:**
- Материал: полиэтилен (ПВХ), новый
- Диаметр: d = 25 мм (внутренний)
- Расход: Q = 1,5 л/мин = 0,025 л/с = 0,000025 м³/с
- Длина стояка: L = 30 м
- Элементы: 2 колена 90°, 1 редукционный клапан (полностью открыт, ζ = 3)

**Решение:**

1. **Скорость потока:**
   - A = π × (0,025)² / 4 = 4,909 × 10⁻⁴ м²
   - v = 0,000025 / (4,909 × 10⁻⁴) = **0,051 м/с** (в пределах нормы 0,3–1,0 м/с ✓)

2. **Число Рейнольдса:**
   - Re = (v × d) / ν = (0,051 × 0,025) / (1,0 × 10⁻⁶) = 1275 **< 2300 (ламинарный режим!)**
   
   Для ламинарного режима (Re < 2300):
   - λ = 64 / Re = 64 / 1275 = **0,050**

3. **Потери на трение:**
   - Δh_трение = 0,050 × (30 / 0,025) × (0,051)² / (2 × 9,81)
   - Δh_трение = 0,050 × 1200 × 0,0026 / 19,62 = **0,008 м вод. ст.**

4. **Потери на местные сопротивления:**
   - ζ_2колена = 2 × 0,3 = 0,6
   - ζ_клапан = 3
   - ζ_общ = 3,6
   - L_эквив = 3,6 × 0,025 / 0,050 = 1,8 м
   - Δh_мест = 3,6 × (0,051)² / (2 × 9,81) = 3,6 × 0,0026 / 19,62 = **0,00048 м вод. ст.**

5. **Общие потери:**
   - Δh_общ = 0,008 + 0,00048 = **0,0085 м вод. ст. ≈ 85 Па**

**Вывод:** Потери небольшие (менее 0,01 м водяного столба на 30 м), скорость низкая (51 см/с) — нормально для внутридомовой сети.

### 2.2 Методика выбора диаметра трубопровода

#### Формула 2.2.1: Расчётный диаметр по скорости

```
d = √(4 × Q / (π × v))  [м]

где выбирается стандартный диаметр ≥ d_расч из ГОСТ 3185 или ГОСТ 6867
```

**Стандартные внутренние диаметры труб (ГОСТ 3185 и ГОСТ 6867):**

| Условный проход (DN), мм | Стальные трубы, d_внут, мм | Пластиковые трубы (ПВХ/ПЭ), мм |
|---|---|---|
| 6 | 5,5 | 6 |
| 8 | 7,5 | 8 |
| 10 | 9,5 | 10 |
| 12 | 11 | 12 |
| 16 | 14,5 | 16 |
| 20 | 18 | 20 |
| 25 | 23 | 25 |
| 32 | 30 | 32 |
| 40 | 38 | 40 |
| 50 | 46 | 50 |
| 63 | 60 | 63 |

#### Практический пример 2.2:

Выбор диаметра для расчётного расхода Q = 1,06 м³/ч = 0,000294 м³/с:

1. Нормативная скорость: v = 0,5...1,0 м/с (выбираем v = 0,7 м/с для минимального диаметра)

2. Расчётный диаметр:
   - d = √(4 × 0,000294 / (π × 0,7)) = √(0,001176 / 2,199) = √0,000535 = **0,0231 м = 23,1 мм**

3. Выбираем из стандарта ближайший большей диаметр: **d = 25 мм** (условный проход DN 25)

4. Проверка скорости:
   - v_факт = (4 × 0,000294) / (π × 0,025²) = 0,001176 / 0,001963 = **0,60 м/с** (в норме ✓)

## РАЗДЕЛ 3: РАСЧЁТЫ РЕЗЕРВУАРОВ И НАКОПИТЕЛЕЙ

### 3.1 Методика расчёта объёма резервуара

**Нормативные документы:** СНиП 2.04.02-84 (водоснабжение), СП 31.13330.2012

#### Формула 3.1.1: Объём резервуара (накопительный)

```
V_рез = Q_сут × (t_макс - t_мин) / 24  [м³]

где:
  Q_сут   — суточное потребление, м³/сут
  t_макс  — время максимального расхода, ч (обычно 6–8 часов)
  t_мин   — время минимального расхода, ч (обычно 2–3 часа)

Типовая формула (для жилых объектов):
V_рез ≈ (0,15...0,25) × Q_сут  [м³]
```

**Или по суточному графику потребления:**

```
V_рез = Q_сут × K_рез  [м³]

где K_рез — коэффициент резервирования:
  K_рез = 0,15...0,25 для ЖД с группой водоснабжения
  K_рез = 0,20...0,30 для отдаленных объектов
  K_рез = 0,30...0,50 для объектов с нестабильным водоснабжением
```

#### Формула 3.1.2: Время заполнения резервуара

```
t_запол = V_рез / Q_подачи  [ч]

где:
  V_рез      — объём резервуара, м³
  Q_подачи   — производительность насоса / поступление воды, м³/ч
```

#### Формула 3.1.3: Время опорожнения резервуара

```
t_опор = V_рез / Q_потреб  [ч]

где:
  Q_потреб — максимальное часовое потребление, м³/ч
```

#### Таблица 3.1: Рекомендуемые объёмы резервуаров для различных объектов

| Тип объекта | Коэффициент K_рез | Примечание |
|---|---|---|
| Жилой дом, централизованное ВС | 0,15–0,20 | Кратковременное регулирование |
| Жилой комплекс с ИВСН | 0,20–0,25 | Полнота регулирования |
| Офисное здание | 0,10–0,15 | Низкие колебания потребления |
| Завод, производство | 0,30–0,40 | Технологический расход |
| Больница, детсад | 0,25–0,35 | Критичность снабжения |
| Поселок на окраине (без центрального ВС) | 0,40–0,50 | Полный запас на 1 сутки нет, на 6–12 ч |
| Система противопожарного водоснабжения | 0,50–1,0 | По объёму пожарных резервов |

#### Практический пример 3.1:

Расчёт объёма резервуара для жилого комплекса (Q_сут = 122,4 м³/сут):

1. Выбираем K_рез = 0,20 (стандартный для ЖД с центральным ВС)
2. V_рез = 122,4 × 0,20 = **24,5 м³** (берём ближайший стандартный объём **25 м³**)
3. Проверка времени заполнения (если подача от сети 2 м³/ч):
   - t_запол = 25 / 2 = **12,5 ч** (приемлемо, ночью долго)
4. Проверка времени опорожнения (при Q_ч_макс = 8,16 м³/ч):
   - t_опор = 25 / 8,16 = **3,06 ч** (хорошее регулирование пиков)

### 3.2 Методика расчёта давления в резервуаре

#### Формула 3.2.1: Минимальное давление у потребителя

```
p_потреб_мин ≥ 10 м вод. ст. (≈ 100 кПа) для жилья
p_потреб_мин ≥ 5 м вод. ст. (≈ 50 кПа) для технических целей
```

#### Формула 3.2.2: Расчётное давление в резервуаре

```
p_рез = p_потреб_мин + Δh_потери + h_гид  [м вод. ст.]

где:
  p_потреб_мин  — минимальное давление у последнего потребителя
  Δh_потери    — потери давления на трубопроводе
  h_гид        — геодезическая разница высот (+ вверх, − вниз)

Стандартное давление в резервуаре (для закрытых систем):
  p_рез = 3–6 м вод. ст. (для напорного резервуара)
  p_рез = 0,3–0,5 м вод. ст. (для атмосферного резервуара)
```

#### Практический пример 3.2:

Жилой комплекс (30-этажный) — расчёт давления в насосной станции:

**Данные:**
- Высота здания: H = 100 м (30 этажей × 3,3 м)
- Потери на трубопроводе (внутри дома): Δh = 2,5 м вод. ст.
- Потребляемое давление на верхних этажах: p_потреб = 10 м вод. ст.
- Геодезическое расположение резервуара: на земле (h_гид = 0)

**Решение:**
- p_рез = 10 + 2,5 + 100 = **112,5 м вод. ст.** (необходимо давление в насосной станции)
- или в паскалях: p = 112,5 × 9810 = **1,1 МПа**

## РАЗДЕЛ 4: РАСЧЁТЫ КАНАЛИЗАЦИОННЫХ ТРУБОПРОВОДОВ

### 4.1 Методика расчёта самотечных канализационных сетей

**Нормативные документы:** СНиП 2.04.01-85 (внутренняя канализация), СНиП 2.04.03-85 (наружная канализация)

#### Формула 4.1.1: Расход сточных вод в самотечном трубопроводе

```
Q_канал = A × v × 1000  [л/мин]

или через формулу Шези-Маннинга:

Q = A × v = A × (1/n) × √(R × i) × 1000  [л/мин]

где:
  A   — площадь сечения трубы, м²
  v   — скорость потока, м/с
  n   — коэффициент шероховатости (число Маннинга)
  R   — гидравлический радиус, R = A / P_мокр (м)
  i   — уклон дна трубы, i = Δh / L (безразмерный)
  P_мокр — смоченный периметр трубы
```

#### Таблица 4.1: Коэффициент шероховатости n (Маннинга) для канализационных труб

| Материал трубы | n (Маннинга) | Примечание |
|---|---|---|
| Керамические трубы (глина, новые) | 0,013–0,015 | Стандарт |
| Бетонные трубы (новые, гладкие) | 0,012–0,014 | |
| Асбестоцементные (новые) | 0,011–0,012 | |
| Чугунные (новые, покрытые) | 0,012–0,013 | |
| Пластиковые (ПВХ, новые) | 0,009–0,011 | Минимум трения |
| Трубы в эксплуатации 5–10 лет | +0,002–0,003 | |
| Трубы в эксплуатации 20+ лет | +0,003–0,005 | Требует очистки |
| Открытые каналы (земляные) | 0,025–0,035 | Зависит от состояния |

#### Формула 4.1.2: Нормирование скоростей потока в канализации

```
v_мин ≥ 0,6 м/с (самоочистка, не оседают частицы)
v_макс ≤ 1,5 м/с (для керамики и бетона)
v_макс ≤ 2,0 м/с (для пластика и стали)

Для открытых каналов:
v_мин ≥ 0,3–0,5 м/с (в зависимости от грунта)
v_макс ≤ 0,8–1,2 м/с (в зависимости от грунта)
```

#### Практический пример 4.1:

Расчёт диаметра канализационного стояка (Q = 1,5 л/мин = 0,0000025 м³/с):

**Условия:**
- Уклон внутридомовой сети: i = 0,03 (3% = 3 см на 1 м)
- Материал: керамическая труба (n = 0,014)
- Требуемая скорость: v ≥ 0,6 м/с (для самоочистки)

**Решение по формуле Маннинга:**

Из формулы Маннинга: Q = A × (1/n) × √(R × i)

Для круглой трубы, заполненной на h/d = 0,5 (половина диаметра):
- A = (π × d²) / 8 (площадь половины круга)
- P_мокр = (π × d) / 2 (смоченный периметр)
- R = A / P_мокр = (π × d²/8) / (π × d/2) = d / 4

Подставляем в Маннинга:
```
Q = (π × d²/8) × (1/0,014) × √((d/4) × 0,03) × 1000
0,0000025 = (π × d²/8) × 71,43 × √(0,0075 × d)
0,0000025 = (π × d²/8) × 71,43 × 0,0866 × √d
0,0000025 = 24,24 × d^2.5
d^2.5 = 1,03 × 10⁻⁷
d ≈ 0,011 м = 11 мм
```

Выбираем стандартный диаметр из ГОСТ: **d = 50 мм** (перестраховка; для повышенных расходов можно D = 40 мм).

**Проверка скорости при d = 50 мм:**
- A = (π × 0,05²) / 8 = 0,000982 м²
- R = 0,05 / 4 = 0,0125 м
- v = (1/0,014) × √(0,0125 × 0,03) = 71,43 × √0,000375 = 71,43 × 0,0194 = **1,39 м/с** ✓ (в норме)

### 4.2 Методика расчёта открытых дренажных каналов

#### Формула 4.2.1: Пропускная способность открытого канала

```
Q = A × v = A × (1/n) × √(R × i)  [м³/с]

где для трапециевидного канала:
  A = (b × h) + (m × h²)
  P_мокр = b + 2h√(1 + m²)
  R = A / P_мокр
  
  где:
    b — ширина дна канала, м
    h — глубина воды, м
    m — коэффициент откоса (1:m, например m = 1,5 для откоса 1:1,5)
    i — уклон дна, i = Δh/L
```

#### Таблица 4.2: Рекомендуемые уклоны открытых дренажных каналов

| Грунт | Рекомендуемый уклон i | v_макс, м/с | Примечание |
|---|---|---|---|
| Глина, суглинок | 0,002–0,005 (0,2–0,5%) | 0,6–0,8 | Низкий уклон, низкая скорость |
| Песок, супесь | 0,005–0,01 (0,5–1%) | 0,7–1,0 | Средний уклон |
| Гравий, галька | 0,01–0,02 (1–2%) | 1,0–1,2 | Высокий уклон |
| Скальный грунт | 0,02–0,05 (2–5%) | 1,5–2,0 | Очень высокий уклон |

#### Практический пример 4.2:

Расчёт открытого дренажного канала вокруг фундамента (Q_дренаж = 0,002 м³/с = 2 л/с):

**Условия:**
- Грунт: суглинок
- Уклон: i = 0,005 (0,5%)
- Форма: трапециевидный канал с откосом 1:1,5 (m = 1,5)
- n = 0,03 (для земляного канала)

**Выбор размеров:**

Предварительно выбираем: b = 0,3 м, h = 0,3 м

1. Площадь сечения:
   - A = (0,3 × 0,3) + (1,5 × 0,3²) = 0,09 + 0,135 = 0,225 м²

2. Смоченный периметр:
   - P = 0,3 + 2 × 0,3 × √(1 + 1,5²) = 0,3 + 0,6 × √3,25 = 0,3 + 1,08 = 1,38 м

3. Гидравлический радиус:
   - R = 0,225 / 1,38 = 0,163 м

4. Скорость (Маннинга):
   - v = (1/0,03) × √(0,163 × 0,005) = 33,33 × √0,000815 = 33,33 × 0,0286 = **0,95 м/с** ✓

5. Расход:
   - Q = 0,225 × 0,95 = **0,214 м³/с = 214 л/с** (перебор, уменьшаем размеры)

**Пересчёт с b = 0,15 м, h = 0,15 м:**
- A = 0,0225 + 0,034 = 0,0565 м²
- P = 0,15 + 0,3 × 1,80 = 0,69 м
- R = 0,082 м
- v = 33,33 × √0,00041 = 0,67 м/с ✓
- Q = 0,0565 × 0,67 = **0,038 м³/с** (ещё много)

**Окончательный выбор с b = 0,1 м, h = 0,1 м:**
- A = 0,01 + 0,015 = 0,025 м²
- P = 0,1 + 0,2 × 1,80 = 0,46 м
- R = 0,054 м
- v = 33,33 × √0,00027 = 0,55 м/с (чуть ниже нормы)
- Q = 0,025 × 0,55 = **0,014 м³/с = 14 л/с** (в пределах ±7 × норма, приемлемо)
