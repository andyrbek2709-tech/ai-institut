## 🧮 Два основных расчёта

### 1️⃣ `tx_heat_balance` — Тепловая мощность

| Вопрос | Ответ |
|--------|-------|
| **Формула** | `Q = G · c · ΔT` |
| **Где найти** | THERMAL_METHODS_STANDARDS.md раздел 1 |
| **Пример в гайде** | THERMAL_CALCULATIONS_QUICK_GUIDE.md: "Выбор котла" |
| **Для чего** | Подбор котла, теплового насоса, кондиционера |
| **Входные данные** | Расход (кг/с), теплоёмкость (кДж/(кг·°C)), ΔT (°C) |
| **Стандарты** | СП 60.13330.2020, СП 50.13330.2012, ISO 13789:2017 |
| **Типовой результат** | кВт (или Вт) |

**Когда использовать:**
- ✅ Система уже выбрана по температурам (есть ΔT)
- ✅ Нужна мощность при известном расходе
- ❌ Нужна температура в конечной точке сети (используйте теплопотери)
- ❌ Нужен расход по требуемой мощности

**Типовые значения для расчётов:**
- Отопление: ΔT = 20-30°C, c = 4.18 кДж/(кг·°C)
- Охлаждение: ΔT = 5-10°C, c = 4.18 кДж/(кг·°C)
- Вентиляция: ΔT = 20-40°C, c = 1.0 кДж/(кг·°C)

### 2️⃣ `tt_pressure_drop` — Потери давления

| Вопрос | Ответ |
|--------|-------|
| **Формула** | `ΔP = λ · (L/d) · (ρ · v² / 2)` |
| **Где найти** | THERMAL_METHODS_STANDARDS.md раздел 2 |
| **Пример в гайде** | THERMAL_CALCULATIONS_QUICK_GUIDE.md: "Выбор насоса" |
| **Для чего** | Подбор насоса, проверка скоростей, расчёт напора |
| **Входные данные** | Длина (м), диаметр (м), расход (м³/с), материал, плотность |
| **Стандарты** | СНиП 2.04.01-85*, СНиП 2.04.02-84*, СП 30.13330.2020, ISO 5241:2018 |
| **Типовой результат** | кПа или м вод.ст. |

**Когда использовать:**
- ✅ Нужен напор насоса (потери в трубах)
- ✅ Нужно проверить скорость потока
- ✅ Сеть с переменным диаметром (рассчитайте участки)
- ❌ Непомещаемое течение (нужна компрессибилити)
- ❌ Осаждение в трубах (нужен расчёт наносов)

**Типовые значения для расчётов:**
- Водопровод: v = 0.5-1.5 м/с, λ = 0.01-0.04
- Отопление: v = 0.3-2 м/с, λ = 0.02-0.05
- Гидросистемы: v = 2-5 м/с, λ = 0.015-0.035

## 🚀 Быстрые ответы на типовые вопросы

### Вопрос: "Какой котёл нужен для дома?"
**Решение:** THERMAL_CALCULATIONS_QUICK_GUIDE.md → Задача 1  
**Ответ:** Q = 20 кВт для 200 м²  
**Процесс:** расход → формула → мощность (+20% запас)

### Вопрос: "Какую трубу выбрать?"
**Решение:** THERMAL_CALCULATIONS_QUICK_GUIDE.md → Задача 2  
**Ответ:** диаметр по скорости v ≤ 1-2 м/с  
**Процесс:** расход → скорость → диаметр (норма СНиП)

### Вопрос: "Какой насос нужен?"
**Решение:** THERMAL_CALCULATIONS_QUICK_GUIDE.md → Задача 3  
**Ответ:** напор по потерям ΔP  
**Процесс:** Re → λ → потери → сумма → выбор насоса

### Вопрос: "Потери давления в сети?"
**Решение:** `tt_pressure_drop` вручную или в EngHub  
**Ответ:** используйте формулу Дарси–Вейсбаха  
**Процесс:** число Re → коэффициент λ → ΔP = λ(L/d)(ρv²/2)

### Вопрос: "Какая мощность теплопотерь?"
**Решение:** `tx_heat_balance` для теплопотерь из системы  
**Ответ:** Q_потери = π·d_н·L·k·ΔT  
**Процесс:** диаметр → длина → коэффициент k → мощность
