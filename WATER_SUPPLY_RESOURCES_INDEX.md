### 2. WATER_SUPPLY_EXAMPLES.md
**Назначение:** 3 реальных проекта с пошаговыми расчётами

**Проекты:**

#### Проект 1: Жилой комплекс (250 квартир, 800 человек, 30 этажей)
```
Структура расчётов:
├── 1. Водопотребление
│   ├── Суточное: 136 м³/сут
│   ├── Максимальное часовое: 9,07 м³/ч (K_макс = 1,6)
│   ├── Расход по приборам: 1,06 м³/ч (на квартиру)
│   └── Баланс: проверка согласованности

├── 2. Гидравлика
│   ├── Выбор диаметра: DN 63 мм (v = 0,89 м/с)
│   ├── Потери в стояке (99 м): 1,05 м вод. ст.
│   └── Общие потери: 1,5 м вод. ст.

├── 3. Резервуар
│   ├── Объём: 30 м³ (K_рез = 0,20)
│   ├── Время заполнения: 15 ч (при подаче 2 м³/ч)
│   └── Время опорожнения: 3,3 ч (при пиковом потреблении)

├── 4. Давление в насосе
│   ├── Требуемое: 110,5 м вод. ст. = 11 бара = 1,08 МПа
│   ├── Мощность насоса: ~40 кВт
│   └── Рекомендуемая модель: Grundfos TP 50-160/2 или аналог

└── 5. Канализация
    ├── Расход сточных вод: 122,4 м³/сут (= 0,9 × потребление)
    ├── Внутридомовые стояки: 3 × DN 100 мм
    ├── Наружная сеть: DN 150 мм, L = 150 м, уклон 0,9%
    └── Глубина залегания: 1,2–2,5 м (ниже промерзания)
```

#### Проект 2: Офисное здание (250 человек)
```
Краткие параметры:
- Q_ч_макс = 0,94 м³/ч
- DN 25 мм магистраль
- V_рез = 1 м³
- Простая инфраструктура
```

#### Проект 3: Производственное предприятие

Краткие параметры:
- Q_технологич = 50 м³/ч
- Q_общ = 67,5 м³/ч
- V_рез = 200 м³
- P = 2 × 10 кВт (с резервом)

### 3. water-calculator.html
**Назначение:** Интерактивный браузерный калькулятор всех расчётов

**Вкладки и функции:**

#### Вкладка 1: Водопотребление
```
Расчёт 1: Суточное потребление
├── Ввод: численность, удельное потребление, K_загр
├── Выход: Q_сут, Q_ч_макс, Q_ч_мин, пиковый расход (л/мин)
└── Стандарты: значения q_м из СНиП 2.04.01

Расчёт 2: Расход по приборам
├── Ввод: тип оборудования, количество, K_одновременности
├── Выход: Σq_пр, Q_расч, пиковый расход
└── Справочник: расходы всех приборов (СП 31.13330)
```

#### Вкладка 2: Гидравлика
```
Расчёт 1: Скорость и диаметр трубы
├── Ввод: расход, тип системы (холодное/горячее/канализация)
├── Выход: расчётный диаметр, рекомендуемый DN, фактическая скорость
└── Валидация: предупреждение если v < 0,5 или v > 1,5 м/с

Расчёт 2: Потери давления (Дарси-Вейсбаха)
├── Ввод: диаметр, длина, скорость, материал, местные сопротивления
├── Расчёты: Re число (режим течения), λ (по Муди)
├── Выход: потери на трение, потери на местные, общие потери
└── Единицы: м вод. ст., Па, кПа
```

#### Вкладка 3: Резервуары
```
Расчёт 1: Объём резервуара
├── Ввод: суточное потребление, тип объекта
├── Выход: объём (м³), время заполнения, время опорожнения
└── K_рез: для жилья 0,20, офиса 0,12, производства 0,35–0,50

Расчёт 2: Давление в насосе
├── Ввод: высота здания, потери, минимальное давление
├── Выход: требуемое давление в м вод. ст., кПа, атм, бара
└── Применение: выбор насоса по характеристикам
```

#### Вкладка 4: Канализация
```
Расчёт 1: Самотечные сети (Маннинга)
├── Ввод: расход, материал трубы, уклон
├── Выход: минимальный диаметр, рекомендуемый DN, скорость
└── Валидация: v ≥ 0,6 м/с (самоочистка)

Расчёт 2: Открытые дренажные каналы
├── Ввод: расход дренажа, тип грунта, уклон, ширина дна
├── Выход: требуемая высота воды, скорость потока
└── Геометрия: трапециевидный канал с откосом 1:1,5
```

#### Вкладка 5: Справочные данные
```
Таблицы (не требуют расчётов):
├── Диаметры труб (ГОСТ 3185, ГОСТ 6867)
├── Нормативные скорости (СНиП 2.04.01)
├── Расходы приборов (СП 31.13330)
└── Единицы давления (м вод. ст. → кПа → бар → атм)
```

**Технические характеристики:**
- JavaScript реализация всех формул
- Числовой ввод с валидацией
- Раскрывающиеся списки для стандартных значений
- Вывод с цветовой кодировкой (успех/предупреждение/ошибка)
- Responsive дизайн (работает на мобильных)

**Когда использовать:**
- ✓ Быстрый расчёт одного параметра
- ✓ Проверка расчётов вручную
- ✓ Выбор диаметра по расходу (самый частый случай)
- ✓ Полевые работы (расчёты прямо на площадке)

**Ссылка:** `d:\ai-institut\water-calculator.html` (откройте в браузере)

**Запуск:**
```bash
# Откройте файл в браузере (Chrome, Firefox, Safari, Edge)
open d:\ai-institut\water-calculator.html
# или скопируйте путь в адресную строку

### Когда применять каждый стандарт

```
СНиП 2.04.01 / СП 31.13330  → Внутридомовое водоснабжение и канализация

ГОСТ 3185 (трубы стальные)  → Если используются стальные трубопроводы
ГОСТ 6867 (пластик)         → Если используются полиэтиленовые трубы
