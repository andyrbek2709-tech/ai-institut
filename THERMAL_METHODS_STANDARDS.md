## 1. TX_HEAT_BALANCE — Тепловая мощность (Нагрев/Охлаждение)

### 1.1. Основная методика

**Название:** Расчёт теплового потока при изменении температуры среды

**Стандарты и нормативы:**
- **Справочник проектировщика. Отопление, вентиляция и кондиционирование воздуха** (Авторы: И. Г. Староверов, В. Н. Богославский)
- **СП 60.13330.2020** — Отопление, вентиляция и кондиционирование воздуха (актуализация СНиП 2.04.05-91*)
- **СП 50.13330.2012** — Тепловая защита зданий (актуализация СНиП 23-02-2003)
- **ГОСТ 30494-2011** — Здания жилые и общественные. Параметры микроклимата в помещениях
- **ISO 13789:2017** — Thermal performance of buildings — Transmission and ventilation heat transfer coefficients
- **EN 12098-1:2013** — Controls of heating systems — Part 1: Controls for heating systems

**Область применения:**
- Расчёт тепловой мощности систем отопления и охлаждения
- Определение требуемой мощности теплогенератора (котла, теплового насоса)
- Оценка охлаждающей способности системы кондиционирования
- Проверка теплового баланса для любых сред (вода, воздух, масло, пар и т.д.)

**Основная формула:**
```
Q = G · c · ΔT
```

Где:
- **Q** — тепловая мощность, кВт (или Вт)
- **G** — массовый расход среды, кг/с
- **c** — удельная теплоёмкость среды, кДж/(кг·°C)
- **ΔT** — разность температур (на входе и выходе), °C (или К)

**Входные параметры:**

| Параметр | Обозначение | Единица | Диапазон | Типовые значения | Источник |
|----------|-------------|---------|----------|------------------|----------|
| Массовый расход | G | кг/с | 0.001 – 10000 | 1 – 100 | СП 60.13330 табл. 7.2 |
| Теплоёмкость воды | c | кДж/(кг·°C) | 4.15 – 4.25 | 4.18 | ГОСТ 29835-2020 |
| Теплоёмкость воздуха | c | кДж/(кг·°C) | 0.9 – 1.1 | 1.0 | Справочник |
| Теплоёмкость масла | c | кДж/(кг·°C) | 1.8 – 2.2 | 2.0 | ГОСТ 17309-72 |
| Разность температур | ΔT | °C (К) | 0.1 – 1000 | 10 – 50 | Проектные условия |

**Теплоёмкость различных сред (при ~20°C, атм. давлении):**

| Среда | c, кДж/(кг·°C) | ρ, кг/м³ | Применение |
|-------|----------------|----------|-----------|
| Вода | 4.18 | 1000 | Системы отопления, охлаждения, ГВС |
| Воздух | 1.0 | 1.2 | Вентиляция, кондиционирование |
| Пар (перегретый) | 2.0 | 0.6–50 | Паровые сети отопления |
| Этиленгликоль | 3.7 | 1100 | Антифризные системы |
| Масло техническое | 2.0 | 850–900 | Промышленные тепловые сети |
| Молоко | 3.9 | 1030 | Молочные производства |

**Примеры расчётов:**

#### Пример 1: Система отопления (вода)
```
Дано:
- Расход воды G = 2.5 кг/с (расход по циркуляции 150 м³/ч = 41.7 л/с ≈ 2.5 кг/с)
- Теплоёмкость воды c = 4.18 кДж/(кг·°C)
- Разность температур ΔT = 20°C (от 70°C к 50°C, типовая ΔT для отопления)

Расчёт:
Q = 2.5 · 4.18 · 20 = 209 кВт

Вывод: Для обогрева системы при таких параметрах требуется котёл мощностью ~210 кВт
```

#### Пример 2: Система охлаждения (вода)
```
Дано:
- Расход охлаждающей воды G = 5 кг/с (циркуляция 18 м³/ч через градирню)
- Теплоёмкость воды c = 4.18 кДж/(кг·°C)
- Разность температур ΔT = 5°C (от 32°C к 27°C, типовая ΔT для охлаждения)

Расчёт:
Q = 5 · 4.18 · 5 = 104.5 кВт

Вывод: Система охлаждения должна отводить ~105 кВт тепла
```

#### Пример 3: Вентиляция (воздух)
```
Дано:
- Расход воздуха V = 1000 м³/ч = 0.278 м³/с
- Плотность воздуха ρ = 1.2 кг/м³ → G = 0.278 · 1.2 ≈ 0.33 кг/с
- Теплоёмкость воздуха c = 1.0 кДж/(кг·°C)
- Разность температур ΔT = 30°C (нагрев холодного воздуха с −10°C до +20°C)

Расчёт:
Q = 0.33 · 1.0 · 30 = 9.9 кВт ≈ 10 кВт

Вывод: Для нагрева приточного воздуха требуется ~10 кВт (система предварительного нагрева)
```

**Расширенная формула (с учётом потерь):**

Когда нужно учитывать потери в окружающую среду:
```
Q_нужн = Q + Q_потери
```

где Q_потери — тепловые потери из системы (трубопровода, батареи, оборудования).

**Для теплопотерь через стенку трубы:**
```
Q_потери = π · d_н · L · k · ΔT_ср
```

Где:
- d_н — наружный диаметр трубы, м
- L — длина участка, м
- k — коэффициент теплоотдачи, Вт/(м²·°C) (для изолированной трубы 5–10 Вт/(м²·°C))
- ΔT_ср — средняя разность температур между теплоносителем и окружающей средой, °C

## 2. TT_PRESSURE_DROP — Потери давления (Дарси–Вейсбах)

### 2.1. Основная методика

**Название:** Расчёт гидравлических потерь на трение по длине трубопровода

**Стандарты и нормативы:**
- **СНиП 2.04.01-85*** (в редакции ГОСТ Р 51858-2002) — Внутренний водопровод и канализация зданий
- **СНиП 2.04.02-84*** — Водоснабжение. Наружные сети и сооружения
- **СП 30.13330.2020** — Внутренний водопровод и канализация зданий (актуализация СНиП 2.04.01-85)
- **СП 73.13330.2016** — Внутренние санитарно-технические системы зданий
- **ГОСТ 32957-2014** — Трубопроводы стальные сварные. Технические условия
- **ISO 5241:2018** — Hydrodynamic fluid power systems and components — Hose assemblies for rated working pressures
- **ГОСТ Р 51858-2002** — Гидравлические системы и компоненты. Гидравлические жидкости. Классификация, техническое обслуживание
- **Справочник. Гидравлика трубных систем** (Авторы: И. Е. Идельчик, А. Л. Майвальд)

**Область применения:**
- Расчёт потерь давления на трение при течении жидкости (вода, масло) в круглых трубах
- Проверка пропускной способности трубопровода
- Подбор насоса (определение требуемого напора)
- Анализ гидравлической сети (магистральные и ответвляющиеся трубопроводы)
- Системы отопления, водоснабжения, кондиционирования, промышленные гидросистемы

**Применима для:**
- Турбулентное течение: Re > 2300 (основной расчётный режим)
- Переходный режим: 2300 ≤ Re ≤ 10000 (требуется уточнение коэффициента трения)
- Ламинарное течение: Re < 2300 (редко в инженерной практике)

### 2.2. Основная формула (Дарси–Вейсбах)

```
ΔP = λ · (L/d) · (ρ · v² / 2)
```

Где:
- **ΔP** — потери давления на трение, Па (или кПа, бар)
- **λ** — коэффициент гидравлического трения (безразмерный)
- **L** — длина участка трубопровода, м
- **d** — внутренний диаметр трубы, м
- **ρ** — плотность жидкости, кг/м³
- **v** — средняя скорость потока, м/с

**Альтернативная форма (через расход):**

```
ΔP = λ · (L/d) · (ρ · Q² / (2 · A²))
```

Где:
- **Q** — объёмный расход, м³/с
- **A** — площадь сечения трубы, м² = π·d²/4

### 2.3. Коэффициент гидравлического трения λ

Коэффициент λ зависит от **режима течения** (числа Рейнольдса) и **относительной шероховатости** трубы.

#### 2.3.1. Число Рейнольдса

```
Re = (v · d) / ν = (ρ · v · d) / μ
```

Где:
- **ν** — кинематическая вязкость жидкости, м²/с
- **μ** — динамическая вязкость, Па·с = кг/(м·с)
- **ρ** — плотность жидкости, кг/м³

**Вязкость различных жидкостей (при 20°C, 1 атм):**

| Жидкость | ν, м²/с | μ, мПа·с | ρ, кг/м³ | Применение |
|----------|---------|----------|----------|-----------|
| Вода | 1.0 × 10⁻⁶ | 1.0 | 1000 | Водопровод, отопление, охлаждение |
| Масло минеральное | 40 × 10⁻⁶ | 40 | 850–900 | Гидравлические системы |
| Этиленгликоль 50% | 5 × 10⁻⁶ | 5 | 1050 | Антифризные системы отопления |
| Бензин | 0.6 × 10⁻⁶ | 0.6 | 750 | Топливные системы |
| Воздух (при 20°C, 1 бар) | 15 × 10⁻⁶ | 0.018 | 1.2 | Пневматические системы |

#### 2.3.2. Режимы течения

| Режим | Условие | λ (формула) | Применение |
|-------|---------|-----------|-----------|
| **Ламинарный** | Re < 2300 | λ = 64/Re | Редко для инженерных сетей |
| **Переходный** | 2300 ≤ Re ≤ 10000 | Диаграмма Муди или уточнённые формулы | Переходная зона (требуется проверка) |
| **Турбулентный** | Re > 10000 | Зависит от шероховатости | Основной расчётный режим |

#### 2.3.3. Коэффициент трения для турбулентного режима

Для турбулентного течения λ зависит от относительной шероховатости трубы **Δ/d**:

**а) Формула Никурадзе (для гладких труб, Re > 10⁴):**

```
λ = 0.3164 / Re^0.25  (при Re < 10⁵)
или
λ = 0.0032 + 0.221/Re^0.237  (универсальнее)
```

**б) Формула Колбрука–Уайта (неявная, универсальная):**

```
1/√λ = -2 · lg((Δ/(3.71·d)) + (2.51/(Re·√λ)))
```

Решается итерационно или по диаграмме Муди.

**в) Явная формула Холла–Кондратьева (удобнее для расчётов):**

```
λ = 0.11 · ((Δ/d) + (68/Re))^0.25
```

**г) Формула Блазиуса (для гладких труб, 4000 < Re < 10⁵):**

```
λ = 0.316 / Re^0.25
```

#### 2.3.4. Абсолютная и относительная шероховатость

**Абсолютная шероховатость различных материалов труб:**

| Материал | Δ, мм | Δ/d при d=50 мм | Примечание |
|----------|-------|----------------|-----------|
| Полиэтилен (ПЭ) | 0.002–0.01 | 0.00004–0.0002 | Гладкие, новые |
| Пластик (ПВХ, полипропилен) | 0.003–0.01 | 0.00006–0.0002 | Гладкие |
| Сталь новая гальванизированная | 0.05–0.1 | 0.001–0.002 | Слабая шероховатость |
| Сталь б/у (9–25 лет) | 0.15–0.5 | 0.003–0.01 | Коррозия |
| Сталь сильно заржавленная | 0.5–2 | 0.01–0.04 | Требуется замена |
| Чугун новый | 0.2–0.4 | 0.004–0.008 | Битумизированный |
| Чугун старый | 1.0–1.5 | 0.02–0.03 | Коррозия |
| Медь | 0.0015–0.01 | 0.00003–0.0002 | Практически гладкие |
| Асбоцемент | 0.1–0.5 | 0.002–0.01 | Деградирует со временем |

### 2.4. Примеры расчётов

#### Пример 1: Стальной трубопровод отопления

```
Дано:
- Диаметр d = 50 мм = 0.05 м
- Длина L = 100 м
- Расход Q = 20 м³/ч = 0.00556 м³/с
- Жидкость: вода, ρ = 1000 кг/м³, ν = 1.0 × 10⁻⁶ м²/с
- Материал: сталь б/у, Δ = 0.3 мм

Шаг 1. Расчёт скорости потока:
v = Q / A = Q / (π·d²/4)
v = 0.00556 / (3.14159 · 0.05² / 4) = 0.00556 / 0.001964 = 2.83 м/с

Шаг 2. Расчёт числа Рейнольдса:
Re = v · d / ν = 2.83 · 0.05 / (1.0 × 10⁻⁶) = 141 500

Режим: турбулентный (Re > 10 000)

Шаг 3. Расчёт относительной шероховатости:
Δ/d = 0.3 / 50 = 0.006

Шаг 4. Расчёт коэффициента трения (формула Холла–Кондратьева):
λ = 0.11 · ((0.006) + (68/141500))^0.25
λ = 0.11 · (0.006 + 0.00048)^0.25
λ = 0.11 · (0.00648)^0.25
λ = 0.11 · 0.2849 = 0.0313

Шаг 5. Расчёт потерь давления:
ΔP = λ · (L/d) · (ρ · v² / 2)
ΔP = 0.0313 · (100 / 0.05) · (1000 · 2.83² / 2)
ΔP = 0.0313 · 2000 · 4007
ΔP = 251 000 Па = 251 кПа = 2.51 бар

Или в метрах столба воды:
h = ΔP / (ρ · g) = 251 000 / (1000 · 9.81) = 25.6 м

Вывод: На трубопровод длиной 100 м потребуется насос, обеспечивающий 
напор не менее 25.6 м вод. ст. для преодоления потерь на трение.
```

#### Пример 2: Пластиковый водопровод (ПВХ)

```
Дано:
- Диаметр d = 32 мм = 0.032 м (стандартная труба ПВХ)
- Длина L = 50 м
- Расход Q = 5 м³/ч = 0.00139 м³/с (типовой расход для дома)
- Жидкость: вода, ρ = 1000 кг/м³, ν = 1.0 × 10⁻⁶ м²/с
- Материал: пластик новый, Δ = 0.01 мм = 0.00001 м

Шаг 1. Скорость потока:
v = 0.00139 / (π · 0.032² / 4) = 0.00139 / 0.000804 = 1.73 м/с

Шаг 2. Число Рейнольдса:
Re = 1.73 · 0.032 / (1.0 × 10⁻⁶) = 55 360

Режим: турбулентный

Шаг 3. Относительная шероховатость:
Δ/d = 0.01 / 32 = 0.0003125

Шаг 4. Коэффициент трения:
λ = 0.11 · ((0.0003125) + (68/55360))^0.25
λ = 0.11 · (0.0003125 + 0.00123)^0.25
λ = 0.11 · (0.0015425)^0.25
λ = 0.11 · 0.1118 = 0.0123

Шаг 5. Потери давления:
ΔP = 0.0123 · (50 / 0.032) · (1000 · 1.73² / 2)
ΔP = 0.0123 · 1562.5 · 1497.5
ΔP = 28 900 Па ≈ 29 кПа = 0.29 бар = 2.9 м вод. ст.

Вывод: Потери на трение в пластиковом трубопроводе невелики, 
достаточно статического давления или слабого насоса.
```

#### Пример 3: Гидравлическая система с маслом

```
Дано:
- Диаметр d = 20 мм = 0.02 м (гидрошланг)
- Длина L = 30 м
- Расход Q = 60 л/мин = 0.001 м³/с
- Жидкость: масло минеральное, ρ = 875 кг/м³, ν = 40 × 10⁻⁶ м²/с (вязкость ISO VG 46)
- Шланг гидравлический, Δ ≈ 0.1 мм

Шаг 1. Скорость потока:
v = 0.001 / (π · 0.02² / 4) = 0.001 / 0.000314 = 3.18 м/с

Шаг 2. Число Рейнольдса:
Re = 3.18 · 0.02 / (40 × 10⁻⁶) = 1590

Режим: ЛАМИНАРНЫЙ (Re < 2300)

Шаг 3. Коэффициент трения для ламинарного течения:
λ = 64 / Re = 64 / 1590 = 0.0402

Шаг 4. Потери давления:
ΔP = 0.0402 · (30 / 0.02) · (875 · 3.18² / 2)
ΔP = 0.0402 · 1500 · 4429
ΔP = 268 000 Па ≈ 268 кПа = 2.68 бар

Вывод: Для гидросистемы требуется давление на насос минимум 2.68 бар 
плюс запас на местные сопротивления (клапаны, фитинги).
```

### 2.5. Местные сопротивления

При полном расчёте гидравлической сети учитываются не только потери на трение, но и **местные сопротивления** (колена, тройники, клапаны, задвижки):

```
ΔP_местное = ξ · (ρ · v² / 2)
```

Где **ξ** — коэффициент местного сопротивления:

| Элемент | ξ | Примечание |
|---------|---|-----------|
| Плавный вход в трубу | 0.05–0.10 | Скруглённая кромка |
| Резкий вход в трубу | 0.5 | Прямоугольный срез |
| Плавный выход из трубы | 0.05–0.10 | Скруглённая кромка |
| Резкий выход из трубы | 1.0 | Прямоугольный срез |
| Колено 90° (r/d = 1) | 0.9–1.3 | r — радиус поворота |
| Колено 90° (r/d = 2) | 0.6–0.8 | Плавнее |
| Колено 45° | 0.4–0.6 | Менее острое |
| Тройник прямой (расход проходит прямо) | 0.1–0.3 | Минимальное сопротивление |
| Тройник боковое ответвление | 1.0–2.0 | Турбулентность |
| Задвижка полностью открыта | 0.1–0.2 | Гидравлический клапан |
| Задвижка на 75% открыта | 0.5–1.0 | Увеличивается с закрытием |
| Задвижка на 50% открыта | 1.5–3.0 | |
| Задвижка на 25% открыта | 5.0–10 | Сильное дросселирование |
| Обратный клапан | 2.0–5.0 | В зависимости от типа |
| Фильтр (слабозагрязненный) | 0.5–1.5 | Зависит от забиваемости |
| Фильтр (загрязненный) | 5.0–20+ | Требует очистки |

**Суммарные потери в сети:**
```
ΔP_общие = ΔP_трение + Σ ΔP_местные
```

### 2.6. Рекомендуемые скорости потока

Чтобы избежать повышенного шума, эрозии и вибрации в трубопроводах, необходимо соблюдать диапазоны скоростей:

| Тип системы | v, м/с | Примечание |
|-------------|--------|-----------|
| Водопровод ХВС | 0.5–1.5 | Норма СНиП 2.04.01-85 ≤ 2 м/с |
| Водопровод ГВС | 0.3–1.2 | Снижение потерь |
| Отопление закрытое | 0.3–2.0 | Типично 0.5–1.5 м/с |
| Отопление открытое | 0.2–0.8 | Меньше, чтобы избежать шума |
| Циркуляция охлаждения | 0.5–2.0 | Эффективность теплообмена |
| Гидравлические системы | 2.5–5.0 | На входе насоса ≤ 1.2 м/с |
| Пневматические системы | 4–15 | В зависимости от давления |

## 3. Сравнение альтернативных методик

### 3.1. Для расчёта тепловой мощности

#### Метод 1: Простая формула Q = G·c·ΔT (используется в EngHub)
- **Плюсы:** Простая, быстрая, без итераций
- **Минусы:** Не учитывает вязкоупругость, изменение свойств с температурой
- **Применение:** Оценочные расчёты, предварительное проектирование

#### Метод 2: Уточнённые формулы с температурной коррекцией
```
c(T) = c₀ · (1 + α · ΔT)
```
- **Плюсы:** Точнее при больших ΔT
- **Минусы:** Требует больше данных
- **Применение:** Проектирование систем с ΔT > 50°C

#### Метод 3: Через энтальпию (для пара)
```
Q = G · (h₁ - h₂)
```
- **Плюсы:** Правильно для фазовых переходов
- **Минусы:** Требует паровых таблиц
- **Применение:** Паровые сети, системы конденсации

### 3.2. Для расчёта потерь давления

#### Метод 1: Формула Дарси–Вейсбаха (универсальная)
- **Плюсы:** Точная, универсальная для всех режимов
- **Минусы:** Требует определения λ итерационно
- **Применение:** Все типы трубопроводов

#### Метод 2: Диаграмма Муди (графический)
- **Плюсы:** Наглядно, историческая база
- **Минусы:** Требует графика, менее точна в цифровом виде
- **Применение:** Когда нет вычислительных средств

#### Метод 3: Таблицы потерь (СНиП, справочники)
- **Плюсы:** Быстро, без расчётов
- **Минусы:** Ограниченная область применения
- **Применение:** Стандартные системы водопровода/отопления

#### Метод 4: Формула Хейзена–Вильямса (для воды)
```
ΔP = 10.67 · L · Q^1.852 / (C^1.852 · d^4.871)
```
где C — коэффициент трения (американский стандарт)
- **Плюсы:** Практическое применение в США
- **Минусы:** Менее универсальна
- **Применение:** Системы водоснабжения в странах с американским стандартом

## 6. Резюме

**tx_heat_balance** требует знаний:
- Физических свойств различных теплоносителей
- Методик расчёта тепловой нагрузки
- Стандартов по параметрам микроклимата

**tt_pressure_drop** требует знаний:
- Гидравлики течения жидкостей
- Определения коэффициента трения λ
- Учёта местных сопротивлений в сетях
- Выбора оптимальных диаметров труб и скоростей потока

Оба расчёта являются фундаментальными для проектирования инженерных систем и должны применяться совместно при расчёте реальных тепловых сетей.
