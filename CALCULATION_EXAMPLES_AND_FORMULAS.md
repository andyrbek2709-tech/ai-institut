# Примеры и Формулы для Calculations Platform

**Документация примеров расчётов для встраивания в приложение.**

---

## 1. Быстрый справочник формул

### 1.1 Мощность и ток

```
ОДНОФАЗНАЯ СЕТЬ (230V):
P = U × I × cos(φ)                [Вт]
I = P / (U × cos(φ))              [А]
S = √(P² + Q²)                    [ВА]
cos(φ) = P / S                    [-]

ТРЁХФАЗНАЯ СЕТЬ (380V):
P = √3 × U × I × cos(φ)           [Вт]
I = P / (√3 × U × cos(φ))         [А]
S = √3 × U × I                    [ВА]

ПАРАЛЛЕЛЬНЫЕ НАГРУЗКИ:
P_total = P₁ + P₂ + ... + Pₙ      [Вт]
Q_total = Q₁ + Q₂ + ... + Qₙ      [ВАр]
S_total = √(P_total² + Q_total²)  [ВА]
```

### 1.2 Кабели

```
СОПРОТИВЛЕНИЕ:
R = ρ × L / S                     [Ом]
ρ(медь) = 0.0178 Ом·мм²/м        [при 20°C]
ρ(алюминий) = 0.0286 Ом·мм²/м    [при 20°C]

ПОТЕРИ НАПРЯЖЕНИЯ:
ΔU (В) = (ρ × L × I) / S          [В]
ΔU (%) = ΔU / U_ном × 100         [%]

НАГРЕВ ПРИ ПЕРЕГРУЗКЕ:
T_линия = T_окр + I²×R/k          [°C]
k = теплоотвод в окружение        [зависит от условий]
```

### 1.3 Освещение

```
СВЕТОВОЙ ПОТОК:
Φ (лм) = (E × S × K_неравн × K_отр) / η   [лм]

КОЛИЧЕСТВО СВЕТИЛЬНИКОВ:
N = (S × E × K_запас) / (Φ_одного × η)   [-]

ОСВЕЩЁННОСТЬ:
E = Φ × η / (S × K_неравн × K_отр)       [люкс]
```

### 1.4 Измерения

```
ЛИНЕЙНОЕ МАСШТАБИРОВАНИЕ:
Y = Y_min + (X - X_min) / (X_max - X_min) × (Y_max - Y_min)  [ед.]

МАСШТАБИРОВАНИЕ 4-20mA:
Y = Y_min + (I - 4) / 16 × (Y_max - Y_min)  [ед.]

ПОГРЕШНОСТЬ:
ΔX = ± (класс / 100) × X_max     [ед.]
δ = ΔX / X_изм × 100%            [%]
```

### 1.5 Вентиляция и отопление

```
ОБЪЁМ ВОЗДУХА:
L = V × K  (по кратности)        [м³/ч]
L = n × l_норм  (по людям)       [м³/ч]
L = S × q_удельн  (по площади)   [м³/ч]

СКОРОСТЬ В ТРУБЕ:
V = L / F                         [м/с]
F = π × D² / 4                    [м²]

ПОТЕРИ ДАВЛЕНИЯ:
ΔP = λ × (L / D) × (ρ × V²) / 2  [Па]

ТЕПЛОПОТЕРИ:
Q = A × U × ΔT                    [Вт]
ΔT = T_внутри - T_снаружи        [К]

ТЕПЛОПОТЕРИ НА ВЕНТИЛЯЦИЮ:
Q_вент = L × 0.33 × ΔT           [Вт]
L — в м³/ч, ΔT в К
```

## 2. Интерактивные примеры (для UI калькулятора)

### Пример 1: Автомат и кабель для квартиры

**Задача:** Рассчитать требуемый автомат и кабель для линии "Кухня".

**Входные данные:**
```
├─ Мощность: 4500 Вт
├─ Напряжение: 230 В
├─ cos(φ): 0.95 (смешанная нагрузка)
├─ Длина кабеля: 25 м (туда-обратно: 50 м)
└─ Окружающая температура: 30°C
```

**Расчёт по шагам:**

```
ШАГ 1: Ток в нормальном режиме
I = P / (U × cos(φ)) = 4500 / (230 × 0.95) = 20.5 А

ШАГ 2: Расчётный ток для выбора автомата
I_расч = 20.5 × 1.25 = 25.6 А → автомат 32 А (стандартный)

ШАГ 3: Проверка потерь напряжения
ΔU = (0.0178 × 50 × 20.5) / 2.5 = 7.3 В = 3.2% ✓ (допустимо < 3%)

ШАГ 4: Проверка нагрева при перегрузке
При I = 25 А (на пределе): 
T_линия = 30 + (25² × 0.0178 × 50 / 2.5) / k
         = 30 + 358 / k  [зависит от условий прокладки]
         ≈ 30 + 45 = 75°C  [в трубе, на воздухе]
         
Допустимо до 70°C (ПВЗ изоляция), пограничный случай

ШАГ 5: Вывод
Минимальное: кабель 2.5 мм² + автомат 32 А
Рекомендуемое: кабель 4 мм² + автомат 25 А (запас на будущее)
```

**Итоговая таблица:**

| Параметр | Расчётное | Выбранное |
|----------|-----------|----------|
| Ток потребления | 20.5 А | — |
| Ток автомата | 25.6 А | **32 А** |
| Сечение кабеля | 2.5 мм² | **4 мм²** (рекомендуемое) |
| Потери напряжения | 3.2% | ✓ Допустимо |
| Макс. ток в кабеле | 25 А | Пограничный |

### Пример 2: Асинхронный двигатель 380V

**Задача:** Подобрать кабель и контактор для двигателя 7.5 кВт, 380V, cos(φ)=0.88.

**Входные данные:**
```
├─ Мощность: 7.5 кВт
├─ Напряжение: 380 В (трёхфазное)
├─ cos(φ): 0.88 (двигатель)
├─ Длина кабеля: 50 м
├─ Условие прокладки: в гофре с 2 другими кабелями
└─ Температура окружения: 40°C
```

**Расчёт:**

```
ШАГ 1: Ток двигателя
I = P / (√3 × U × cos(φ)) = 7500 / (1.732 × 380 × 0.88)
I = 7500 / 580.3 = 12.9 А

ШАГ 2: Расчётный ток с коэффициентом запуска
При пуске асинхронный двигатель потребляет I_пуска ≈ 5–7 × I_номин
I_расч = 12.9 × 1.15 = 14.8 А (консервативный запас для ПНВ)

ШАГ 3: Выбор сечения кабеля (из таблицы ГОСТ 52373)
3-жильный кабель (фаза + фаза + фаза) для 380V
Допустимый ток при 30°C: 35 А для сечения 4 мм²

ШАГ 4: Температурная коррекция
При T = 40°C: K_T = 0.91
При скоплении (2 кабеля с ним): K_скоп = 0.8
I_доп_реальн = 35 × 0.91 × 0.8 = 25.5 А ✓ (> 14.8 А)

ШАГ 5: Проверка потерь напряжения (для 380V 3-фазной)
ΔU = (0.0178 × 50 × 12.9) / 4 / √3 = 0.33 В = 0.09% ✓ (отличный)

ШАГ 6: Выбор контактора и защиты
- Контактор: IEC60 А (номинал >= 14.8 А)
- Тепловая защита: 16–20 А (для двигателя)
- Кабель: ВВГ 3×4 мм² (стандартная трёхжильная для 380V)
```

**Итоговая таблица:**

| Параметр | Значение |
|----------|----------|
| Номинальный ток | 12.9 А |
| Ток пуска | ~80 А |
| Расчётный ток | 14.8 А |
| Кабель | **ВВГ 3×4 мм²** |
| Контактор | **IEC 65 А** |
| Тепловая защита | **16 А** |
| Потери напряжения | 0.09% ✓ |

### Пример 3: Конденсатор для коррекции cos(φ)

**Задача:** Поднять cos(φ) с 0.85 до 0.95 для асинхронного двигателя 5.5 кВт.

**Входные данные:**
```
├─ Мощность двигателя: 5.5 кВт
├─ Исходный cos(φ): 0.85
├─ Целевой cos(φ): 0.95
├─ Напряжение: 380 В (трёхфазное)
└─ Частота: 50 Гц
```

**Решение:**

```
ШАГ 1: Найти tan(φ) для исходного и целевого
cos(φ₁) = 0.85 → φ₁ = 31.8° → tan(φ₁) = 0.62
cos(φ₂) = 0.95 → φ₂ = 18.2° → tan(φ₂) = 0.33

ШАГ 2: Требуемая реактивная мощность конденсатора
Q_конд = P × (tan(φ₁) - tan(φ₂))
Q_конд = 5500 × (0.62 - 0.33) = 5500 × 0.29 = 1595 ВАр

ШАГ 3: Расчёт ёмкости (для трёхфазной сети звездой)
C = Q × 10⁶ / (2π × f × U_фаз²)
U_фаз = 380 / √3 = 220 В

C = 1595 × 10⁶ / (2π × 50 × 220²)
C = 1595 × 10⁶ / (2π × 50 × 48400)
C = 1595 × 10⁶ / 15,207,866 = 104.9 мкФ

Выбираем стандартную батарею: 4 × 25 мкФ = 100 мкФ (или 3 × 35 мкФ = 105 мкФ)

ШАГ 4: Проверка новый cos(φ)
Q_новое = Q_исходное - Q_конд
tan(φ_новое) = (P × tan(φ₁) - Q_конд / P) = 0.62 - 0.29 = 0.33 ✓
cos(φ_новое) = 1 / √(1 + 0.33²) = 0.950 ✓

ШАГ 5: Защита конденсатора
Предохранитель: 16 А (на каждой фазе, или общий 25 А)
Контактор (если подключение через ручной переключатель): IEC16 А
```

**Результат:**

| Параметр | Исходное | Целевое |
|----------|----------|---------|
| cos(φ) | 0.85 | 0.95 ✓ |
| Угол φ | 31.8° | 18.2° |
| Q | 3819 ВАр | 2224 ВАр |
| ΔQ | — | 1595 ВАр |
| Ёмкость | — | **100 мкФ** |
| Экономия в сети | — | ~30% (штраф отменен) |

### Пример 4: Расчёт освещения кабинета

**Задача:** Рассчитать количество светильников для кабинета 4×5 м.

**Входные данные:**
```
├─ Площадь: 20 м²
├─ Высота потолка: 2.7 м
├─ Требуемая Е: 400 люкс (ГОСТ 54944)
├─ Тип светильника: потолочный светодиодный
├─ Номинальная мощность светильника: 25 Вт
└─ Световой поток одного: 2500 лм
```

**Решение:**

```
ШАГ 1: Требуемый суммарный поток
Φ_требуемый = (E × S × K_неравн × K_отр) / η

При стандартных коэффициентах:
K_неравн = 0.5 (потолочный светильник)
K_отр = 1.2 (запас на загрязнение)
η = 0.7 (КПД светильника в помещении)

Φ_требуемый = (400 × 20 × 0.5 × 1.2) / 0.7 = 6857 лм

ШАГ 2: Количество светильников
N = Φ_требуемый / Φ_одного = 6857 / 2500 = 2.74 → 3 светильника

ШАГ 3: Проверка равномерности (сетка светильников)
Оптимально: 3 светильника в один ряд или треугольник
сетка 2 × 2.5 м (в помещении 4 × 5 м)

Центральная точка: E_центр ≈ 400 + 100 = 500 люкс (максимум)
Угловая точка (3 м от ближайшего): E_углы ≈ 150–200 люкс (минимум)
Отношение: 200 / 500 = 0.4 ✓ (допустимо, >0.4)

ШАГ 4: Электрическая мощность
P_итого = 3 × 25 Вт = 75 Вт (очень экономно!)
Годовый расход при 8ч/день = 75 × 8 × 365 / 1000 = 219 кВч/год

ШАГ 5: Размещение
Рекомендация: один светильник в центре + два по краям
или симметричная сетка 1.5 м × 2.5 м
```

**Результат:**

| Параметр | Значение |
|----------|----------|
| Требуемый поток | 6857 лм |
| Количество светильников | **3 шт** |
| Мощность одного | 25 Вт × 3 = **75 Вт** |
| Минимальная E | 150–200 люкс |
| Максимальная E | 500 люкс |
| Коэффициент неравномерности | 0.4 ✓ |
| Годовый расход (8ч/день) | 219 кВч |

### Пример 5: Датчик температуры (0–10V → 0–100°C)

**Задача:** Сифилировать и масштабировать сигнал от датчика температуры.

**Входные данные:**
```
├─ Диапазон входного сигнала: 0–10 В
├─ Диапазон выхода: 0–100°C
├─ Класс точности датчика: 0.5
├─ Максимум диапазона: 100°C
└─ Пять измеренных значений: 2.1, 4.5, 5.8, 7.2, 9.9 В
```

**Калибровка и расчёт:**

```
ШАГ 1: Определить калибровочные точки
Обычно датчик линеен в своём диапазоне:
U = 0 В → T = 0°C
U = 10 В → T = 100°C

ШАГ 2: Формула преобразования
T(°C) = (U / 10) × 100 = U × 10

ШАГ 3: Перевести измеренные значения
U = 2.1 В → T = 2.1 × 10 = 21°C
U = 4.5 В → T = 4.5 × 10 = 45°C
U = 5.8 В → T = 5.8 × 10 = 58°C
U = 7.2 В → T = 7.2 × 10 = 72°C
U = 9.9 В → T = 9.9 × 10 = 99°C

ШАГ 4: Оценить погрешность измерения
ΔT_датчика = ± (0.5 / 100) × 100 = ± 0.5°C
ΔU_вольтметра = ± (0.5 / 100) × 10 = ± 0.05 В → ± 0.5°C
Суммарная погрешность ≈ ± 0.7°C (в худшем случае ± √(0.5² + 0.5²) ≈ ± 0.7°C)

ШАГ 5: Табличный результат
```

**Результат:**

| U (В) | T (°C) | Интервал погрешности |
|-------|--------|----------------------|
| 0 | 0 | 0 ± 0.7°C |
| 2.1 | 21 | 21 ± 0.7°C |
| 4.5 | 45 | 45 ± 0.7°C |
| 5.8 | 58 | 58 ± 0.7°C |
| 7.2 | 72 | 72 ± 0.7°C |
| 9.9 | 99 | 99 ± 0.7°C |
| 10 | 100 | 100 ± 0.7°C |

### Пример 6: Расчёт вентиляции квартиры 70м²

**Задача:** Определить требуемый расход воздуха и выбрать вентилятор.

**Входные данные:**
```
├─ Тип: квартира в многоэтажном доме
├─ Площадь: 70 м²
├─ Высота потолков: 2.7 м
├─ Количество людей: 3 постоянно
├─ Наличие кухни: газовая плита
├─ Окна: стеклопакеты (низкая естественная вентиляция)
└─ Климат: Москва (холодная зима, нужна рекуперация)
```

**Расчёт:**

```
ШАГ 1: Требуемый расход по трём методам

Метод 1 — по кратности (воздухооб换 за час):
Объём квартиры: V = 70 м² × 2.7 м = 189 м³
- Спальни (2×14 м² = 28 м²): K = 0.5 → L₁ = 28 × 0.5 = 14 м³/ч
- Гостиная (15 м²): K = 1.0 → L₂ = 15 м³/ч
- Кухня (9 м²): K = 5.0 (газовая!) → L₃ = 45 м³/ч
- Ванная + санузел (8 м²): K = 3.0 → L₄ = 24 м³/ч

Сумма: 14 + 15 + 45 + 24 = 98 м³/ч

Метод 2 — по количеству людей:
3 человека × 60 м³/ч на человека (комфортный уровень) = 180 м³/ч

Метод 3 — по площади кухни:
Кухня газовая: 9 м² × 20 м³/ч на м² = 180 м³/ч

ШАГ 2: Выбрать максимум
L_выбора = MAX(98, 180, 180) = 180 м³/ч

Добавить запас 20%:
L_проектный = 180 × 1.2 = 216 м³/ч ≈ 250 м³/ч (стандартный размер)

ШАГ 3: Выбор воздуховода
Допустимая скорость для жилых помещений: V = 4 м/с
F = L / (V × 3600) = (250 / 3600) / 4 = 0.0173 м²
D = √(4 × F / π) = √(4 × 0.0173 / 3.14) = 0.148 м = 148 мм
→ Выбираем стандартный 150 мм (или 160 мм)

ШАГ 4: Потери давления в воздуховодах
Длина от кухни до крыши: 10 м
Потери: ≈ 0.5 Па/м × 10 м = 5 Па
+ Входное сопротивление: 20 Па
+ Выход на улицу: 15 Па
= Итого: ≈ 40 Па (мало, вентилятор выбирается слабый)

ШАГ 5: Выбор вентилятора
- Расход: 250 м³/ч (230 Вт эквивалент)
- Напор: 40–50 Па
- Тип: радиальный вентилятор, установка в воздуховод
- Мощность: ~40–60 Вт (питание 230V)
- Шум: ~28–32 дБ (при низкой скорости вращения)

ШАГ 6: Рекуперация (опционально)
Без рекуперации: потери тепла на вентиляцию = 250 × 0.33 × 40 = 3300 Вт (зимой)
С рекуператором (η=70%): 3300 × (1-0.70) = 990 Вт (экономия ~3 кВт)
Затраты на рекуператор: 15–30 тыс. рублей, окупается за 2–3 года.
```

**Результат:**

| Параметр | Значение |
|----------|----------|
| Требуемый расход | **250 м³/ч** |
| Диаметр воздуховода | **150–160 мм** |
| Требуемый напор | **40–50 Па** |
| Мощность вентилятора | **40–60 Вт** |
| Тип | **Радиальный, в воздуховод** |
| Шум при работе | **<32 дБ** |
| Потери тепла зимой | 3300 Вт (без рекуп.) |
| С рекуператором | 990 Вт |
| Экономия в год | ~2500 кВч (без рекуп.) |

### Пример 7: Расчёт отопления дома 150 м²

**Задача:** Рассчитать требуемую мощность котла для частного дома.

**Входные данные:**
```
├─ Тип: кирпичный частный дом, частично утеплён
├─ Площадь: 150 м²
├─ Год постройки: 1990 (плохая изоляция)
├─ Регион: Московская область
├─ Характеристика:
│  ├─ Стены: кирпич 2 слоя, уже имеет пену 30 мм (не доходит до норм)
│  ├─ Окна: двойные стекла (старые, K=2.8 Вт/м²К)
│  ├─ Крыша: шифер + утепление 80 мм (нормально)
│  ├─ Пол: на фундаменте, без утепления
│  └─ Вентиляция: естественная, требуется ~200 м³/ч
├─ Расчётная температура: −25°C (Московская область)
├─ Требуемая температура в доме: +20°C → ΔT = 45 К
└─ Высота: 2 этажа, потолки 2.8 м, средняя высота воздуха 3.5 м
```

**Расчёт:**

```
ШАГ 1: Быстрый расчёт по удельной мощности
Кирпичный дом, старый, частично утеплён: q = 120 Вт/м²
Q_базовая = 150 × 120 = 18000 Вт = 18 кВт

ШАГ 2: Детальный расчёт по теплопотерям (если есть размеры)
Площадь стен (примерно):
Длина периметра ≈ 2 × (11 + 14) = 50 м
Наружная стена: 50 × 2.8 × 2 этажа = 280 м² (две стены)
Но есть внутренние стены, общая площадь наружных ≈ 180 м²

U_стены = 0.5 Вт/м²К (кирпич + пена 30 мм, усредненно)
Q_стены = 180 × 0.5 × 45 = 4050 Вт

Окна (примерно 4 окна × 1.5 м²):
A_окна = 6 м²
U_окна = 2.8 Вт/м²К
Q_окна = 6 × 2.8 × 45 = 756 Вт

Крыша (150 м²):
U_крыша = 0.4 Вт/м²К (шифер + 80 мм утепления)
Q_крыша = 150 × 0.4 × 45 = 2700 Вт

Пол на фундаменте (150 м²):
U_пол = 0.5 Вт/м²К (бетон, не утеплён, над холодным подвалом)
Q_пол = 150 × 0.5 × 45 = 3375 Вт

Двери (2 × 2 м² × 3.0 Вт/м²К):
Q_двери = 4 × 3.0 × 45 = 540 Вт

Итого потери через ограждения:
Q_огр = 4050 + 756 + 2700 + 3375 + 540 = 11420 Вт ≈ 11.5 кВт

ШАГ 3: Добавить потери на вентиляцию
L = 200 м³/ч (естественная, дополняется кондиционированием)
Q_вент = 200 × 0.33 × 45 = 2970 Вт ≈ 3 кВт

Q_итого = 11.5 + 3 = 14.5 кВт

ШАГ 4: Сравнение методов
- Быстрый метод: 18 кВт (консервативный)
- Детальный: 14.5 кВт (реалистичный)
- Рекомендуемый выбор: 16 кВт (средний, с запасом)

ШАГ 5: Рекомендуемый котёл
- Настенный газовый котёл: 16–18 кВт (типовой размер для этого дома)
- Эффективность современного котла: >90%
- Дополнительно: 100 литров буфера (тепловой аккумулятор) для стабильности
- Дровяной камин/печь: +5 кВт для пиковых нагрузок (опционально)

ШАГ 6: Расширенная система с рекуперацией
Если установить рекуператор:
Q_вент_новая = 2970 × (1 - 0.70) = 891 Вт
Q_итого_с_рекуп = 11.5 + 0.9 = 12.4 кВт
Новый расчётный котёл: 14 кВт (экономия 2 кВт, ~12%)
```

**Результат:**

| Параметр | Быстрый метод | Детальный метод | Рекомендуемое |
|----------|---|---|---|
| Потери через ограждения | — | 11.5 кВт | — |
| Потери на вентиляцию | — | 3.0 кВт | — |
| Базовая мощность | 18 кВт | 14.5 кВт | **16 кВт** |
| Выбираемый котёл | 18 кВт | — | **16–18 кВт** |
| С рекуператором | — | 12.4 кВт | **14–15 кВт** |
