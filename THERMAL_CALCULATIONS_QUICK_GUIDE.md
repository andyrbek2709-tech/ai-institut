## Когда использовать

### `tx_heat_balance` — Тепловая мощность

**Используйте когда нужно:**
- Подобрать котёл для отопления
- Определить мощность тепловой насоса
- Рассчитать холодопроизводительность кондиционера
- Проверить способность системы охлаждения отвести тепло от оборудования

**Не используйте для:**
- Расчёта теплопотерь через стены (для этого используйте специальные расчёты теплозащиты по СП 50.13330)
- Определения потерь в трубах (используйте `tt_pressure_drop` + удельные потери)

**Входные данные которых вам нужно:**
- Расход жидкости (м³/ч → пересчитайте в кг/с)
- Изменение температуры (°C или K)
- Тип жидкости (вода, воздух, масло — см. таблицу теплоёмкостей)

---

### `tt_pressure_drop` — Потери давления

**Используйте когда нужно:**
- Подобрать насос (определить требуемый напор)
- Проверить скорость потока в трубе (не более 2-3 м/с для жилых зданий)
- Рассчитать падение давления по трубопроводу
- Определить, будет ли достаточно статического давления для подачи воды

**Не используйте для:**
- Расчёта потерь в компонентах сети по отдельности (клапаны, фильтры)
- Определения расхода в сетях с переменным сечением

**Входные данные которых вам нужно:**
- Длина участка трубопровода (м)
- Диаметр трубы (мм или см)
- Объёмный расход (м³/ч или л/с)
- Материал трубы (для определения шероховатости)
- Тип жидкости (плотность, вязкость)

## Пошаговые примеры

### Задача 1: Выбор котла для дома

**Исходные данные:**
- Дом площадью 200 м² в средней полосе России
- Радиаторное отопление со скоростью циркуляции 1 м³/ч = 0.278 л/с

**Шаг 1. Определите расход**
- Стандартная схема: ΔT = 20°C (подача 70°C, обратка 50°C)
- Для 200 м² типично G = 10-15 кг/с (расход зависит от температуры питания)
- При ΔT = 20°C и требуемой мощности ~20 кВт: G = Q / (c · ΔT) = 20 000 / (4.18 · 20) ≈ 240 кг/ч ≈ 0.067 кг/с

**Неправильно расчитали — возьмём из практики:**
- Для дома 200 м² типовой расход 0.2-0.3 кг/с (12-18 л/мин)
- Подставим G = 0.25 кг/с

**Шаг 2. Определите требуемую тепловую мощность**

Используем формулу `Q = G · c · ΔT`:
```
Q = 0.25 · 4.18 · 20 = 20.9 кВт
```

Учитываем коэффициент безопасности 1.2-1.3 (котёл не будет работать на максимум):
```
Q_котла = 20.9 · 1.2 = 25 кВт
```

**Вывод:** Требуется котёл мощностью примерно **25-30 кВт**

---

### Задача 2: Проверка скорости в водопроводе

**Исходные данные:**
- Внутридомовой водопровод ХВС
- Диаметр трубы 32 мм (стандартная ПВХ для одного стояка)
- Расход на стояк ~5 л/мин = 0.083 л/с = 0.00083 м³/с

**Шаг 1. Рассчитайте скорость потока**
```
A = π · d² / 4 = 3.14159 · 0.032² / 4 = 0.000804 м²
v = Q / A = 0.00083 / 0.000804 = 1.03 м/с
```

**Шаг 2. Проверьте норму**
- Максимальная скорость для ХВС по СНиП: **≤ 2 м/с**
- Ваша скорость: **1.03 м/с** ✅ В норме

**Вывод:** Скорость приемлема, шум и вибрация в трубе будут минимальны

---

### Задача 3: Выбор насоса для отопления

**Исходные данные:**
- Замкнутая система отопления
- Трубопровод до радиаторов и обратно: L = 150 м (примерно 3 этажа + разводка)
- Диаметр трубы 20 мм (типовой для отопления)
- Расход Q = 0.2 м³/ч = 0.0000556 м³/с
- Материал: сталь б/у (диаметр после коррозии ~19 мм, Δ = 0.3 мм)

**Шаг 1. Рассчитайте скорость**
```
v = Q / (π · d² / 4) = 0.0000556 / (3.14159 · 0.02² / 4)
v = 0.0000556 / 0.0003142 = 0.177 м/с
```

**Шаг 2. Определите число Рейнольдса**
```
Re = v · d / ν = 0.177 · 0.02 / (1.0 × 10⁻⁶) = 3540

Режим: турбулентный (Re > 2300) ✓
```

**Шаг 3. Определите коэффициент трения λ**

Относительная шероховатость:
```
Δ/d = 0.3 / 20 = 0.015
```

По формуле Холла–Кондратьева:
```
λ = 0.11 · ((0.015) + (68/3540))^0.25
λ = 0.11 · (0.015 + 0.0192)^0.25
λ = 0.11 · (0.0342)^0.25
λ = 0.11 · 0.433 = 0.0476
```

**Шаг 4. Рассчитайте потери давления в трубах**
```
ΔP_трубы = 0.0476 · (150 / 0.02) · (1000 · 0.177² / 2)
ΔP_трубы = 0.0476 · 7500 · 15.65
ΔP_трубы = 55 900 Па = 55.9 кПа ≈ 5.7 м вод.ст.
```

**Шаг 5. Учтите местные сопротивления**

Типовая система отопления содержит:
- 20+ колен 90° (ξ = 1.0 каждое)
- 5+ тройников (ξ = 0.3 каждое)
- Задвижки (ξ = 0.2 каждая)
- Радиаторы (ξ ≈ 2-5)

**Примерное суммарное ξ ≈ 20-30**

```
ΔP_местные = Σξ · (ρ · v² / 2)
ΔP_местные = 25 · (1000 · 0.177² / 2)
ΔP_местные = 25 · 15.65 = 391 Па ≈ 0.4 м вод.ст.
```

**Шаг 6. Суммируйте потери**
```
ΔP_общие = 55.9 + 0.4 = 56.3 кПа ≈ 5.75 м вод.ст.
```

**Вывод:** Для отопления требуется циркуляционный насос с напором **6-7 м вод.ст.** (примерно 0.6-0.7 бара, мощность 50-100 Вт)

## Коэффициенты

### Теплоёмкость (c, кДж/(кг·°C))

| Жидкость | Значение | Точность | Для какой системы |
|----------|----------|----------|------------------|
| Вода дистиллированная | 4.18 | ±0.1 | Отопление, ХВС, охлаждение |
| Вода водопроводная | 4.19 | ±0.1 | Отопление, ХВС |
| Этиленгликоль 50% | 3.70 | ±0.2 | Антифризные системы |
| Воздух (при ~20°C) | 1.00 | ±0.05 | Вентиляция, кондиционирование |
| Пар перегретый | 1.90 | ±0.3 | Паровое отопление |
| Масло минеральное (ISO 46) | 2.00 | ±0.2 | Гидравлические системы, промышленность |

### Плотность (ρ, кг/м³)

| Жидкость | Значение | Точность |
|----------|----------|----------|
| Вода при 4°C | 1000 | +0% |
| Вода при 20°C | 998 | ≈1000 |
| Вода при 50°C | 988 | ≈1000 |
| Вода при 100°C | 958 | 1% |
| Воздух при 20°C, 1 атм | 1.2 | ±0.1 |
| Масло минеральное | 850-900 | ±2% |

### Вязкость (ν, м²/с, или сСт = 10⁻⁶ м²/с)

| Жидкость | ν, м²/с | Типовой Reynolds |
|----------|---------|-----------------|
| Вода (20°C) | 1.0 × 10⁻⁶ | Турбулентный > 4000 |
| Вода (50°C) | 0.556 × 10⁻⁶ | Турбулентный > 3000 |
| Масло ISO 32 | 32 × 10⁻⁶ | Ламинарный < 4000 |
| Масло ISO 46 | 46 × 10⁻⁶ | Ламинарный < 3000 |
| Масло ISO 68 | 68 × 10⁻⁶ | Ламинарный < 2500 |

### Коэффициент местного сопротивления (ξ)

| Элемент | Минимум | Типично | Максимум | Примечание |
|---------|---------|---------|----------|-----------|
| Колено 90° | 0.6 | 1.0 | 1.3 | Зависит от r/d |
| Колено 45° | 0.3 | 0.5 | 0.8 | Плавнее |
| Вход в трубу (острый) | 0.5 | 0.5 | 0.5 | Классический |
| Вход в трубу (плавный) | 0.05 | 0.08 | 0.1 | Лучше |
| Выход из трубы | 0.8 | 1.0 | 1.2 | Мало влияет |
| Тройник (боковой отвод) | 0.5 | 1.0 | 2.0 | Сильная турбулентность |
| Задвижка полностью открыта | 0.1 | 0.15 | 0.2 | Минимум |
| Задвижка 75% открыта | 0.5 | 0.8 | 1.2 | Быстро растёт |
| Обратный клапан | 2.0 | 3.0 | 5.0 | Зависит от типа |

### Шероховатость труб (Δ, мм)

| Материал | Δ, мм | Состояние | Применение |
|----------|-------|----------|-----------|
| **Новые трубы** | | | |
| Полиэтилен (ПЭ) | 0.005 | Очень гладкие | Водопровод холодный |
| Полипропилен (ПП) | 0.007 | Очень гладкие | Водопровод горячий |
| ПВХ | 0.01 | Гладкие | Водопровод, вентиляция |
| Медь | 0.0015 | Очень гладкие | ХВС, ГВС (дорого) |
| Нержавеющая сталь | 0.01-0.03 | Гладкие | Системы под давлением |
| Сталь гальванизированная | 0.08 | Слегка гладкая | Отопление, водопровод |
| | | | |
| **Б/у трубы (5-10 лет)** | | | |
| Пластик | 0.01 | Без изменений | Ещё нормальные |
| Сталь | 0.3 | С лёгкой коррозией | Замена через 5-10 лет |
| Чугун | 0.5 | Коррозия | Замена обязательна |
| | | | |
| **Старые трубы (20+ лет)** | | | |
| Сталь | 1.0 | Сильная коррозия | Требует замены |

## Частые ошибки

### Ошибка 1: Забыли перевести единицы

**Неправильно:**
```
G = 10 м³/ч (оставили в м³/ч вместо кг/с)
c = 4.18 кДж/(кг·°C)
ΔT = 20°C

Q = 10 · 4.18 · 20 = 836 кДж??? (смешанные единицы!)
```

**Правильно:**
```
G = 10 м³/ч = 10 / 3.6 м³/с = 2.78 м³/с
G = 2.78 · 1000 = 2780 кг/с (масса в кг/с)

Q = 2780 · 4.18 · 20 = 232 500 кВт = 232.5 МВт (огромная мощность!)
```

✅ **Проверка правдоподобности:** 232.5 МВт — это мощность ТЭЦ. Для дома это нереально. Значит, расход задан неверно.

### Ошибка 2: Использовали объёмный расход вместо массового

**Неправильно:**
```
Q = 10 м³/ч · 4.18 · 20 = 836 кДж (смешанные единицы)
```

**Правильно:**
```
Q_массовый = 10 м³/ч · 1000 кг/м³ = 10 000 кг/ч = 2.78 кг/с

Q = 2.78 · 4.18 · 20 = 232.5 кВт (более реалистично для большого здания)
```

### Ошибка 3: Забыли про местные сопротивления

**Неправильно:**
```
ΔP = только потери на трение в прямых участках

Вывод: "Достаточно насоса на 5 м вод.ст."
```

**Правильно:**
```
ΔP = потери на трение + суммарные ξ для всех колен, тройников, радиаторов

Местные сопротивления часто составляют 20-50% от общих потерь!

Вывод: "Требуется насос на 7-10 м вод.ст."
```

### Ошибка 4: Неправильный коэффициент трения λ

**Неправильно:**
```
Re = 5000 (турбулентный)
Диаметр не важен → использовать λ = 0.03 для всех случаев
```

**Правильно:**
```
λ зависит от:
1) Числа Рейнольдса (Re)
2) Относительной шероховатости (Δ/d)
3) Возраста и состояния трубы

Для новой ПЕ-трубы: λ ≈ 0.015-0.02
Для старой стальной трубы: λ ≈ 0.03-0.05
Для сильно заржавленной: λ ≈ 0.05-0.08
```

### Ошибка 5: Забыли про сжимаемость воздуха

**Неправильно:**
```
Воздух как несжимаемая жидкость (используете формулы для воды)
```

**Правильно:**
```
Для воздуха в вентиляции:
- Используйте реальную плотность ρ = 1.2 кг/м³ (не 1000!)
- При больших скоростях (> 5 м/с) учитывайте компрессibility
- Для больших сетей может быть значительное расширение воздуха
```

## Когда нужны уточнения и справки

| Когда | Что делать |
|------|-----------|
| Температура среды сильно отличается от 20°C | Пересчитайте c и ρ по свойствам при реальной T |
| Система с несколькими типами труб | Рассчитайте потери для каждого участка отдельно |
| Большой перепад высот (> 10 м) | Учтите гидростатическое давление: ΔP_гидр = ρ · g · h |
| Расход переменный (насос, клапан) | Рассчитайте потери при максимальном расходе |
| Трубопровод на улице в мороз | Используйте вязкость и плотность при реальной температуре |
